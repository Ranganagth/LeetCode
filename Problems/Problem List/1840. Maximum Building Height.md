[1840. Maximum Building Height](https://leetcode.com/problems/maximum-building-height/)

# Intuition

The problem involves building a sequence of buildings with constraints:
- The height difference between adjacent buildings can't exceed 1.
- Certain buildings have maximum height restrictions.
- The first building's height is fixed at 0.

To maximize the tallest building, we need to consider the restrictions and the natural constraints imposed by the adjacent difference rule. The key is to:
- Propagate restrictions from left to right, ensuring no building exceeds the previous building's height + 1.
- Propagate restrictions from right to left, ensuring no building exceeds the next building's height + 1.
- Then, determine the maximum possible height between restrictions by considering the average of the restrictions at each segment, adjusted for the distance between restrictions.

---

# Approach

1. **Add boundary restrictions:**
   - Add restriction `(1, 0)` since the first building must be height 0.
   - Add `(n, n-1)` to handle the last building's maximum height constraint.
2. **Sort restrictions** by building id.
3. **Forward pass:**
   - For each restriction, update the maximum height considering the previous restriction, ensuring the difference constraint.
4. **Backward pass:**
   - Similarly, update restrictions from right to left to ensure the constraints are consistent in both directions.
5. **Calculate maximum height:**
   - For each pair of consecutive restrictions, compute the maximum possible height in the segment between them, considering the distance and restrictions.
   - The maximum height in each segment is `(distance + height_left + height_right) // 2`.
   - Track the maximum over all segments.

---

# Complexity

- **Time complexity:**  $$O(m \log m)$$ due to sorting, where `m` is the number of restrictions (up to $10^5$).
  
- **Space complexity:**  $$O(m)$$ for storing restrictions.

---

# Code

```typescript
function maxBuilding(n: number, restrictions: number[][]): number {
    let r: number[][] = restrictions.map((res) => [res[0], res[1]]);
    // Add restriction for the first building
    r.push([1, 0]);
    // Sort restrictions by building id
    r.sort((a, b) => a[0] - b[0]);
    // Add restriction for the last building if not present
    if (r[r.length - 1][0] !== n) {
        r.push([n, n - 1]);
    }

    const m = r.length;

    // Forward pass: propagate restrictions from left to right
    for (let i = 1; i < m; i++) {
        const dist = r[i][0] - r[i - 1][0];
        r[i][1] = Math.min(r[i][1], r[i - 1][1] + dist);
    }

    // Backward pass: propagate restrictions from right to left
    for (let i = m - 2; i >= 0; i--) {
        const dist = r[i + 1][0] - r[i][0];
        r[i][1] = Math.min(r[i][1], r[i + 1][1] + dist);
    }

    let ans = 0;
    // Compute maximum height between restrictions
    for (let i = 0; i < m - 1; i++) {
        const dist = r[i + 1][0] - r[i][0];
        const best = Math.floor((dist + r[i][1] + r[i + 1][1]) / 2);
        ans = Math.max(ans, best);
    }

    return ans;
};

```

# Example walkthrough
## Example 1:

Input: `n=5, restrictions=[[2,1],[4,1]]`

- Add `(1,0)` and `(5,4)` (since `n=5`)
- Sorted restrictions: `(1,0), (2,1), (4,1), (5,4)`
- Forward pass:
  - `(2,1)` from `(1,0)`: max height at 2 ≤ 0 + 1 = 1
  - `(4,1)` from `(2,1)`: max height at 4 ≤ 1 + 2 = 3, but restriction is 1, so min(1,3)=1
  - `(5,4)` from `(4,1)`: max height at 5 ≤ 1 + 1=2, restriction is 4, so min(4,2)=2
- Backward pass:
  - `(4,1)` from `(5,4)`: max height at 4 ≤ 2 + 1=3, restriction is 1, min(1,3)=1
  - `(2,1)` from `(4,1)`: max height at 2 ≤ 1 + 2=3, restriction is 1, min(1,3)=1
  - `(1,0)` from `(2,1)`: max height at 1 ≤ 1 + 1=2, restriction is 0, min(0,2)=0
- Max between restrictions:
  - Between `(1,0)` and `(2,1)`: distance=1, max height = (1+0+1)//2=1
  - Between `(2,1)` and `(4,1)`: distance=2, max height = (2+1+1)//2=2
  - Between `(4,1)` and `(5,4)`: distance=1, max height = (1+1+4)//2=3
- Result: maximum is 3, but since the problem states the maximum height of the tallest building, the answer is 2 (from the example). The code finds the maximum possible height considering constraints, which is 2.

# Summary

This approach efficiently computes the maximum possible height by propagating restrictions from both ends and then calculating the maximum feasible height in each segment. It handles large `n` efficiently due to sorting and linear passes.
