[381. Insert Delete GetRandom O(1) - Duplicates allowed](https://leetcode.com/problems/insert-delete-getrandom-o1-duplicates-allowed/)

# Intuition

Three operations required:

```
insert → O(1)
remove → O(1)
getRandom → O(1)
```

Key constraint:

```
duplicates allowed
```

Direct structures fail:

* Array → fast random, slow removal
* Map → fast lookup, no random access

Combine both:

```
1. Dynamic array (list)
2. HashMap: value → set of indices in array
```

Why this works:

* Array → enables O(1) random
* Map → enables O(1) locate + delete

---

# Core Idea

Maintain:

```
list = [values]
map = value → {indices in list}
```

Example:

```
list = [1,1,2]
map:
1 → {0,1}
2 → {2}
```

## Insert Operation

Steps:

```
1. Append value to list
2. Add index to map[val]
3. Return true if first occurrence
```

## Remove Operation (Critical)

Goal:

```
remove in O(1)
```

Direct removal from array is O(n). Avoid shifting.

Strategy:

```
swap with last element, then pop
```

Steps:

```
1. Get any index of val
2. Get last element in list
3. Overwrite index with last element
4. Update map for both elements
5. Remove last element
```

## getRandom

```
return list[randomIndex]
```

Uniform distribution naturally handled.

## Approach Summary

```
list → stores elements
map → tracks indices of each value
swap-delete → ensures O(1) removal
```

---

# Complexity

- **Time Complexity:**
	- insert  → $O(1)$
	- remove  → $O(1)$
	- getRandom → $O(1)$
All operations are average constant time.

- **Space Complexity:** $O(n)$
	* list stores all elements
	* map stores indices

# Code

```typescript 
class RandomizedCollection {
    private map: Map<number, Set<number>>;
    private list: number[];

    constructor() {
        this.map = new Map();
        this.list = [];
    }

    insert(val: number): boolean {

        let isNew = !this.map.has(val);

        if (isNew) {
            this.map.set(val, new Set());
        }

        this.map.get(val)!.add(this.list.length);
        this.list.push(val);

        return isNew;
    }

    remove(val: number): boolean {

        if (!this.map.has(val) || this.map.get(val)!.size === 0) {
            return false;
        }

        const indexToRemove = this.map.get(val)!.values().next().value;
        const lastIndex = this.list.length - 1;
        const lastElement = this.list[lastIndex];

        // swap
        this.list[indexToRemove] = lastElement;

        // update map for val
        this.map.get(val)!.delete(indexToRemove);

        // update map for last element
        if (indexToRemove !== lastIndex) {
            this.map.get(lastElement)!.delete(lastIndex);
            this.map.get(lastElement)!.add(indexToRemove);
        }

        if (this.map.get(val)!.size === 0) {
            this.map.delete(val);
        }

        this.list.pop();

        return true;
    }

    getRandom(): number {
        const randomIndex = Math.floor(Math.random() * this.list.length);
        return this.list[randomIndex];
    }
};

```

- Array enables randomness
- Hash map enables constant-time indexing
- Swap-delete avoids O(n) removal
