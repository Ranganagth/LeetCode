[378. Kth Smallest Element in a Sorted Matrix](https://leetcode.com/problems/kth-smallest-element-in-a-sorted-matrix/)

# Intuition

Matrix property:

```
rows sorted
columns sorted
```

This creates a **monotonic structure**.

Instead of flattening:

```
O(n²) memory → not allowed
```

Search directly in **value space**, not index space.

Observation:

For any value `x`, you can count:

```
how many elements ≤ x
```

If count ≥ k → answer is ≤ x
If count < k → answer is > x

This enables **binary search on values**.

---

# Approach

### Step 1 - Define search range

```
low  = matrix[0][0]
high = matrix[n-1][n-1]
```

### Step 2 - Binary Search

```
while low < high:
    mid = (low + high) // 2

    count = number of elements ≤ mid

    if count < k:
        low = mid + 1
    else:
        high = mid
```


### Step 3 - Efficient Counting

Use **bottom-left traversal**:

Start at:

```
row = n-1, col = 0
```

Rule:

```
if matrix[row][col] ≤ mid:
    all elements above are ≤ mid
    count += row + 1
    move right
else:
    move up
```

Time:

```
O(n)
```

## Algorithm

1. Binary search over values.
2. For each `mid`, count elements ≤ `mid` in `O(n)`.
3. Narrow range until answer found.

---

# Complexity

- **Time Complexity:**
	Binary search: $O(log(max - min))$
	Each count: $O(n)$
	
	**Total:** $O(n log(max - min))$

- **Space Complexity:** $O(1)$
	No extra structures used.

# Code

```typescript 
function kthSmallest(matrix: number[][], k: number): number {

    const n = matrix.length;

    let low = matrix[0][0];
    let high = matrix[n - 1][n - 1];

    function countLessEqual(x: number): number {

        let count = 0;
        let row = n - 1;
        let col = 0;

        while (row >= 0 && col < n) {

            if (matrix[row][col] <= x) {
                count += row + 1;
                col++;
            } else {
                row--;
            }
        }

        return count;
    }

    while (low < high) {

        const mid = Math.floor((low + high) / 2);

        const count = countLessEqual(mid);

        if (count < k) {
            low = mid + 1;
        } else {
            high = mid;
        }
    }

    return low;
};

```

8 elements

## Key Insight

- Sorted rows + columns → enables monotonic counting
- Value-based binary search avoids O(n²) memory
