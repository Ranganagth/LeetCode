[948. Bag of Tokens](https://leetcode.com/problems/bag-of-tokens/)
# Intuition
To maximize the score, we should prioritize playing tokens with the smallest value face-up to gain points and use tokens with the largest value face-down to regain power when needed. This greedy approach ensures efficient use of resources, balancing between gaining points and restoring power.

---

# Approach
1. Sort the `tokens` array in ascending order.
2. Use two pointers:
   - `left` starting at the beginning (smallest tokens).
   - `right` starting at the end (largest tokens).
3. Maintain variables:
   - `power` (initial power).
   - `score` (initially 0).
   - `maxScore` to track the maximum score achieved.
4. Loop while `left <= right`:
   - If `power >= tokens[left]`, play the token face-up:
     - Deduct `tokens[left]` from `power`.
     - Increment `score`.
     - Update `maxScore`.
     - Move `left` forward.
   - Else if `score >= 1`, play the token face-down:
     - Add `tokens[right]` to `power`.
     - Decrement `score`.
     - Move `right` backward.
   - Else, break the loop.
5. Return `maxScore`.

---

# Complexity
- **Time complexity:** 
  $$O(n \log n)$$ due to sorting, where \( n \) is the number of tokens.
- **Space complexity:** 
  $$O(1)$$ or \( O(n) \) depending on the sorting implementation.

---

# Code

```typescript
function bagOfTokensScore(tokens: number[], power: number): number {
    tokens.sort((a, b) => a - b);
    let left = 0;
    let right = tokens.length - 1;
    let score = 0;
    let maxScore = 0;

    while (left <= right) {
        if (power >= tokens[left]) {
            // Play face-up
            power -= tokens[left];
            score++;
            maxScore = Math.max(maxScore, score);
            left++;
        } else if (score >= 1) {
            // Play face-down
            power += tokens[right];
            score--;
            right--;
        } else {
            break;
        }
    }

    return maxScore;
};

```

# Example walkthrough

## Example 1:

`tokens = [100], power = 50`

| Step | power | score | action | remaining tokens | maxScore |
|-------|--------|--------|---------|---------------------|----------|
| Initial | 50 | 0 | - | [100] | 0 |
| Check if power >= 100? | 50 >= 100? | No | - | - | 0 |
| Can play face-down? | score >= 1? | No | - | - | 0 |
| Loop ends | - | - | - | - | 0 |

**Output:** `0`

`tokens = [200, 100], power = 150`

| Step | power | score | action | remaining tokens | maxScore |
|-------|--------|--------|---------|---------------------|----------|
| Initial | 150 | 0 | - | [100, 200] | 0 |
| Play face-up with 100 | 150 >= 100 | Yes | power=50, score=1 | [200] | 1 |
| Cannot play face-up (power=50 < 200) | No | - | - | - | 1 |
| Play face-down with 200 | score=1 >= 1 | Yes | power=250, score=0 | [] | 1 |
| Loop ends | - | - | - | - | 1 |

**Output:** `1`
