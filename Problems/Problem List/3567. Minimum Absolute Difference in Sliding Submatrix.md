[3567. Minimum Absolute Difference in Sliding Submatrix](https://leetcode.com/problems/minimum-absolute-difference-in-sliding-submatrix/)

# Intuition

For each `k × k` submatrix:

```
collect all elements → find minimum |a - b| among distinct values
```

Key observation:

```
minimum absolute difference occurs between adjacent elements in sorted order
```

So:

```
sort values → check consecutive differences
```

Constraint:

```
m, n ≤ 30
k ≤ 30
```

Brute force per submatrix is acceptable.

---

# Approach

For every valid top-left `(i, j)`:

### Step 1 - Extract elements

```
collect all k*k elements
```

### Step 2 - Sort

```
sort ascending
```

### Step 3 - Compute min difference

```
iterate adjacent pairs
skip duplicates
track minimum difference
```

If all elements same:

```
return 0
```

## Algorithm

```
for i in 0 → m-k
  for j in 0 → n-k
    extract submatrix elements
    sort
    compute min adjacent difference
```

---

# Complexity

Let:

```
m, n ≤ 30
k ≤ 30
```

Submatrices:

```
O(m * n)
```

Each submatrix:

```
collect = O(k²)
sort = O(k² log k²)
scan = O(k²)
```

Total:

```
O(m * n * k² log k)
```

**Worst case:**

```
≈ 30 * 30 * 900 * log(900)
```

Efficient.

- **Space Complexity**

```
O(k²)
```

Temporary array for each submatrix.

# Code

```typescript 
function minAbsDiff(grid: number[][], k: number): number[][] {

    const m = grid.length;
    const n = grid[0].length;

    const result: number[][] = [];

    for (let i = 0; i <= m - k; i++) {

        const row: number[] = [];

        for (let j = 0; j <= n - k; j++) {

            const values: number[] = [];

            // collect k x k elements
            for (let x = i; x < i + k; x++) {
                for (let y = j; y < j + k; y++) {
                    values.push(grid[x][y]);
                }
            }

            values.sort((a, b) => a - b);

            let minDiff = Infinity;

            for (let t = 1; t < values.length; t++) {

                if (values[t] !== values[t - 1]) {
                    minDiff = Math.min(minDiff, values[t] - values[t - 1]);
                } else {
                    minDiff = 0;
                    break;
                }
            }

            if (minDiff === Infinity) minDiff = 0;

            row.push(minDiff);
        }

        result.push(row);
    }

    return result;
}
```

# Example Walkthrough

### Example 1

```
grid =
[
[1, 8],
[3,-2]
]

k = 2
```

Only one submatrix:

```
[1, 8]
[3,-2]
```

Values:

```
[1,8,3,-2]
```

Sort:

```
[-2,1,3,8]
```

Differences:

```
| -2 - 1 | = 3
| 1 - 3 | = 2   ← minimum
| 3 - 8 | = 5
```

Result:

```
[[2]]
```

---

### Example 2

```
grid = [[3,-1]]
k = 1
```

Submatrices:

```
[3], [-1]
```

Each has single element:

```
no pair → answer = 0
```

Result:

```
[[0,0]]
```

### Example 3

```
grid =
[
[1,-2,3],
[2, 3,5]
]

k = 2
```

Submatrix (0,0):

```
[1,-2]
[2, 3]
```

Values:

```
[-2,1,2,3]
```

Differences:

```
3,1,1 → min = 1
```


Submatrix (0,1):

```
[-2,3]
[3,5]
```

Values:

```
[-2,3,3,5]
```

Sorted:

```
[-2,3,3,5]
```

Differences:

```
5,0 → min = 0 → but duplicates → skip → next is 2
```

Actual distinct pairs:

```
3 and 5 → 2
```

Result:

```
[[1,2]]
```

## Key Insight

- Minimum absolute difference always lies between adjacent sorted elements
