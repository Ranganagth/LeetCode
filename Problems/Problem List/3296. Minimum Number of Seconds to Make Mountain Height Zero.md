[3296. Minimum Number of Seconds to Make Mountain Height Zero](https://leetcode.com/problems/minimum-number-of-seconds-to-make-mountain-height-zero/)

# Intuition

For worker `i` with time `t`:

Time required to remove height `x`:

```
t + 2t + 3t + ... + xt
```

Arithmetic series:

```
t * (1 + 2 + ... + x)
```

Sum formula:

```
1 + 2 + ... + x = x(x+1)/2
```

Therefore worker time:

```
time = t * x(x+1) / 2
```

Workers operate **simultaneously**.
Total removed height must reach:

```
≥ mountainHeight
```

Goal:

```
minimize maximum finishing time
```

Equivalent transformation:

Binary search the minimum time `T` such that workers can remove enough height within `T`.

## Key Relation

Worker removes `x` height if:

```
t * x(x+1)/2 ≤ T
```

Solve inequality for `x`:

```
x(x+1) ≤ 2T / t
```

Let

```
k = floor(2T / t)
```

Solve quadratic:

```
x² + x − k ≤ 0
```

Maximum integer `x`:

```
x = floor((√(1 + 4k) − 1) / 2)
```

This gives height removed by one worker within time `T`.

---

# Approach

1. Binary search time `T`.
2. For each candidate `T`:

   * Compute height each worker can reduce.
   * Sum heights.
3. If total height ≥ `mountainHeight`, time is feasible.
4. Search smaller time.

Search range:

```
low = 0
high = min(workerTimes) * H(H+1)/2
```

Worst case: fastest worker does entire work.

## Algorithm

Binary search:

```
while low ≤ high
    mid = (low + high)/2

    if can(mid)
        answer = mid
        high = mid - 1
    else
        low = mid + 1
```

Feasibility check:

```
totalHeight = 0

for each worker
    compute x
    totalHeight += x

return totalHeight ≥ mountainHeight
```

---

# Complexity

Let
$$N = workers
H = mountainHeight
$$
Binary search range: $log(H²) ≈ log(10¹⁰) ≈ 34$

Feasibility check: $O(N)$

**Total time:** $O(N log H)$

Maximum: $10⁴ * 34$ - Efficient.

**Space Complexity:** $O(1)$

Only scalar variables used.

# Code

```typescript
function minNumberOfSeconds(
    mountainHeight: number,
    workerTimes: number[]
): number {

    const minTime = Math.min(...workerTimes);

    let left = 0;
    let right = minTime * mountainHeight * (mountainHeight + 1) / 2;
    let ans = right;

    function canFinish(time: number): boolean {

        let total = 0;

        for (const t of workerTimes) {

            const k = Math.floor((2 * time) / t);

            const x = Math.floor((Math.sqrt(1 + 4 * k) - 1) / 2);

            total += x;

            if (total >= mountainHeight) return true;
        }

        return false;
    }

    while (left <= right) {

        const mid = Math.floor((left + right) / 2);

        if (canFinish(mid)) {
            ans = mid;
            right = mid - 1;
        }
        else {
            left = mid + 1;
        }
    }

    return ans;
};

```

Input

```

workerTimes = [2,1,1]
```

Test time `T = 3`.

Worker `t = 2`

```
2 * x(x+1)/2 ≤ 3
x(x+1) ≤ 3

x = 1
```

Worker `t = 1`

```
x(x+1)/2 ≤ 3
x = 2
```

Worker `t = 1`

```
x = 2
```

Total removed:

```
1 + 2 + 2 = 5 ≥ 4
```

Time `3` works.

Try smaller time:

```
T = 2
```

Heights:

```
1 + 1 + 1 = 3 < 4
```

Insufficient.

Minimum time:

```
3
```
