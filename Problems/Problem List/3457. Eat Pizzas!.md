[3457. Eat Pizzas!](https://leetcode.com/problems/eat-pizzas/)

# Intuition

The key insight is that we want to **maximize the total weight gained**. On odd days, we gain the heaviest pizza's weight (Z), and on even days, we gain the second heaviest pizza's weight (Y). To optimize:
- Sort the pizzas in ascending order.
- On odd days, greedily pick the heaviest available pizzas.
- On even days, pick the second heaviest available pizzas.
- For the remaining two pizzas each day, pick the lightest ones to minimize waste.

This greedy approach ensures we maximize the sum of Z values on odd days and Y values on even days.

---

# Approach

1. **Sort pizzas** in ascending order.
2. **Calculate number of days**: `days = n / 4`.
3. **Pick heaviest pizzas for odd days**:
   - Start from the end of the sorted array.
   - For each odd day, take the heaviest pizza and add it to total weight.
   - Decrement the index.
4. **Pick second heaviest pizzas for even days**:
   - Skip one pizza (to account for the lightest pizza we'll pair with).
   - For each even day, take the second heaviest pizza and add it to total weight.
   - Decrement the index by 2 (skip the lightest pizza).
5. **Return total weight**.

---

# Complexity

- **Time complexity:** 
  $$O(n \log n)$$ due to sorting.
- **Space complexity:** 
  $$O(1)$$ if we don't count the sorting space, or $$O(n)$$ if we do.

---

# Code

```typescript
function maxWeight(pizzas: number[]): number {
    pizzas.sort((a, b) => a - b);
    const n = pizzas.length;
    const days = n / 4;
    let totalWeight = 0;
    let index = n - 1;

    // Pick heaviest pizza on odd days
    for (let day = 1; day <= days; day += 2) {
        totalWeight += pizzas[index--];
    }

    // Pick second heaviest pizza on even days
    index--;
    for (let day = 2; day <= days; day += 2) {
        totalWeight += pizzas[index];
        index -= 2;
    }

    return totalWeight;
};

```

# Example walkthrough

## Example 1:
`pizzas = [1,2,3,4,5,6,7,8]`

**After sorting:**
`[1,2,3,4,5,6,7,8]`

**Number of days:** `8 / 4 = 2`

**Pick heaviest on odd days (day 1):**
- `index = 7`, `pizzas[7] = 8`
- `totalWeight = 8`
- `index = 6`

**Pick second heaviest on even days:**
- `index = 5` (skip one: `index--`)
- Day 2: `pizzas[5] = 6`
- `totalWeight = 8 + 6 = 14`
- `index = 3` (decrement by 2)

**Output:** `14` 

---

## Example 2:
`pizzas = [2,1,1,1,1,1,1,1]`

**After sorting:**
`[1,1,1,1,1,1,1,2]`

**Number of days:** `8 / 4 = 2`

**Pick heaviest on odd days (day 1):**
- `index = 7`, `pizzas[7] = 2`
- `totalWeight = 2`
- `index = 6`

**Pick second heaviest on even days:**
- `index = 5` (skip one)
- Day 2: `pizzas[5] = 1`
- `totalWeight = 2 + 1 = 3`
- `index = 3` (decrement by 2)

**Output:** `3`

## Detailed walkthrough for Example 1:

| Day | Type | Heaviest (Z) | Second Heaviest (Y) | Lightest 2 | Weight Gained | Total |
|-----|------|--------------|---------------------|------------|---------------|-------|
| 1 | Odd | 8 | - | 2, 3 | 8 | 8 |
| 2 | Even | - | 6 | 1, 4 | 6 | 14 |

The strategy works because:
- On **odd days**, we maximize by taking the largest pizza.
- On **even days**, we take the second-largest available pizza.
- We always pair these with the two smallest remaining pizzas to minimize waste.
