[2864. Maximum Odd Binary Number](https://leetcode.com/problems/maximum-odd-binary-number/)

# Intuition
To create the **maximum odd binary number** from the given bits, the key is to ensure the number ends with `'1'` (since odd numbers end with `'1'`). To maximize the number, we should place the remaining `'1'`s as far to the left as possible, followed by `'0'`s, and finally place one `'1'` at the end.

---

# Approach
1. Count the total number of `'1'`s and `'0'`s in the string.
2. Place **all `'1'`s except one** at the beginning to maximize the number.
3. Append all `'0'`s next.
4. Append the **last `'1'`** at the end to ensure the number is odd.
5. The resulting string is the maximum odd binary number possible.

---

# Complexity
- **Time complexity:** 
  $$O(n)$$ - Counting characters and constructing the string.
- **Space complexity:** 
  $$O(n)$$ - For the output string.

---

# Code
```typescript
function maximumOddBinaryNumber(s: string): string {
    let countOnes = 0;
    let countZeros = 0;

    // Count number of '1's and '0's
    for (const ch of s) {
        if (ch === '1') {
            countOnes++;
        } else {
            countZeros++;
        }
    }

    // Build the maximum odd binary number
    // Place all '1's except one at the start
    let result = '1'.repeat(countOnes - 1);
    // Append all zeros
    result += '0'.repeat(countZeros);
    // Append the last '1' to make the number odd
    result += '1';

    return result;
};

```

# Example walkthrough

## Example 1:
`s = "010"` 
- Count `'1'`: 1 
- Count `'0'`: 2 

**Constructed number:**
- All `'1'`s except one: none (since only one `'1'`)
- Append zeros: `"00"` 
- Append `'1'` at the end: `"001"` 

**Output:** `"001"` 

## Example 2:
`s = "0101"` 
- Count `'1'`: 2 
- Count `'0'`: 2 

**Constructed number:**
- `'1'`s except one: `"1"` 
- Zeros: `"00"` 
- Last `'1'`: `"1"` 

**Result:** `"1001"`
