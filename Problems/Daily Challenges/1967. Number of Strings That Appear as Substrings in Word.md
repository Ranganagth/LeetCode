[1967. Number of Strings That Appear as Substrings in Word](https://leetcode.com/problems/number-of-strings-that-appear-as-substrings-in-word/)

# Intuition

The problem is straightforward: for each pattern in `patterns`, check if it exists as a substring in `word`. Count how many patterns are substrings.

Since the constraints are small (`patterns.length <= 100`, each pattern and word length <= 100), a simple approach of checking substring existence using `includes()` is efficient enough.

---

# Approach

1. Initialize a counter to zero.
2. Loop through each pattern in `patterns`.
3. For each pattern, check if `word.includes(pattern)`.
4. If yes, increment the counter.
5. Return the counter.

---

# Code

```typescript
function numOfStrings(patterns: string[], word: string): number {
    let count = 0;
    for (const pattern of patterns) {
        if (word.includes(pattern)) {
            count++;
        }
    }
    return count;
};

```

---

# Example

```typescript
console.log(numOfStrings(["a","abc","bc","d"], "abc")); // Output: 3
console.log(numOfStrings(["a","b","c"], "aaaaabbbbb")); // Output: 2
console.log(numOfStrings(["a","a","a"], "ab")); // Output: 3
```

This solution is simple, clear, and efficient given the problem constraints.
