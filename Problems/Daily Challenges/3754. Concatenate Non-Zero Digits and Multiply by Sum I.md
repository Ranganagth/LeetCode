[3754. Concatenate Non-Zero Digits and Multiply by Sum I](https://leetcode.com/problems/concatenate-non-zero-digits-and-multiply-by-sum-i/)

# Intuition

The problem requires us to:
1. Extract all non-zero digits from `n` and form a new integer `x`.
2. Calculate the sum of digits in `x`.
3. Return `x * sum`.

**Approach:**
- Convert `n` to a string to easily iterate through digits.
- Filter out zeros and concatenate non-zero digits to form `x`.
- Calculate the sum of digits in `x`.
- Return the product.

---

# Approach

1. Convert `n` to a string.
2. Iterate through each character (digit):
   - If the digit is not '0', add it to the string representation of `x`.
   - Add the digit value to the sum.
3. Convert the string representation of `x` to an integer.
4. Return `x * sum`.

---

# Complexity

- **Time complexity:** 
  $$O(d)$$, where `d` is the number of digits in `n` (at most 10 for `n <= 10^9`).

- **Space complexity:** 
  $$O(d)$$ for the string representation of `x`.

---

# Implementation

```typescript
function sumAndMultiply(n: number): number {
    const str = n.toString();
    let xStr = '';
    let sum = 0;

    for (const digit of str) {
        if (digit !== '0') {
            xStr += digit;
            sum += parseInt(digit);
        }
    }

    // If no non-zero digits, x = 0
    const x = xStr === '' ? 0 : parseInt(xStr);

    return x * sum;
};

```

# Alternative Implementation (More Concise)

```typescript
function sumAndMultiply(n: number): number {
    const nonZeroDigits = n
        .toString()
        .split('')
        .filter(d => d !== '0');

    if (nonZeroDigits.length === 0) {
        return 0;
    }

    const x = parseInt(nonZeroDigits.join(''));
    const sum = nonZeroDigits.reduce((acc, d) => acc + parseInt(d), 0);

    return x * sum;
};

```
