[1345. Jump Game IV](https://leetcode.com/problems/jump-game-iv/)

# Intuition

Each index is a node in a graph.

Possible edges from index `i`:

* `i + 1`
* `i - 1`
* Any index `j` where:

  ```
  arr[i] == arr[j]
  ```

Need minimum steps → shortest path in an unweighted graph → **BFS**

Optimization:
After processing all indices with same value, clear them to avoid revisiting:

```
map[value] = []
```

Otherwise worst-case:

```
[7,7,7,7,...]
```

becomes $(O(n^2))$.

---

# Approach

### Step 1: Build map

Store:

```
value → list of indices
```

### Step 2: BFS

Start:

```
queue = [0]
visited[0] = true
```

For each index:
Explore:

* `i-1`
* `i+1`
* all equal-value indices

### Step 3: Stop

Reach:

```
n - 1
```

Return steps.

---

# Complexity

* **Time complexity:**
  $$O(n)$$

Every index/value processed once.

* **Space complexity:**
  $$O(n)$$

---

# Code

```typescript
function minJumps(arr: number[]): number {
    const n = arr.length;

    if (n === 1) return 0;

    const map = new Map<number, number[]>();

    // value -> indices
    for (let i = 0; i < n; i++) {
        if (!map.has(arr[i])) {
            map.set(arr[i], []);
        }
        map.get(arr[i])!.push(i);
    }

    const visited = new Array(n).fill(false);
    visited[0] = true;

    let queue = [0];
    let steps = 0;

    while (queue.length > 0) {
        const next: number[] = [];

        for (const idx of queue) {
            if (idx === n - 1) {
                return steps;
            }

            const neighbors = [
                idx - 1,
                idx + 1,
                ...(map.get(arr[idx]) || [])
            ];

            for (const nei of neighbors) {
                if (
                    nei >= 0 &&
                    nei < n &&
                    !visited[nei]
                ) {
                    visited[nei] = true;
                    next.push(nei);
                }
            }

            // clear processed equal-value jumps
            map.set(arr[idx], []);
        }

        queue = next;
        steps++;
    }

    return -1;
};

```
