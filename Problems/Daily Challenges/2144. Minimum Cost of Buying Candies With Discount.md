[2144. Minimum Cost of Buying Candies With Discount](https://leetcode.com/problems/minimum-cost-of-buying-candies-with-discount/)

# Intuition

To minimize the total cost, we should maximize the benefit of the "buy two, get one free" offer. The best strategy is to:
- Sort the candies in descending order (most expensive first).
- Buy the two most expensive candies, then take the next cheapest candy for free (if it satisfies the condition).
- Repeat this process, always buying the two most expensive remaining candies and taking the next affordable candy for free.

This way, we ensure the free candies are as cheap as possible, maximizing savings.

---

# Approach

1. Sort the `cost` array in descending order.
2. Initialize total cost to 0.
3. Iterate through the sorted array:
   - For every group of three candies, buy the first two and skip the third (free).
   - Only add the cost of candies that are not free.
4. Return the accumulated total cost.

---

# Complexity

- **Time complexity:**  $$O(n \log n)$$ for sorting, where \( n \) is the number of candies.

- **Space complexity:**  $$O(1)$$ or $$O(n)$$ depending on sorting implementation.

---

# Code

```typescript
function minimumCost(cost: number[]): number {
    // Sort candies in descending order
    cost.sort((a, b) => b - a);
    let total = 0;

    for (let i = 0; i < cost.length; i++) {
        // For every third candy, skip adding its cost
        if ((i + 1) % 3 === 0) continue;
        total += cost[i];
    }

    return total;
};

```

# Example walkthrough

## Example 1:
`cost = [1, 2, 3]`

- Sorted: `[3, 2, 1]`
- Buy 3 and 2, take 1 for free:
  - Total = 3 + 2 = 5
- Output: `5`

## Example 2:
`cost = [6, 5, 7, 9, 2, 2]`

- Sorted: `[9, 7, 6, 5, 2, 2]`
- Process:
  - Buy 9, buy 7, skip 6 (free)
  - Buy 5, buy 2, skip 2 (free)
  - Total = 9 + 7 + 5 + 2 = 23
- Output: `23`
