[1391. Check if There is a Valid Path in a Grid](https://leetcode.com/problems/check-if-there-is-a-valid-path-in-a-grid/)

# Intuition

Each cell has fixed **entry/exit directions**.
A move is valid only if:

* Current cell allows movement in that direction
* Neighbor cell allows movement from the opposite direction

This is a **graph traversal with constraints**.

---

# Approach

### Step 1: Define directions

For each street type, define allowed directions:

```
1 → left, right
2 → up, down
3 → left, down
4 → right, down
5 → left, up
6 → right, up
```

### Step 2: BFS / DFS

Start from `(0,0)`:

* Try all allowed directions
* Check neighbor:

  * Inside grid
  * Has matching opposite direction
* Visit if valid

### Step 3: Stop condition

If `(m-1, n-1)` is reached → return true

---

# Complexity

* **Time complexity:**  $$O(m \cdot n)$$

* **Space complexity:**  $$O(m \cdot n)$$

---

# Code

```typescript
function hasValidPath(grid: number[][]): boolean {
    const m = grid.length, n = grid[0].length;

    const dirs: Record<number, number[][]> = {
        1: [[0, -1], [0, 1]],
        2: [[-1, 0], [1, 0]],
        3: [[0, -1], [1, 0]],
        4: [[0, 1], [1, 0]],
        5: [[0, -1], [-1, 0]],
        6: [[0, 1], [-1, 0]],
    };

    const visited = Array.from({ length: m }, () => Array(n).fill(false));

    const queue: [number, number][] = [[0, 0]];
    visited[0][0] = true;

    while (queue.length > 0) {
        const [x, y] = queue.shift()!;

        if (x === m - 1 && y === n - 1) return true;

        for (const [dx, dy] of dirs[grid[x][y]]) {
            const nx = x + dx, ny = y + dy;

            if (nx < 0 || ny < 0 || nx >= m || ny >= n) continue;
            if (visited[nx][ny]) continue;

            // check reverse direction
            for (const [rdx, rdy] of dirs[grid[nx][ny]]) {
                if (nx + rdx === x && ny + rdy === y) {
                    visited[nx][ny] = true;
                    queue.push([nx, ny]);
                    break;
                }
            }
        }
    }

    return false;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
[[2,4,3],
 [6,5,2]]
```

Path:

* (0,0) → down → (1,0)
* (1,0) → right → (1,1)
* (1,1) → up → (0,1)
* (0,1) → right → (0,2)
* (0,2) → down → (1,2)

Reached target → true

---

## Example 2:

Input:

```
[[1,2,1],
 [1,2,1]]
```

Start (0,0):

* only left/right → no valid connection → stuck

Return false

```
[[1,1,2]]
```
