[2540. Minimum Common Value](https://leetcode.com/problems/minimum-common-value/)

# Intuition 

Since both arrays are **sorted in non-decreasing order**, we can efficiently find the smallest common element by traversing both arrays simultaneously, similar to the merge step in merge sort. This avoids unnecessary comparisons and leverages the sorted property.

---

# Approach 

- **Use two pointers**, one for each array (`i` for `nums1`, `j` for `nums2`).
- Compare `nums1[i]` and `nums2[j]`:
  - If they are equal, we've found the smallest common element (since arrays are sorted), so return it.
  - If `nums1[i] < nums2[j]`, increment `i` to move to a larger value in `nums1`.
  - If `nums1[i] > nums2[j]`, increment `j` to move to a larger value in `nums2`.
- If we reach the end of either array without finding a match, return `-1`.

---

# Complexity 

- **Time complexity:** 
  $$O(n + m)$$ where $n$ and $m$ are the lengths of `nums1` and `nums2` respectively. Each pointer moves at most once per element.

- **Space complexity:** 
  $$O(1)$$ - Only a constant amount of extra space is used for pointers.

---

# Code

```typescript
function getCommon(nums1: number[], nums2: number[]): number {
    let i = 0, j = 0;
    while (i < nums1.length && j < nums2.length) {
        if (nums1[i] === nums2[j]) {
            return nums1[i];
        } else if (nums1[i] < nums2[j]) {
            i++;
        } else {
            j++;
        }
    }
    return -1;
};

```

# Example walkthrough

## Example 1:
**Input:** 
`nums1 = [1,2,3]`, `nums2 = [2,4]`

**Walkthrough:** 
- i = 0, j = 0: nums1[0]=1, nums2[0]=2 → 1 < 2, increment i
- i = 1, j = 0: nums1[1]=2, nums2[0]=2 → 2 == 2, **return 2**

## Example 2:
**Input:** 
`nums1 = [1,2,3,6]`, `nums2 = [2,3,4,5]`

**Walkthrough:** 
- i = 0, j = 0: nums1[0]=1, nums2[0]=2 → 1 < 2, increment i
- i = 1, j = 0: nums1[1]=2, nums2[0]=2 → 2 == 2, **return 2**

---

> If there is no common element (e.g., `nums1 = [1,3,5]`, `nums2 = [2,4,6]`), the pointers reach the end and the function returns `-1`.
