[3637. Trionic Array I](https://leetcode.com/problems/trionic-array-i/)

# Intuition

A **trionic array** has **three consecutive phases**:

1. **Strictly increasing**
2. **Strictly decreasing**
3. **Strictly increasing again**

These phases must appear **in order**, and each phase must contain **at least one valid comparison**, which is why we need:

```
0 < p < q < n − 1
```

So the array must:

* Go **up**
* Then **down**
* Then **up again**

If at any point this pattern breaks, the array is **not trionic**.

---

# Approach

We simulate the three phases using a single pass and a pointer `i`.

### Step 1: Increasing phase

Move forward while `nums[i] > nums[i - 1]`.

* If this phase ends immediately → invalid (no increasing part)

### Step 2: Decreasing phase

Move forward while `nums[i] < nums[i - 1]`.

* If this phase ends immediately → invalid (no decreasing part)

### Step 3: Final increasing phase

Move forward while `nums[i] > nums[i - 1]`.

Finally:

* We must **reach the end of the array**
* All three phases must exist

## Why this works

* Strict comparisons ensure correctness
* Single pass ensures efficiency
* Explicit phase validation avoids edge cases

---

# Complexity

* **Time:** `O(n)`
* **Space:** `O(1)`

---

# Code

```ts
function isTrionic(nums: number[]): boolean {
    const n = nums.length;
    if (n < 4) return false;

    let i = 1;

    // Phase 1: strictly increasing
    while (i < n && nums[i] > nums[i - 1]) {
        i++;
    }
    if (i === 1) return false; // no increasing part

    // Phase 2: strictly decreasing
    while (i < n && nums[i] < nums[i - 1]) {
        i++;
    }
    if (i === n || nums[i - 1] >= nums[i - 2]) return false;

    // Phase 3: strictly increasing
    while (i < n && nums[i] > nums[i - 1]) {
        i++;
    }

    return i === n;
};

```

# Example Walkthrough

### Example 1

```
nums = [1, 3, 5, 4, 2, 6]
```

* Increasing: `1 → 3 → 5`
* Decreasing: `5 → 4 → 2`
* Increasing: `2 → 6`

All phases exist and cover the full array → **true**

### Example 2

```
nums = [2, 1, 3]
```

* Starts decreasing immediately
* No initial increasing phase

→ **false**

---

## Final Takeaway

A trionic array is a **three-phase monotonic sequence**:

> **up → down → up**

Once you model the problem as **state transitions**, the solution becomes clean, fast, and reliable.
