[2840. Check if Strings Can be Made Equal With Operations II](https://leetcode.com/problems/check-if-strings-can-be-made-equal-with-operations-ii/)

# Intuition

Swaps are only allowed between indices with even distance. That means:

* Characters at **even indices** can only move among even indices.
* Characters at **odd indices** can only move among odd indices.

So the problem reduces to checking whether:

* The multiset of characters at even positions in `s1` matches that in `s2`
* The multiset of characters at odd positions in `s1` matches that in `s2`

---

# Approach

1. Traverse both strings.
2. Maintain 4 frequency arrays:

   * even positions for `s1`
   * odd positions for `s1`
   * even positions for `s2`
   * odd positions for `s2`
3. For each index `i`:

   * If `i % 2 == 0`: update even frequency
   * Else: update odd frequency
4. Compare:

   * even frequency arrays of `s1` and `s2`
   * odd frequency arrays of `s1` and `s2`
5. If both match → return `true`, else `false`

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(1)$$ (since only 26 lowercase letters)

---

# Code

```typescript [] id="gkq102"
function checkStrings(s1: string, s2: string): boolean {
    const n = s1.length;

    const even1 = new Array(26).fill(0);
    const odd1 = new Array(26).fill(0);
    const even2 = new Array(26).fill(0);
    const odd2 = new Array(26).fill(0);

    for (let i = 0; i < n; i++) {
        const idx1 = s1.charCodeAt(i) - 97;
        const idx2 = s2.charCodeAt(i) - 97;

        if (i % 2 === 0) {
            even1[idx1]++;
            even2[idx2]++;
        } else {
            odd1[idx1]++;
            odd2[idx2]++;
        }
    }

    for (let i = 0; i < 26; i++) {
        if (even1[i] !== even2[i] || odd1[i] !== odd2[i]) {
            return false;
        }
    }

    return true;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input: `s1 = "abcdba"`, `s2 = "cabdab"`

Even indices (0,2,4):

* s1 → a, c, b → counts: {a:1, b:1, c:1}
* s2 → c, b, a → counts: {a:1, b:1, c:1}

Odd indices (1,3,5):

* s1 → b, d, a → counts: {a:1, b:1, d:1}
* s2 → a, d, b → counts: {a:1, b:1, d:1}

**Both match** → `true`
