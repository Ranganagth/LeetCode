[2515. Shortest Distance to Target String in a Circular Array](https://leetcode.com/problems/shortest-distance-to-target-string-in-a-circular-array/)

# Intuition

The array is circular, so distance between two indices `i` and `j` is:
$$
\min(|i - j|,\ n - |i - j|)
$$

Check all indices where `words[i] == target` and compute the minimum circular distance.

---

# Approach

1. Let `n = words.length`
2. Initialize `ans = Infinity`
3. Traverse all indices:

   * If `words[i] == target`:

     * Compute:

       ```
       d = |i - startIndex|
       circularDist = min(d, n - d)
       ```
     * Update minimum
4. If no match found → return `-1`
5. Else return `ans`

---

# Complexity

* **Time complexity:**  $$O(n)$$

* **Space complexity:**  $$O(1)$$

---

# Code

```typescript
function closestTarget(words: string[], target: string, startIndex: number): number {
    const n = words.length;
    let ans = Infinity;

    for (let i = 0; i < n; i++) {
        if (words[i] === target) {
            const d = Math.abs(i - startIndex);
            const dist = Math.min(d, n - d);
            ans = Math.min(ans, dist);
        }
    }

    return ans === Infinity ? -1 : ans;
};

```
