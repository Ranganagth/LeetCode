[3612. Process String with Special Operations I](https://leetcode.com/problems/process-string-with-special-operations-i/)

# Intuition

We need to process the string from left to right, applying operations as we encounter each character. Since the string is short, we can directly manipulate the result string for each operation. The operations are simple: append, remove last, duplicate, and reverse.

---

# Approach

- Initialize an empty string `result`.
- For each character in the input string:
  - If it's a lowercase letter, append it to `result`.
  - If it's '*', remove the last character from `result` (if it exists).
  - If it's '#', duplicate the current `result` and append it to itself.
  - If it's '%', reverse the current `result`.
- Return the final `result` string.

---

# Complexity

- **Time complexity:**
  $$O(n^2)$$ (since duplicating or reversing the string can take up to O(n) time for each operation, and there are up to n operations)
  
- **Space complexity:**
  $$O(n)$$ (for the result string)

---

# Code

```typescript
function processStr(s: string): string {
    let result = "";

    for (const ch of s) {
        if (ch >= 'a' && ch <= 'z') {
            result += ch;
        } else if (ch === '*') {
            if (result.length > 0) {
                result = result.slice(0, -1);
            }
        } else if (ch === '#') {
            result += result;
        } else if (ch === '%') {
            result = result.split('').reverse().join('');
        }
    }

    return result;
};

```

# Example walkthrough

## Example 1:
**Input:** 
`s = "a#b%*"`

**Process/Approach explanation:** 
- 'a' → append → "a" 
- '#' → duplicate → "aa" 
- 'b' → append → "aab" 
- '%' → reverse → "baa" 
- '*' → remove last → "ba"

**Output:** 
`"ba"`

---

## Example 2:
**Input:** 
`s = "z*#"`

**Process/Approach explanation:** 
- 'z' → append → "z" 
- '*' → remove last → "" 
- '#' → duplicate → ""

**Output:** 
`""`
