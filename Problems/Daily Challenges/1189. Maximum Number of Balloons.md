[1189. Maximum Number of Balloons](https://leetcode.com/problems/maximum-number-of-balloons/)

# Intuition

To determine how many times we can form the word "balloon" from the characters in `text`, we need to count the frequency of each character in `text` and compare it to the frequency needed for "balloon". Since "balloon" contains:
- 'b' once
- 'a' once
- 'l' twice
- 'o' twice
- 'n' once

The maximum number of "balloon" instances is limited by the minimum ratio of available characters to their required counts.

---

# Approach

1. Count the frequency of each character in `text`.
2. For each character in "balloon", determine how many times it can contribute to forming "balloon" based on its count in `text`.
3. The answer is the minimum of these ratios, since all characters are needed in specific counts.

---

# Complexity

- **Time complexity:**  $$O(n)$$ where n is the length of `text` (counting characters).
  
- **Space complexity:**  $$O(1)$$, since the character count array is fixed size (26 lowercase letters).

---

# Code

```typescript
function maxNumberOfBalloons(text: string): number {
    const count = new Array(26).fill(0);
    for (const ch of text) {
        count[ch.charCodeAt(0) - 'a'.charCodeAt(0)]++;
    }

    const targetCounts = {
        'b': 1,
        'a': 1,
        'l': 2,
        'o': 2,
        'n': 1
    };

    // Calculate how many times we can form "balloon"
    const ratios = [
        Math.floor(count['b'.charCodeAt(0) - 'a'.charCodeAt(0)] / targetCounts['b']),
        Math.floor(count['a'.charCodeAt(0) - 'a'.charCodeAt(0)] / targetCounts['a']),
        Math.floor(count['l'.charCodeAt(0) - 'a'.charCodeAt(0)] / targetCounts['l']),
        Math.floor(count['o'.charCodeAt(0) - 'a'.charCodeAt(0)] / targetCounts['o']),
        Math.floor(count['n'.charCodeAt(0) - 'a'.charCodeAt(0)] / targetCounts['n'])
    ];

    return Math.min(...ratios);
};

```
