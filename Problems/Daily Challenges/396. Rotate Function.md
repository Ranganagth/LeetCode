[396. Rotate Function](https://leetcode.com/problems/rotate-function/)

# Intuition

Brute rotation is $(O(n^2))$.

Key relation between consecutive rotations:
$$
F(k) = F(k-1) + \text{totalSum} - n \cdot nums[n-k]
$$

Reason:

* Every element shifts right → contributes +sum
* Last element moves to front → subtract ($n \cdot nums[n-k]$)

---

# Approach

1. Compute:

   * `totalSum = sum(nums)`
   * `F(0)`
2. Iterate `k = 1 → n-1`:
$$   
   F(k) = F(k-1) + totalSum - n * nums[n-k]
 $$
3. Track maximum

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(1)$$

---

# Code

```typescript
function maxRotateFunction(nums: number[]): number {
    const n = nums.length;

    let total = 0;
    let F = 0;

    for (let i = 0; i < n; i++) {
        total += nums[i];
        F += i * nums[i];
    }

    let ans = F;

    for (let k = 1; k < n; k++) {
        F = F + total - n * nums[n - k];
        ans = Math.max(ans, F);
    }

    return ans;
}
```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
[4,3,2,6]
```

Compute:

* total = 15
* F(0) = 25

k=1:
$$
F(1) = 25 + 15 - 4*6 = 16
$$

k=2:
$$
F(2) = 16 + 15 - 4*2 = 23
$$

k=3:
$$
F(3) = 23 + 15 - 4*3 = 26
$$

Max = 26
