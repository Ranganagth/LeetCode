[2452. Words Within Two Edits of Dictionary](https://leetcode.com/problems/words-within-two-edits-of-dictionary/)

# Intuition

A query matches if it differs from any dictionary word in **at most 2 positions**.
This is equivalent to checking **Hamming distance ≤ 2** between strings.

---

# Approach

1. For each query:

   * Compare with every dictionary word
2. For each comparison:

   * Count number of differing characters
   * Stop early if differences exceed 2
3. If any dictionary word matches → include query in result

---

# Complexity

* **Time complexity:**
  $$O(q \cdot d \cdot n)$$
  where:

  * q = queries length
  * d = dictionary length
  * n = word length

* **Space complexity:**
  $$O(1)$$

---

# Code

```typescript
function twoEditWords(queries: string[], dictionary: string[]): string[] {
    const res: string[] = [];

    for (const q of queries) {
        let found = false;

        for (const d of dictionary) {
            let diff = 0;

            for (let i = 0; i < q.length; i++) {
                if (q[i] !== d[i]) {
                    diff++;
                    if (diff > 2) break;
                }
            }

            if (diff <= 2) {
                found = true;
                break;
            }
        }

        if (found) res.push(q);
    }

    return res;
}
```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
queries = ["word","note","ants","wood"]
dictionary = ["wood","joke","moat"]
```

* "word" vs "wood" → 1 diff → valid
* "note" vs "joke" → 2 diffs → valid
* "ants" vs all → >2 diffs → invalid
* "wood" vs "wood" → 0 diff → valid

Output:

```
["word","note","wood"]
```
