[3070. Count Submatrices with Top-Left Element and Sum Less Than k](https://leetcode.com/problems/count-submatrices-with-top-left-element-and-sum-less-than-k/)

# Intuition

Every valid submatrix must:

```
start at (0,0)
end at (i,j)
```

So each submatrix is uniquely identified by its **bottom-right corner**.

Total candidates:

```
m * n
```

Instead of recomputing sums repeatedly, use **2D prefix sum**.

Prefix sum gives:

```
sum(0,0 → i,j) in O(1)
```

So the problem reduces to:

```
count (i,j) such that prefix[i][j] ≤ k
```

---

# Approach

### Step 1 - Build Prefix Sum

Define:

```
prefix[i][j] = sum of rectangle (0,0) → (i,j)
```

Formula:

```
prefix[i][j] =
    grid[i][j]
  + prefix[i-1][j]
  + prefix[i][j-1]
  - prefix[i-1][j-1]
```

Handle boundaries carefully.

### Step 2 - Count Valid Submatrices

For every `(i,j)`:

```
if prefix[i][j] ≤ k
    count++
```

## Algorithm

1. Initialize prefix matrix.
2. Compute prefix sums.
3. Traverse all cells:

   * check condition
   * increment count.

---

# Complexity

- **Time Complexity:** $O(m * n)$

- **Space Complexity:** $O(m * n)$
	*Can be optimized to:* $O(n)$ by using rolling prefix sums.

---

# Code

```typescript 
function countSubmatrices(grid: number[][], k: number): number {

    const m = grid.length;
    const n = grid[0].length;

    const prefix: number[][] = Array.from({ length: m }, () =>
        new Array(n).fill(0)
    );

    let count = 0;

    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {

            prefix[i][j] = grid[i][j];

            if (i > 0) prefix[i][j] += prefix[i - 1][j];
            if (j > 0) prefix[i][j] += prefix[i][j - 1];
            if (i > 0 && j > 0) prefix[i][j] -= prefix[i - 1][j - 1];

            if (prefix[i][j] <= k) count++;
        }
    }

    return count;
}
```

# Example Walkthrough

### Example 1

```
grid =
[
[7,6,3],
[6,6,1]
]

k = 18
```

### Step 1 - Prefix Sum

Row 0:

```
[7, 13, 16]
```

Row 1:

```
[13, 25, 29]
```

### Step 2 - Check ≤ k

```
(0,0) → 7   ≤ 18 ✅
(0,1) → 13  ≤ 18 ✅
(0,2) → 16  ≤ 18 ✅
(1,0) → 13  ≤ 18 ✅
(1,1) → 25  > 18 ❌
(1,2) → 29  > 18 ❌
```

Count:

```
4
```
