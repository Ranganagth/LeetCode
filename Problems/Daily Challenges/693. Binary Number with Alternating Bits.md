[693. Binary Number with Alternating Bits](https://leetcode.com/problems/binary-number-with-alternating-bits/)

# Intuition

If a number has alternating bits (e.g., `101010`), then XORing the number with itself right-shifted by one position produces a sequence of all `1`s:

```
n        = 101010
n >> 1   = 010101
n ^ (n>>1)=111111
```

A number consisting only of `1`s satisfies:

```
x & (x + 1) == 0
```

Thus:

```
x = n ^ (n >> 1)
return (x & (x + 1)) == 0
```

---

# Approach

1. Compute `x = n ^ (n >> 1)`.
2. Check if `x` is of the form `111...1` using `x & (x + 1) == 0`.
3. If true → alternating bits exist.

---

# Complexity

* **Time:** `O(1)`
* **Space:** `O(1)`

---

# Code

```typescript
function hasAlternatingBits(n: number): boolean {
    const x = n ^ (n >> 1);
    return (x & (x + 1)) === 0;
}
```

# Example Walkthrough

### Example 1

```
n = 5
binary = 101
```

```
n >> 1 = 010
x = 101 ^ 010 = 111
x & (x+1) = 111 & 1000 = 0
→ true
```

### Example 2

```
n = 7
binary = 111
```

```
n >> 1 = 011
x = 111 ^ 011 = 100
x & (x+1) = 100 & 101 ≠ 0
→ false
```
