[2839. Check if Strings Can be Made Equal With Operations I](https://leetcode.com/problems/check-if-strings-can-be-made-equal-with-operations-i/)

# Intuition

Swaps are only allowed between indices with distance 2:

* Index 0 ↔ 2
* Index 1 ↔ 3

This splits the string into two independent groups:

* Group 1: indices {0, 2}
* Group 2: indices {1, 3}

Within each group, characters can be freely swapped → order does not matter, only multiset matters.

---

# Approach

1. Extract two groups from both strings:

   * Even group → indices (0, 2)
   * Odd group → indices (1, 3)
2. Sort both groups for `s1` and `s2`
3. Compare:

   * sorted(s1 even) == sorted(s2 even)
   * sorted(s1 odd) == sorted(s2 odd)
4. If both match → return true, else false

---

# Complexity

* **Time complexity:**
  $$O(1)$$ (fixed size 4)

* **Space complexity:**
  $$O(1)$$

---

# Code

```typescript [] id="eq442"
function canBeEqual(s1: string, s2: string): boolean {
    const s1Even = [s1[0], s1[2]].sort().join('');
    const s1Odd = [s1[1], s1[3]].sort().join('');

    const s2Even = [s2[0], s2[2]].sort().join('');
    const s2Odd = [s2[1], s2[3]].sort().join('');

    return s1Even === s2Even && s1Odd === s2Odd;
};

```
