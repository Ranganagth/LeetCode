[3783. Mirror Distance of an Integer](https://leetcode.com/problems/mirror-distance-of-an-integer/)

# Intuition

Reverse the digits of `n`, then compute absolute difference:
$$
|n - reverse(n)|
$$
---

# Approach

1. Reverse the number:

   * Extract digits using `% 10`
   * Build reversed number
2. Compute:

   ```
   Math.abs(n - reversed)
   ```
3. Return result

---

# Complexity

* **Time complexity:**
  $$O(d)$$ (d = number of digits ≤ 10)

* **Space complexity:**
  $$O(1)$$

---

# Code

```typescript
function mirrorDistance(n: number): number {
    let original = n;
    let reversed = 0;

    while (n > 0) {
        reversed = reversed * 10 + (n % 10);
        n = Math.floor(n / 10);
    }

    return Math.abs(original - reversed);
};

```
