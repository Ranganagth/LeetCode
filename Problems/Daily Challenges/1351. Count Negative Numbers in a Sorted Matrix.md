[1351. Count Negative Numbers in a Sorted Matrix](https://leetcode.com/problems/count-negative-numbers-in-a-sorted-matrix/)

# Intuition

Rows and columns are both sorted **non-increasing** (left → right decreases, top → bottom decreases).
So if a cell is negative, then **everything below it in that column is also negative**.
Likewise, if a cell is non-negative, then **everything left of it in that row is also non-negative**.

Exploit this monotonic structure instead of checking every cell.

---

# Approach (O(m + n))

Start at the **top-right corner**:

* If `grid[row][col] < 0`
  then every element **below** it (same column) is negative.
  Count all of them at once: `rows - row`, then move **left** (`col--`).

* Otherwise (`>= 0`)
  everything **left** is also non-negative, so move **down** (`row++`).

Stop once we fall outside the grid. Each move advances either a row or column exactly once → linear in `m+n`.

---

# Complexity

* **Time:** `O(m + n)`
* **Space:** `O(1)`

---

# Code

```ts
function countNegatives(grid: number[][]): number {
    let rows = grid.length;
    let cols = grid[0].length;
    let row = 0;
    let col = cols - 1;
    let count = 0;

    while (row < rows && col >= 0) {
        if (grid[row][col] < 0) {
            count += rows - row; // all below are negative
            col--;               // move left
        } else {
            row++;               // move down
        }
    }

    return count;
};

```

# Example walkthrough

`grid = [[4,3,2,-1],          [3,2,1,-1],          [1,1,-1,-2],          [-1,-1,-2,-3]]`

Start `(row=0,col=3)`:

* `-1 < 0` → add `4 - 0 = 4`, move left → col=2
* `2 >= 0` → move down → row=1
* `1 >= 0` → move down → row=2
* `-1 < 0` → add `4 - 2 = 2`, move left → col=1
* `1 >= 0` → move down → row=3
* `-1 < 0` → add `4 - 3 = 1`, move left → col=0
* `-1 < 0` → add `4 - 3 = 1`, move left → col=-1 stop

Total = **8**
