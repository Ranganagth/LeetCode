[1674. Minimum Moves to Make Array Complementary](https://leetcode.com/problems/minimum-moves-to-make-array-complementary/)

# Intuition

For every mirrored pair:
$$
(a, b)
$$
We want all pairs to sum to the same value `S`.

For one pair:

* 0 moves if:

  $$a + b = S
$$
* 1 move if `S` can be formed by changing one number
* 2 moves otherwise

Instead of checking every sum for every pair directly, use a **difference array sweep line**.

---

# Approach

For pair `(a, b)`:

```
low = min(a,b)
high = max(a,b)
```

Possible costs for target sum `S`:

### 2 moves initially

Every sum starts with cost `2`.

### 1 move range

With one replacement:

```
[low + 1, high + limit]
```

cost becomes `1`.

### 0 move point

At:

```
a + b
```

cost becomes `0`.

### Difference array updates

For each pair:

```
diff[2] += 2

diff[low + 1] -= 1
diff[high + limit + 1] += 1

diff[a + b] -= 1
diff[a + b + 1] += 1

diff[a + b + 1] += 1
diff[a + b + 1] -= 1
```

Simplified standard form:

* Convert ranges into prefix updates

### Final sweep

Prefix sum over all target sums:

$$2 → 2*limit
$$
Minimum prefix value = answer.

---

# Complexity

* **Time complexity:**
  $$O(n + limit)$$

* **Space complexity:**
  $$O(limit)$$

---

# Code

```typescript
function minMoves(nums: number[], limit: number): number {
    const n = nums.length;

    const diff = new Array(2 * limit + 2).fill(0);

    for (let i = 0; i < n / 2; i++) {
        const a = nums[i];
        const b = nums[n - 1 - i];

        const low = Math.min(a, b);
        const high = Math.max(a, b);

        const sum = a + b;

        // initially 2 moves
        diff[2] += 2;

        // one move range starts
        diff[low + 1] -= 1;

        // zero move point
        diff[sum] -= 1;
        diff[sum + 1] += 1;

        // back to 2 moves after one-move range
        diff[high + limit + 1] += 1;
    }

    let ans = Infinity;
    let curr = 0;

    for (let s = 2; s <= 2 * limit; s++) {
        curr += diff[s];
        ans = Math.min(ans, curr);
    }

    return ans;
};

```
