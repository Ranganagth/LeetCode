# Intuition

The problem is straightforward: find the minimum and maximum values in the array, then compute their GCD. Since we only need the min and max values, we can find them in a single pass through the array.

---

# Approach

1. **Find minimum and maximum:** Iterate through the array once to find both the smallest and largest values

2. **Compute GCD:** Use the Euclidean algorithm to calculate GCD(min, max)

3. **Euclidean Algorithm:** 
   - GCD(a, b) = GCD(b, a % b)
   - Base case: GCD(a, 0) = a

---

# Complexity

- **Time complexity:** O(n + log(min(min, max)))
  - Finding min/max: O(n)
  - GCD computation: O(log(min(min, max)))

- **Space complexity:** O(1) - only using a few variables

---

# Code

## Solution 1: Using Math.min/Math.max

```typescript
function findGCD(nums: number[]): number {
    // Find minimum and maximum values
    const minVal = Math.min(...nums);
    const maxVal = Math.max(...nums);
    
    // Helper function to compute GCD using Euclidean algorithm
    const gcd = (a: number, b: number): number => {
        if (b === 0) return a;
        return gcd(b, a % b);
    };
    
    return gcd(minVal, maxVal);
};

```

## Solution 2: Single Pass (More Efficient)

```typescript
function findGCD(nums: number[]): number {
    // Find min and max in a single pass
    let minVal = nums[0];
    let maxVal = nums[0];
    
    for (let i = 1; i < nums.length; i++) {
        if (nums[i] < minVal) {
            minVal = nums[i];
        }
        if (nums[i] > maxVal) {
            maxVal = nums[i];
        }
    }
    
    // Helper function to compute GCD
    const gcd = (a: number, b: number): number => {
        if (b === 0) return a;
        return gcd(b, a % b);
    };
    
    return gcd(minVal, maxVal);
};

```

---

## Solution 3: Iterative GCD (Avoid Recursion)

```typescript
function findGCD(nums: number[]): number {
    // Find min and max
    let minVal = nums[0];
    let maxVal = nums[0];
    
    for (const num of nums) {
        minVal = Math.min(minVal, num);
        maxVal = Math.max(maxVal, num);
    }
    
    // Iterative GCD computation
    let a = maxVal;
    let b = minVal;
    
    while (b !== 0) {
        const temp = b;
        b = a % b;
        a = temp;
    }
    
    return a;
};

```

## Key Insights

- **Single pass optimization:** Find min and max in one iteration instead of two separate passes
- **Euclidean algorithm:** Efficient GCD computation in O(log(min(a, b))) time
- **Iterative vs Recursive:** Iterative version avoids potential stack overflow for large numbers
- **Math.min/max with spread:** Convenient but creates temporary array (less efficient for large arrays)

## Edge Cases

| Case | Input | Min | Max | GCD | Notes |
|------|-------|-----|-----|-----|-------|
| Two elements | [6, 9] | 6 | 9 | 3 | Minimal array |
| All same | [5, 5, 5] | 5 | 5 | 5 | Min = Max |
| Coprime | [7, 11] | 7 | 11 | 1 | No common factors |
| Powers of 2 | [2, 4, 8, 16] | 2 | 16 | 2 | Common factor |
| Large range | [1, 1000] | 1 | 1000 | 1 | Max difference |
