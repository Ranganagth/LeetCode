[3093. Longest Common Suffix Queries](https://leetcode.com/problems/longest-common-suffix-queries/)

# Intuition

To efficiently find the string in `wordsContainer` with the **longest common suffix** for each query, we can use a **Trie (prefix tree)**, but built on reversed strings (so suffixes become prefixes). 
Each Trie node keeps track of the best candidate index (shortest length, earliest occurrence) for that suffix.

---

# Approach

1. **Build a Trie on reversed wordsContainer strings:**
   - For each word, insert its reversed form into the Trie.
   - At each node, store the index of the best candidate (shortest length, earliest index).
2. **For each query:**
   - Reverse the query string.
   - Traverse the Trie as far as possible, tracking the deepest node reached.
   - The node reached gives the index of the best matching string.
3. **If no suffix matches, return the best overall candidate (shortest, earliest).**

---

# Complexity

- **Time complexity:** 
  $$O(\text{sum of all chars in wordsContainer} + \text{sum of all chars in wordsQuery})$$
- **Space complexity:** 
  $$O(\text{sum of all chars in wordsContainer})$$ for the Trie.

---

# Code

```typescript
class TrieNode {
    children: Map<string, TrieNode>;
    bestIdx: number; // index of best candidate for this suffix

    constructor() {
        this.children = new Map();
        this.bestIdx = -1;
    }
}

function stringIndices(wordsContainer: string[], wordsQuery: string[]): number[] {
    // Find the best overall candidate (shortest, earliest)
    let bestIdx = 0;
    for (let i = 1; i < wordsContainer.length; i++) {
        if (
            wordsContainer[i].length < wordsContainer[bestIdx].length ||
            (wordsContainer[i].length === wordsContainer[bestIdx].length && i < bestIdx)
        ) {
            bestIdx = i;
        }
    }

    // Build Trie
    const root = new TrieNode();
    for (let i = 0; i < wordsContainer.length; i++) {
        let node = root;
        const word = wordsContainer[i];
        for (let j = word.length - 1; j >= 0; j--) {
            const c = word[j];
            if (!node.children.has(c)) {
                node.children.set(c, new TrieNode());
            }
            node = node.children.get(c)!;
            // Update best index at this node
            if (
                node.bestIdx === -1 ||
                wordsContainer[i].length < wordsContainer[node.bestIdx].length ||
                (wordsContainer[i].length === wordsContainer[node.bestIdx].length && i < node.bestIdx)
            ) {
                node.bestIdx = i;
            }
        }
    }

    // Answer queries
    const res: number[] = [];
    for (const query of wordsQuery) {
        let node = root;
        let answerIdx = bestIdx; // default to best overall
        for (let j = query.length - 1; j >= 0; j--) {
            const c = query[j];
            if (!node.children.has(c)) break;
            node = node.children.get(c)!;
            if (node.bestIdx !== -1) {
                answerIdx = node.bestIdx;
            }
        }
        res.push(answerIdx);
    }
    return res;
};

```

# Example walkthrough

## Example 1

**Input:** 
`wordsContainer = ["abcd","bcd","xbcd"]` 
`wordsQuery = ["cd","bcd","xyz"]`

- Trie is built on reversed: "dcba", "dcb", "dcbx"
- For "cd" (reversed "dc"):
  - Traverse 'd' → 'c', all three words match, best is index 1 (shortest length 3).
- For "bcd" (reversed "dcb"):
  - Traverse 'd' → 'c' → 'b', all three words match, best is index 1.
- For "xyz" (reversed "zyx"):
  - No match, return best overall (index 1).

**Output:** `[1, 1, 1]`

---

## Example 2

**Input:** 
`wordsContainer = ["abcdefgh","poiuygh","ghghgh"]` 
`wordsQuery = ["gh","acbfgh","acbfegh"]`

- Trie is built on reversed: "hgfedcba", "hgyuioip", "hghghg"
- For "gh" (reversed "hg"):
  - All three words match, best is index 2 (length 6).
- For "acbfgh" (reversed "hgfbcba"):
  - Only index 0 matches the suffix "fgh".
- For "acbfegh" (reversed "hgefbcba"):
  - All three words match "gh", best is index 2.

**Output:** `[2, 0, 2]`
