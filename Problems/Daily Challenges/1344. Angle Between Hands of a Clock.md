[1344. Angle Between Hands of a Clock](https://leetcode.com/problems/angle-between-hands-of-a-clock/description/?envType=daily-question&envId=2026-06-18)

# Intuition

To find the smaller angle between the hour and minute hands:
- Calculate the positions (in degrees) of both hands.
- The minute hand moves 6° per minute.
- The hour hand moves 30° per hour plus 0.5° per minute.
- The angle between them is the absolute difference, and the smaller angle is at most 180°.

---

# Approach

1. Compute the minute hand's angle: `minute_angle = minutes * 6`.
2. Compute the hour hand's angle: `hour_angle = (hour % 12) * 30 + minutes * 0.5`.
3. Find the absolute difference: `diff = Math.abs(hour_angle - minute_angle)`.
4. Return the smaller angle: `Math.min(diff, 360 - diff)`.

---

# Complexity

- **Time complexity:**
    $O(1)$ (constant time arithmetic)

- **Space complexity:**
    $O(1)$

---

# Code

```typescript
function angleClock(hour: number, minutes: number): number {
    // Calculate angles
    const minute_angle = minutes * 6;
    const hour_angle = (hour % 12) * 30 + minutes * 0.5;
    const diff = Math.abs(hour_angle - minute_angle);
    return Math.min(diff, 360 - diff);
};

```

# Example walkthrough

## Example 1:

**Input:** hour = 12, minutes = 30 
**Process:** 
- minute_angle = 30 * 6 = 180 
- hour_angle = (12 % 12) * 30 + 30 * 0.5 = 0 + 15 = 15 
- diff = |15 - 180| = 165 
- smaller angle = 165

**Output:** 165
