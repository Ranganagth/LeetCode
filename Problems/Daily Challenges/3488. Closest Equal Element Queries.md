[3488. Closest Equal Element Queries](https://leetcode.com/problems/closest-equal-element-queries/)

# Intuition

For each value, all its indices form a circular list.
The closest same value for any index must be either:

* Previous occurrence
* Next occurrence

Circular nature handled by extending indices:

* Add `(last - n)` at front
* Add `(first + n)` at end

Then distance reduces to **difference between neighbors**.

---

# Approach

### Step 1: Group indices

Map each value → sorted list of indices

### Step 2: Extend for circular handling

For each list:

```
[pos_k - n, pos_0, pos_1, ..., pos_k, pos_0 + n]
```

### Step 3: For each query

1. Get value `nums[q]`
2. If only one occurrence → return -1
3. Binary search to find position of `q`
4. Compute:

   ```
   min(next - current, current - prev)
   ```

---

# Complexity

* **Time complexity:**  $$O(n + q \log n)$$

* **Space complexity:**  $$O(n)$$

---

# Code

```typescript
function solveQueries(nums: number[], queries: number[]): number[] {
    const n = nums.length;
    const map = new Map<number, number[]>();

    // Step 1: group indices
    for (let i = 0; i < n; i++) {
        if (!map.has(nums[i])) map.set(nums[i], []);
        map.get(nums[i])!.push(i);
    }

    // Step 2: extend lists
    for (const pos of map.values()) {
        if (pos.length > 1) {
            const first = pos[0];
            const last = pos[pos.length - 1];
            pos.unshift(last - n);
            pos.push(first + n);
        }
    }

    const res: number[] = [];

    for (const q of queries) {
        const val = nums[q];
        const posList = map.get(val)!;

        if (posList.length <= 1) {
            res.push(-1);
            continue;
        }

        // binary search
        let l = 0, r = posList.length;
        while (l < r) {
            const mid = (l + r) >> 1;
            if (posList[mid] < q) l = mid + 1;
            else r = mid;
        }

        const dist = Math.min(
            posList[l] - posList[l - 1],
            posList[l + 1] - posList[l]
        );

        res.push(dist);
    }

    return res;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
nums = [1,3,1,4,1,3,2]
queries = [0,3,5]
```

Value 1 → indices `[0,2,4]` → extended `[-3,0,2,4,7]`

Query 0:

* neighbors: 2 and -3
* distances: 2, 3 → min = 2

Query 3:

* value 4 → only one → -1

Value 3 → `[1,5]` → extended `[-2,1,5,8]`

Query 5:

* neighbors: 1 and 8
* distances: 4, 3 → min = 3

Output: `[2,-1,3]`
