[3633. Earliest Finish Time for Land and Water Rides I](https://leetcode.com/problems/earliest-finish-time-for-land-and-water-rides-i/)

# Intuition

The key insight is that we need to try **all possible combinations** of rides from each category, not just the earliest ones. For each pair of rides (one land, one water), we calculate the finish time for both possible orders and track the minimum.

For a given pair of rides:
- **Order 1 (Land → Water):** Start land at its earliest time, finish land, then start water at max(water's earliest start, land's finish time).
- **Order 2 (Water → Land):** Start water at its earliest time, finish water, then start land at max(land's earliest start, water's finish time).

We need to check all combinations because different ride pairs might yield different optimal finish times.

---

# Approach

1. **Iterate through all land rides** and **all water rides** (all combinations).
2. For each combination `(land_i, water_j)`:
   - **Plan A (Land first):**
     - Start land at `landStartTime[i]`
     - Finish land at `landStartTime[i] + landDuration[i]`
     - Start water at `max(waterStartTime[j], finishLand)`
     - Finish water at `startWater + waterDuration[j]`
   - **Plan B (Water first):**
     - Start water at `waterStartTime[j]`
     - Finish water at `waterStartTime[j] + waterDuration[j]`
     - Start land at `max(landStartTime[i], finishWater)`
     - Finish land at `startLand + landDuration[i]`
3. Track the minimum finish time across all combinations and both orders.
4. Return the minimum.

---

# Complexity

- **Time complexity:** 
  $$O(n \times m)$$ where \( n \) is the number of land rides and \( m \) is the number of water rides.
  
- **Space complexity:** 
  $$O(1)$$


---

# Code

```typescript
function earliestFinishTime(
  landStartTime: number[],
  landDuration: number[],
  waterStartTime: number[],
  waterDuration: number[]
): number {
  let minFinishTime = Infinity;

  // Try all combinations of land and water rides
  for (let i = 0; i < landStartTime.length; i++) {
    for (let j = 0; j < waterStartTime.length; j++) {
      // Plan A: Land ride first, then water ride
      const startLand = landStartTime[i];
      const finishLand = startLand + landDuration[i];
      const startWater = Math.max(waterStartTime[j], finishLand);
      const finishWaterA = startWater + waterDuration[j];

      // Plan B: Water ride first, then land ride
      const startWater2 = waterStartTime[j];
      const finishWater = startWater2 + waterDuration[j];
      const startLand2 = Math.max(landStartTime[i], finishWater);
      const finishLandB = startLand2 + landDuration[i];

      // Track minimum finish time
      minFinishTime = Math.min(minFinishTime, finishWaterA, finishLandB);
    }
  }

  return minFinishTime;
};

```
