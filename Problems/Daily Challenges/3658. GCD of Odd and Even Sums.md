[3658. GCD of Odd and Even Sums](https://leetcode.com/problems/gcd-of-odd-and-even-sums/description/?envType=daily-question&envId=2026-07-15)


# Intuition

Instead of computing the sums by iteration, we can use **mathematical formulas**:

- **Sum of first n odd numbers**: 1 + 3 + 5 + ... + (2n-1) = **n²**
- **Sum of first n even numbers**: 2 + 4 + 6 + ... + 2n = **n(n+1)**

Then we need to find **GCD(n², n(n+1))**.

We can simplify this further using GCD properties!

---

# Approach

## Mathematical Derivation

1. **Sum formulas:**
   - sumOdd = 1 + 3 + 5 + ... + (2n-1) = n²
   - sumEven = 2 + 4 + 6 + ... + 2n = 2(1 + 2 + 3 + ... + n) = 2 × n(n+1)/2 = n(n+1)

2. **GCD simplification:**
   - GCD(n², n(n+1))
   - Factor out n: GCD(n², n(n+1)) = n × GCD(n, n+1)
   
3. **Key insight:** 
   - GCD(n, n+1) = 1 (consecutive integers are always coprime)
   - Therefore: GCD(n², n(n+1)) = **n × 1 = n**

4. **Direct answer:** The result is simply **n**!

---

# Complexity

- **Time complexity:** $$O(\log n)$$ - for GCD computation (though we can optimize to O(1))

- **Space complexity:** $$O(1)$$ - only storing a few variables

# Code

## Solution 1: Direct Mathematical Result

```typescript
function gcdOfOddEvenSums(n: number): number {
    // Sum of first n odd numbers = n²
    // Sum of first n even numbers = n(n+1)
    // GCD(n², n(n+1)) = n × GCD(n, n+1) = n × 1 = n
    return n;
};

```

## Solution 2: Explicit Calculation (for verification)

```typescript
function gcdOfOddEvenSums(n: number): number {
    // Calculate sums using formulas
    const sumOdd = n * n;
    const sumEven = n * (n + 1);
    
    // Helper function to compute GCD
    const gcd = (a: number, b: number): number => {
        if (b === 0) return a;
        return gcd(b, a % b);
    };
    
    return gcd(sumOdd, sumEven);
};

```

## Solution 3: Iterative Calculation (brute force)

```typescript
function gcdOfOddEvenSums(n: number): number {
    // Calculate sum of first n odd numbers
    let sumOdd = 0;
    for (let i = 1; i <= n; i++) {
        sumOdd += 2 * i - 1;
    }
    
    // Calculate sum of first n even numbers
    let sumEven = 0;
    for (let i = 1; i <= n; i++) {
        sumEven += 2 * i;
    }
    
    // Helper function to compute GCD
    const gcd = (a: number, b: number): number => {
        if (b === 0) return a;
        return gcd(b, a % b);
    };
    
    return gcd(sumOdd, sumEven);
};

```

## Mathematical Proof

### Why GCD(n², n(n+1)) = n?

**Step 1:** Factor out n
```
GCD(n², n(n+1)) = n × GCD(n, n+1)
```

**Step 2:** Prove GCD(n, n+1) = 1

Let d = GCD(n, n+1)
- If d divides n, then d divides n
- If d divides n+1, then d divides (n+1) - n = 1
- Therefore d divides 1, so d = 1

**Step 3:** Conclusion
```
GCD(n², n(n+1)) = n × 1 = n
```
