[657. Robot Return to Origin](https://leetcode.com/problems/robot-return-to-origin/)

# Intuition

Each move cancels with its opposite:

* `U` cancels `D`
* `L` cancels `R`

Robot returns to origin only if total vertical and horizontal movements balance to zero.

---

# Approach

1. Initialize:

   * `x = 0` (horizontal)
   * `y = 0` (vertical)
2. Traverse each move:

   * `U → y++`
   * `D → y--`
   * `R → x++`
   * `L → x--`
3. At end:

   * Return `true` if `(x === 0 && y === 0)`

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(1)$$

---

# Code

```typescript [] id="mv001"
function judgeCircle(moves: string): boolean {
    let x = 0, y = 0;

    for (const move of moves) {
        if (move === 'U') y++;
        else if (move === 'D') y--;
        else if (move === 'R') x++;
        else if (move === 'L') x--;
    }

    return x === 0 && y === 0;
}
```
