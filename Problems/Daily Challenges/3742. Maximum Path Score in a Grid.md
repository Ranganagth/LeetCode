[3742. Maximum Path Score in a Grid](https://leetcode.com/problems/maximum-path-score-in-a-grid/)

# Intuition

Each move has:

* **score gain** (0,1,2)
* **cost** (0 or 1)

Constraint: total cost ≤ k
Goal: maximize score

This is a **knapsack + grid DP**:

* State depends on position and cost used

---

# Approach

### State

$$
dp[i][j][c] = max score reaching (i,j) with cost c
$$

### Transition

From:

* top `(i-1, j)`
* left `(i, j-1)`

For each cost `c`:

* newCost = `c + cost(grid[i][j])`
* if `newCost ≤ k`:

$$
  dp[i][j][newCost] = max(
      dp[i][j][newCost],
      dp[prev][c] + score(grid[i][j])
  )
$$ 
### Optimization

Use rolling 2D DP per row:

* `dp[j][c]` → current row
* reuse for next row

### Result

Take:

$$max(dp[m-1][n-1][c]) for c ≤ k
$$

If none valid → return -1

---

# Complexity

* **Time complexity:**
  $$O(m \cdot n \cdot k)$$

* **Space complexity:**
  $$O(n \cdot k)$$

---

# Code

```typescript
function maxPathScore(grid: number[][], k: number): number {
    const m = grid.length, n = grid[0].length;

    const getCost = (v: number) => (v === 0 ? 0 : 1);
    const getScore = (v: number) => v;

    // dp[j][c] → column j, cost c
    let dp = Array.from({ length: n }, () => Array(k + 1).fill(-1));

    dp[0][0] = 0; // start

    for (let i = 0; i < m; i++) {
        const newDp = Array.from({ length: n }, () => Array(k + 1).fill(-1));

        for (let j = 0; j < n; j++) {
            for (let c = 0; c <= k; c++) {
                let bestPrev = -1;

                if (i > 0) bestPrev = Math.max(bestPrev, dp[j][c]);
                if (j > 0) bestPrev = Math.max(bestPrev, newDp[j - 1][c]);

                if (i === 0 && j === 0) bestPrev = 0;

                if (bestPrev === -1) continue;

                const cost = getCost(grid[i][j]);
                const score = getScore(grid[i][j]);

                const newCost = c + cost;
                if (newCost <= k) {
                    newDp[j][newCost] = Math.max(
                        newDp[j][newCost],
                        bestPrev + score
                    );
                }
            }
        }

        dp = newDp;
    }

    let ans = Math.max(...dp[n - 1]);
    return ans === -1 ? -1 : ans;
};

```
