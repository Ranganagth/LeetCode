[61. Rotate List](https://leetcode.com/problems/rotate-list/)

# Intuition

Rotating right by `k` moves the last `k` nodes to the front.

Instead of shifting repeatedly:

* Treat the list as **circular**
* Break it at the correct position

---

# Approach

### Step 1: Edge cases

* If empty or single node → return as is

### Step 2: Find length and tail

* Traverse list → get `length` and `tail`

### Step 3: Normalize k

```
k = k % length
```

If `k == 0` → return original

### Step 4: Make circular

```
tail.next = head
```

### Step 5: Find new tail

New tail is at:

```
(length - k - 1)
```

### Step 6: Break circle

* newHead = newTail.next
* newTail.next = null

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(1)$$

---

# Code

```typescript
function rotateRight(head: ListNode | null, k: number): ListNode | null {
    if (!head || !head.next || k === 0) return head;

    // Step 1: find length and tail
    let length = 1;
    let tail = head;

    while (tail.next) {
        tail = tail.next;
        length++;
    }

    // Step 2: normalize k
    k = k % length;
    if (k === 0) return head;

    // Step 3: make circular
    tail.next = head;

    // Step 4: find new tail
    let stepsToNewTail = length - k - 1;
    let newTail = head;

    while (stepsToNewTail > 0) {
        newTail = newTail.next!;
        stepsToNewTail--;
    }

    // Step 5: break
    const newHead = newTail.next;
    newTail.next = null;

    return newHead;
}
```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
[1,2,3,4,5], k = 2
```

Length = 5
k = 2

New tail index = 5 - 2 - 1 = 2 → node `3`

Break:

```
new head = 4
result = [4,5,1,2,3]
```

---

## Example 2:

Input:

```
[0,1,2], k = 4
```

k = 4 % 3 = 1

New tail = index 1 → node `1`

Result:

```
[2,0,1]
```
