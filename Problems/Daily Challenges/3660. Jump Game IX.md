[3660. Jump Game IX](https://leetcode.com/problems/jump-game-ix/)

# Intuition

From index `i`, movement rules create a structure where:

* Moving right → strictly decreasing values
* Moving left → strictly increasing values

This forms **monotonic segments**, and reachable values are bounded by:

* nearest greater on left
* nearest smaller on right

Instead of simulating jumps, collapse segments and propagate the **maximum reachable value**.

---

# Approach

### Key observation

Each index belongs to a segment controlled by:

* a **pivot (maximum so far)**
* propagation flows backward

### Step 1: Prefix maximum

```
prevMax[i] = max value in nums[0..i] and its index
```

### Step 2: Recursive segmentation

Process from right to left:

* Take rightmost segment `[pivotIndex → r]`
* Assign same reachable max to entire segment
* Recurse on left side

### Step 3: Transition logic

At position `r`:

* If prefix max ≤ rightMin → use prefix max
* Else propagate `rightMax`

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**  $$O(n)$$

---

# Code

```typescript
type Item = [number, number]; // [value, index]

function maxValue(nums: number[]): number[] {
    const n = nums.length;
    const ans = new Array<number>(n).fill(0);

    const prevMax: Item[] = new Array(n);

    // Step 1: prefix max
    nums.reduce(
        (prev, value, index) => {
            if (value > prev[0]) {
                prev = [value, index];
            }
            return (prevMax[index] = [...prev]);
        },
        [-Infinity, -1] as Item
    );

    const process = (r: number, rightMin: number, rightMax: number) => {
        const [pMax, pivotIndex] = prevMax[r];

        const currMax = pMax <= rightMin ? pMax : rightMax;

        let nextRightMin = Math.min(pMax, rightMin);

        // assign segment
        for (let i = pivotIndex; i <= r; i++) {
            ans[i] = currMax;
            nextRightMin = Math.min(nextRightMin, nums[i]);
        }

        if (pivotIndex === 0) return;

        // recurse left
        process(pivotIndex - 1, nextRightMin, currMax);
    };

    process(n - 1, Infinity, 0);

    return ans;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
[2,1,3]
```

* Index 2 → max = 3
* Index 1 → can go to 0 → reach 2
* Index 0 → no higher reachable → 2

Output:

```
[2,2,3]
```

---

## Example 2:

Input:

```
[2,3,1]
```

* Index 2 → jump to 1 → 3
* Index 0 → → 2 → 1 → 3
* Index 1 → already max

Output:

```
[3,3,3]
```
