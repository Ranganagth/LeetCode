[3548. Equal Sum Grid Partition II](https://leetcode.com/problems/equal-sum-grid-partition-ii/)

# Intuition

Basic partition requires equal sums. With one allowed removal, the difference between two sections can be neutralized by removing a single cell equal to that difference.

Let:

* leftSum = sum of first section
* rightSum = total - leftSum

Then:

* If equal → valid
* Else difference = |leftSum - rightSum|
  → must exist a cell with value = difference in the **larger section**

Constraint: removing that cell must not break connectivity.
In a rectangular section, removing a cell is safe **only if it is not a “bridge”**.
For this problem, connectivity is preserved if:

* section has more than 1 row and more than 1 column → always safe
* single row or single column → only endpoints can be removed safely

---

# Approach

1. Compute total sum
2. Try all horizontal cuts:

   * Maintain running sum
   * For each cut:

     * Compute difference = |2 * sum - total|
     * If difference == 0 → return true
     * Else check:

       * Identify larger section
       * Check if a cell with value = difference exists in that section
       * Ensure removing that cell keeps section connected:

         * If section is 1D → only edges allowed
         * Else → any cell allowed
3. Repeat same logic for vertical cuts
4. Return false if no valid partition found

Optimization:

* Maintain frequency maps dynamically for top/bottom or left/right

---

# Complexity

* **Time complexity:**
  $$O(m \cdot n)$$

* **Space complexity:**
  $$O(m \cdot n)$$

function canPartitionGrid(grid: number[][]): boolean {
    const m = grid.length, n = grid[0].length;

    let total = 0;
    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            total += grid[i][j];
        }
    }

    // Helper: check if removing value keeps connectivity
    function canRemove(isRow: boolean, len1: number, len2: number, val: number, edgeVals: number[]): boolean {
        if (len1 > 1 && len2 > 1) return true; // 2D block → always safe
        // 1D case → only edges allowed
        return edgeVals.includes(val);
    }

    // Horizontal cut
    let sum = 0;
    const topMap = new Map<number, number>();
    const bottomMap = new Map<number, number>();

    // initialize bottomMap
    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            bottomMap.set(grid[i][j], (bottomMap.get(grid[i][j]) || 0) + 1);
        }
    }

    for (let i = 0; i < m - 1; i++) {
        for (let j = 0; j < n; j++) {
            const val = grid[i][j];
            sum += val;

            topMap.set(val, (topMap.get(val) || 0) + 1);
            bottomMap.set(val, bottomMap.get(val)! - 1);
            if (bottomMap.get(val) === 0) bottomMap.delete(val);
        }

        if (sum * 2 === total) return true;

        const diff = Math.abs(sum * 2 - total);

        if (sum > total - sum) {
            // remove from top
            if (topMap.has(diff)) {
                if (canRemove(true, i + 1, n, diff, [grid[0][0], grid[i][0], grid[0][n - 1], grid[i][n - 1]])) {
                    return true;
                }
            }
        } else {
            // remove from bottom
            if (bottomMap.has(diff)) {
                if (canRemove(true, m - i - 1, n, diff, [grid[i + 1][0], grid[m - 1][0], grid[i + 1][n - 1], grid[m - 1][n - 1]])) {
                    return true;
                }
            }
        }
    }

    // Vertical cut
    sum = 0;
    const leftMap = new Map<number, number>();
    const rightMap = new Map<number, number>();

    for (let i = 0; i < m; i++) {
        for (let j = 0; j < n; j++) {
            rightMap.set(grid[i][j], (rightMap.get(grid[i][j]) || 0) + 1);
        }
    }

    for (let j = 0; j < n - 1; j++) {
        for (let i = 0; i < m; i++) {
            const val = grid[i][j];
            sum += val;

            leftMap.set(val, (leftMap.get(val) || 0) + 1);
            rightMap.set(val, rightMap.get(val)! - 1);
            if (rightMap.get(val) === 0) rightMap.delete(val);
        }

        if (sum * 2 === total) return true;

        const diff = Math.abs(sum * 2 - total);

        if (sum > total - sum) {
            if (leftMap.has(diff)) {
                if (canRemove(false, m, j + 1, diff, [grid[0][0], grid[0][j], grid[m - 1][0], grid[m - 1][j]])) {
                    return true;
                }
            }
        } else {
            if (rightMap.has(diff)) {
                if (canRemove(false, m, n - j - 1, diff, [grid[0][j + 1], grid[0][n - 1], grid[m - 1][j + 1], grid[m - 1][n - 1]])) {
                    return true;
                }
            }
        }
    }

    return false;
};
