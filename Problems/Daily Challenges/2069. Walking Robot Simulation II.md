[2069. Walking Robot Simulation II](https://leetcode.com/problems/walking-robot-simulation-ii/)

# Intuition

The robot always moves along the **boundary of the rectangle** in a fixed cycle:

* Bottom row → right
* Right column → up
* Top row → left
* Left column → down

This forms a **perimeter loop**.
Instead of simulating step-by-step, precompute this loop and move using modulo.

---

# Approach

### Step 1: Precompute perimeter path

Store all boundary positions in order:

1. Bottom row → `(0 → width-1, 0)`
2. Right column → `(width-1, 1 → height-1)`
3. Top row → `(width-2 → 0, height-1)`
4. Left column → `(0, height-2 → 1)`

Also store direction at each position.

### Step 2: Maintain index

* Robot position = `pos[idx]`
* Move: `idx = (idx + num) % perimeterLength`

### Step 3: Direction handling

* Special case: before any movement → `"East"`
* Otherwise → direction from precomputed array

---

# Complexity

* **Time complexity:**
  $$O(1)$$ per operation

* **Space complexity:**
  $$O(width + height)$$

# Code

```typescript
class Robot {
    private pos: number[][] = [];
    private dir: number[] = [];
    private idx: number = 0;
    private moved: boolean = false;

    private toDir: string[] = ["East", "North", "West", "South"];

    constructor(width: number, height: number) {
        // bottom row
        for (let i = 0; i < width; i++) {
            this.pos.push([i, 0]);
            this.dir.push(0);
        }

        // right column
        for (let i = 1; i < height; i++) {
            this.pos.push([width - 1, i]);
            this.dir.push(1);
        }

        // top row
        for (let i = width - 2; i >= 0; i--) {
            this.pos.push([i, height - 1]);
            this.dir.push(2);
        }

        // left column
        for (let i = height - 2; i > 0; i--) {
            this.pos.push([0, i]);
            this.dir.push(3);
        }

        // special direction fix
        this.dir[0] = 3;
    }

    step(num: number): void {
        this.moved = true;
        const len = this.pos.length;
        this.idx = (this.idx + num) % len;
    }

    getPos(): number[] {
        return this.pos[this.idx];
    }

    getDir(): string {
        if (!this.moved) return "East";
        return this.toDir[this.dir[this.idx]];
    }
};

```
