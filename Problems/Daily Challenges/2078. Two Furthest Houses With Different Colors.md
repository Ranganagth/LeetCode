[2078. Two Furthest Houses With Different Colors](https://leetcode.com/problems/two-furthest-houses-with-different-colors/)

# Intuition

Maximum distance comes from endpoints.
For any pair `(i, j)` with different colors, the best candidate is:

* Either farthest from left (index 0)
* Or farthest from right (index n-1)

No need to check all pairs.

---

# Approach

1. Let `n = colors.length`
2. Initialize `ans = 0`

**From left (index 0):**

* Find farthest index `j` where `colors[j] != colors[0]`
* Update `ans = j`

**From right (index n-1):**

* Find farthest index `i` where `colors[i] != colors[n-1]`
* Update `ans = max(ans, n - 1 - i)`

3. Return `ans`

---

# Complexity

* **Time complexity:**  $$O(n)$$

* **Space complexity:**  $$O(1)$$

---

# Code

```typescript
function maxDistance(colors: number[]): number {
    const n = colors.length;

    let ans = 0;

    // compare with first
    for (let i = n - 1; i >= 0; i--) {
        if (colors[i] !== colors[0]) {
            ans = Math.max(ans, i);
            break;
        }
    }

    // compare with last
    for (let i = 0; i < n; i++) {
        if (colors[i] !== colors[n - 1]) {
            ans = Math.max(ans, n - 1 - i);
            break;
        }
    }

    return ans;
};

```
