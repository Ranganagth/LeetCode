[2784. Check if Array is Good](https://leetcode.com/problems/check-if-array-is-good/)

# Intuition

For a valid array:

```
base[n] = [1,2,...,n-1,n,n]
```

Properties:

* Length must be `n + 1`
* Maximum value must be `n`
* Numbers `1 → n-1` appear exactly once
* Number `n` appears exactly twice

---

# Approach

1. Let:

```
n = max(nums)
```

2. Check:

```
nums.length === n + 1
```

3. Count frequencies

4. Validate:

* `1 → n-1` frequency = 1
* `n` frequency = 2

If all conditions satisfy → true

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(n)$$

---

# Code

```typescript
function isGood(nums: number[]): boolean {
    const n = Math.max(...nums);

    // length must be n + 1
    if (nums.length !== n + 1) {
        return false;
    }

    const freq = new Map<number, number>();

    for (const num of nums) {
        freq.set(num, (freq.get(num) || 0) + 1);
    }

    // 1..n-1 must appear once
    for (let i = 1; i < n; i++) {
        if (freq.get(i) !== 1) {
            return false;
        }
    }

    // n must appear twice
    return freq.get(n) === 2;
};

```
