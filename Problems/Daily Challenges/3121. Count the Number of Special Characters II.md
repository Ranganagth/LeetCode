[3121. Count the Number of Special Characters II](https://leetcode.com/problems/count-the-number-of-special-characters-ii/)

# Intuition
A letter is **special** if:
1. It appears in both lowercase and uppercase forms.
2. **All lowercase occurrences come before the first uppercase occurrence**.

The key insight is to track:
- The **last occurrence** of each lowercase letter.
- The **first occurrence** of each uppercase letter.

Then, for each letter that appears in both forms, check if the last lowercase occurrence comes before the first uppercase occurrence.

---

# Approach
1. Create two maps:
   - `lastLowercase`: stores the last index of each lowercase letter.
   - `firstUppercase`: stores the first index of each uppercase letter.
2. Iterate through the word once:
   - Update `lastLowercase[char]` whenever we see a lowercase letter.
   - Update `firstUppercase[char]` only on the first occurrence of an uppercase letter.
3. Count special letters:
   - For each letter that appears in both maps, check if `lastLowercase[letter] < firstUppercase[letter]`.
   - If true, increment the count.
4. Return the count.

---

# Complexity
- **Time complexity:**  $$O(n)$$ where \( n \) is the length of the word. We iterate through the word once and then check at most 26 letters.
- **Space complexity:** $$O(1)$$ since we store at most 26 lowercase and 26 uppercase letters.

---

# Code
```typescript
function numberOfSpecialChars(word: string): number {
    const lastLowercase = new Map<string, number>();
    const firstUppercase = new Map<string, number>();

    // Track last occurrence of lowercase and first occurrence of uppercase
    for (let i = 0; i < word.length; i++) {
        const char = word[i];
        
        if (char >= 'a' && char <= 'z') {
            // Update last occurrence of lowercase
            lastLowercase.set(char, i);
        } else if (char >= 'A' && char <= 'Z') {
            // Update first occurrence of uppercase
            const lowerChar = char.toLowerCase();
            if (!firstUppercase.has(lowerChar)) {
                firstUppercase.set(lowerChar, i);
            }
        }
    }

    // Count special letters
    let count = 0;
    for (const [letter, lastIndex] of lastLowercase) {
        if (firstUppercase.has(letter)) {
            const firstIndex = firstUppercase.get(letter)!;
            // Special if all lowercase occurrences come before first uppercase
            if (lastIndex < firstIndex) {
                count++;
            }
        }
    }

    return count;
};

```

# Example walkthrough

## Example 1:
`word = "aaAbcBC"`

**Iteration through word:**

| Index | Char | lastLowercase | firstUppercase |
|-------|------|---------------|-----------------|
| 0 | 'a' | {a: 0} | {} |
| 1 | 'a' | {a: 1} | {} |
| 2 | 'A' | {a: 1} | {a: 2} |
| 3 | 'b' | {a: 1, b: 3} | {a: 2} |
| 4 | 'c' | {a: 1, b: 3, c: 4} | {a: 2} |
| 5 | 'B' | {a: 1, b: 3, c: 4} | {a: 2, b: 5} |
| 6 | 'C' | {a: 1, b: 3, c: 4} | {a: 2, b: 5, c: 6} |

**Count special letters:**
- 'a': lastLowercase['a'] = 1, firstUppercase['a'] = 2 → 1 < 2 count = 1
- 'b': lastLowercase['b'] = 3, firstUppercase['b'] = 5 → 3 < 5 count = 2
- 'c': lastLowercase['c'] = 4, firstUppercase['c'] = 6 → 4 < 6 count = 3

**Output:** `3`
