[2169. Count Operations to Obtain Zero](https://leetcode.com/problems/count-operations-to-obtain-zero/)

# Intuition

This operation is a variation of the **Euclidean algorithm** used to find the greatest common divisor (GCD).
In each step, we subtract the smaller number from the larger until one becomes zero.

However, instead of performing repeated subtractions one by one, we can observe that:

* When `num1 >= num2`, we can perform `floor(num1 / num2)` subtractions at once.
* After that, the remainder is `num1 % num2`, and we continue with `(num1 % num2, num2)`.

Thus, the process is similar to GCD calculation, except we **count how many subtractions occur** in total.

---

# Approach

1. Initialize `operations = 0`.
2. While both numbers are non-zero:

   * If `num1 >= num2`:

     * Add `Math.floor(num1 / num2)` to `operations`.
     * Update `num1 = num1 % num2`.
   * Else:

     * Add `Math.floor(num2 / num1)` to `operations`.
     * Update `num2 = num2 % num1`.
3. When one number becomes `0`, stop.
4. Return `operations`.

This approach efficiently computes the result without looping for each subtraction.

---

# Complexity

* **Time Complexity:** `O(log(max(num1, num2)))` — same as Euclidean GCD algorithm.
* **Space Complexity:** `O(1)` — uses constant space.

---

# Code

```typescript
function countOperations(num1: number, num2: number): number {
    let operations = 0;

    while (num1 !== 0 && num2 !== 0) {
        if (num1 >= num2) {
            operations += Math.floor(num1 / num2);
            num1 %= num2;
        } else {
            operations += Math.floor(num2 / num1);
            num2 %= num1;
        }
    }

    return operations;
};

```

## Example Walkthrough

#### Example 1:

```
num1 = 2, num2 = 3
```

* Step 1: `num1 < num2`

  * operations += 3 // 2 = 1
  * num2 = 3 % 2 = 1
    → (num1, num2) = (2, 1), operations = 1
* Step 2: `num1 > num2`

  * operations += 2 // 1 = 2
  * num1 = 2 % 1 = 0
    → (num1, num2) = (0, 1), operations = 3

**Output: 3**


#### Example 2:

```
num1 = 10, num2 = 10
```

* Step 1: `num1 == num2`

  * operations += 10 // 10 = 1
  * num1 = 10 % 10 = 0

**Output: 1**
