[3474. Lexicographically Smallest Generated String](https://leetcode.com/problems/lexicographically-smallest-generated-string/)

# Intuition

Each `'T'` enforces a strict placement of `str2` into `word` at position `i`.
Overlapping `'T'` constraints must be consistent.

Each `'F'` enforces that the substring at position `i` must differ from `str2` in **at least one position**.

Goal:
* Build the lexicographically smallest valid `word`
* Satisfy all `'T'` constraints first (hard constraints)
* Then ensure `'F'` constraints are not violated

---

# Approach

### Step 1: Initialize

* Length of result = `n + m - 1`
* Initialize `word` with `'?'`

### Step 2: Apply all `'T'` constraints

For every `i` where `str1[i] == 'T'`:

* Force `word[i + j] = str2[j]`
* If conflict occurs → return `""`

### Step 3: Fill remaining `'?'` with smallest char `'a'`

* Greedy for lexicographically smallest result

### Step 4: Fix `'F'` constraints

For every `i` where `str1[i] == 'F'`:

* Check if substring equals `str2`
* If yes → must break it:

  * Try modifying one position inside `[i, i + m - 1]`
  * Choose smallest possible change (to keep lexicographically minimal)
  * Ensure modification does not break any `'T'` constraint

If unable to break → return `""`

---

# Complexity

* **Time complexity:**
  $$O(n \cdot m)$$

* **Space complexity:**
  $$O(n + m)$$

---

# Code

```typescript [] id="gen992"
function generateString(str1: string, str2: string): string {
    const n = str1.length;
    const m = str2.length;
    const size = n + m - 1;

    const word: string[] = new Array(size).fill('?');

    // Step 1: Apply 'T'
    for (let i = 0; i < n; i++) {
        if (str1[i] === 'T') {
            for (let j = 0; j < m; j++) {
                if (word[i + j] === '?' || word[i + j] === str2[j]) {
                    word[i + j] = str2[j];
                } else {
                    return "";
                }
            }
        }
    }

    // Step 2: Fill remaining with 'a'
    for (let i = 0; i < size; i++) {
        if (word[i] === '?') word[i] = 'a';
    }

    // Step 3: Fix 'F'
    for (let i = 0; i < n; i++) {
        if (str1[i] === 'F') {
            let match = true;

            for (let j = 0; j < m; j++) {
                if (word[i + j] !== str2[j]) {
                    match = false;
                    break;
                }
            }

            if (match) {
                let changed = false;

                for (let j = m - 1; j >= 0; j--) {
                    const idx = i + j;

                    // Try changing character
                    for (let c = 0; c < 26; c++) {
                        const ch = String.fromCharCode(97 + c);
                        if (ch !== word[idx]) {
                            const original = word[idx];
                            word[idx] = ch;

                            // Check if breaks T constraints
                            let valid = true;
                            for (let k = Math.max(0, idx - m + 1); k <= Math.min(idx, n - 1); k++) {
                                if (str1[k] === 'T') {
                                    for (let t = 0; t < m; t++) {
                                        if (word[k + t] !== str2[t]) {
                                            valid = false;
                                            break;
                                        }
                                    }
                                    if (!valid) break;
                                }
                            }

                            if (valid) {
                                changed = true;
                                break;
                            } else {
                                word[idx] = original;
                            }
                        }
                    }

                    if (changed) break;
                }

                if (!changed) return "";
            }
        }
    }

    return word.join("");
};

```
