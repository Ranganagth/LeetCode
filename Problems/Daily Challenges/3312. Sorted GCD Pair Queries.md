[3312. Sorted GCD Pair Queries](https://leetcode.com/problems/sorted-gcd-pair-queries/description/?envType=daily-question&envId=2026-07-17)

# Intuition

Instead of building the full array, we can:
1. Count pairs for each GCD value
2. Build a **cumulative count array**
3. Use **binary search** to find which GCD corresponds to each query index

---

# Approach

## Optimized Strategy: Cumulative Counts + Binary Search

1. **Count divisors and pairs** (same as before)
2. **Use inclusion-exclusion** to get exact GCD counts
3. **Build cumulative array:** Store cumulative counts instead of expanding all values
4. **Binary search:** For each query, find which GCD value it corresponds to

This reduces space from O(n²) to O(M) where M = max value.

---

# Complexity

- **Time complexity:** $$O(M \log M + n \cdot \sqrt{M} + q \log M)$$
  - Preprocessing: $$O(M \log M + n \cdot \sqrt{M})$$
  - Each query: $$O(\log M)$$ for binary search

- **Space complexity:** $$O(M)$$ - only store counts, not expanded array

# Code

```typescript
function gcdValues(nums: number[], queries: number[]): number[] {
    const n = nums.length;
    const maxVal = Math.max(...nums);
    
    // Step 1: Count how many numbers are divisible by each divisor
    const divisorCount = new Array(maxVal + 1).fill(0);
    for (const num of nums) {
        for (let d = 1; d * d <= num; d++) {
            if (num % d === 0) {
                divisorCount[d]++;
                if (d !== num / d) {
                    divisorCount[num / d]++;
                }
            }
        }
    }
    
    // Step 2: Calculate pairs with GCD at least d
    const pairsWithGcdAtLeast = new Array(maxVal + 1).fill(0);
    for (let d = 1; d <= maxVal; d++) {
        const cnt = divisorCount[d];
        if (cnt >= 2) {
            pairsWithGcdAtLeast[d] = (cnt * (cnt - 1)) / 2;
        }
    }
    
    // Step 3: Use inclusion-exclusion to get exact GCD counts
    const pairsWithExactGcd = new Array(maxVal + 1).fill(0);
    for (let d = maxVal; d >= 1; d--) {
        pairsWithExactGcd[d] = pairsWithGcdAtLeast[d];
        
        // Subtract pairs with GCD = k*d for k > 1
        for (let multiple = 2 * d; multiple <= maxVal; multiple += d) {
            pairsWithExactGcd[d] -= pairsWithExactGcd[multiple];
        }
    }
    
    // Step 4: Build cumulative count array
    // gcdList[i] = [gcd_value, cumulative_count]
    const gcdList: [number, number][] = [];
    let cumulativeCount = 0;
    
    for (let gcd = 1; gcd <= maxVal; gcd++) {
        if (pairsWithExactGcd[gcd] > 0) {
            cumulativeCount += pairsWithExactGcd[gcd];
            gcdList.push([gcd, cumulativeCount]);
        }
    }
    
    // Step 5: Answer queries using binary search
    const answer: number[] = [];
    
    for (const query of queries) {
        // Binary search to find which GCD corresponds to this index
        // We want the smallest i where gcdList[i][1] > query
        let left = 0;
        let right = gcdList.length - 1;
        let result = gcdList[0][0];
        
        while (left <= right) {
            const mid = Math.floor((left + right) / 2);
            const [gcdValue, cumCount] = gcdList[mid];
            
            // Check if query index falls in this range
            const prevCount = mid > 0 ? gcdList[mid - 1][1] : 0;
            
            if (query < cumCount && query >= prevCount) {
                result = gcdValue;
                break;
            } else if (query < prevCount) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        
        answer.push(result);
    }
    
    return answer;
};
```

## Alternative: Simpler Binary Search

```typescript
function gcdValues(nums: number[], queries: number[]): number[] {
    const n = nums.length;
    const maxVal = Math.max(...nums);
    
    // Step 1: Count divisors
    const divisorCount = new Array(maxVal + 1).fill(0);
    for (const num of nums) {
        for (let d = 1; d * d <= num; d++) {
            if (num % d === 0) {
                divisorCount[d]++;
                if (d !== num / d) {
                    divisorCount[num / d]++;
                }
            }
        }
    }
    
    // Step 2: Calculate pairs with GCD at least d
    const pairsWithGcdAtLeast = new Array(maxVal + 1).fill(0);
    for (let d = 1; d <= maxVal; d++) {
        const cnt = divisorCount[d];
        if (cnt >= 2) {
            pairsWithGcdAtLeast[d] = (cnt * (cnt - 1)) / 2;
        }
    }
    
    // Step 3: Inclusion-exclusion
    const pairsWithExactGcd = new Array(maxVal + 1).fill(0);
    for (let d = maxVal; d >= 1; d--) {
        pairsWithExactGcd[d] = pairsWithGcdAtLeast[d];
        for (let multiple = 2 * d; multiple <= maxVal; multiple += d) {
            pairsWithExactGcd[d] -= pairsWithExactGcd[multiple];
        }
    }
    
    // Step 4: Build prefix sum array
    const prefixSum = new Array(maxVal + 1).fill(0);
    for (let gcd = 1; gcd <= maxVal; gcd++) {
        prefixSum[gcd] = prefixSum[gcd - 1] + pairsWithExactGcd[gcd];
    }
    
    // Step 5: Answer queries
    const answer: number[] = [];
    
    for (const query of queries) {
        // Binary search for the GCD value
        let left = 1;
        let right = maxVal;
        
        while (left < right) {
            const mid = Math.floor((left + right) / 2);
            
            if (prefixSum[mid] <= query) {
                left = mid + 1;
            } else {
                right = mid;
            }
        }
        
        answer.push(left);
    }
    
    return answer;
};

```

# Example Walkthrough

## Example 1:

**Input:** `nums = [2,3,4], queries = [0,2,2]`

**After preprocessing:**
- `pairsWithExactGcd[1] = 2`
- `pairsWithExactGcd[2] = 1`

**Cumulative counts:**
- GCD=1: cumulative = 2 (indices 0-1)
- GCD=2: cumulative = 3 (index 2)

**Query processing:**
- query=0: Binary search finds GCD=1 (0 < 2)
- query=2: Binary search finds GCD=2 (2 >= 2 and 2 < 3)
- query=2: Binary search finds GCD=2

**Output:** `[1, 2, 2]` 

---

## Key Optimizations

- **No array expansion** - Store counts, not individual elements
- **Prefix sum** - Enable O(log M) query answering
- **Binary search** - Find GCD value efficiently
- **Memory efficient** - O(M) instead of O(n²)

---

## Binary Search Logic

For query index `q`, we want to find the smallest GCD value `g` such that:
```
prefixSum[g-1] <= q < prefixSum[g]
```

This means the q-th element (0-indexed) has GCD value `g`.

---

## Edge Case Handling

- Empty ranges: Check if `pairsWithExactGcd[gcd] > 0` before adding
- Single pair: Works correctly with cumulative counts
- Large queries: Binary search handles efficiently
