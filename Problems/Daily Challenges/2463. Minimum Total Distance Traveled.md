[2463. Minimum Total Distance Traveled](https://leetcode.com/problems/minimum-total-distance-traveled/)

# Intuition

Robots and factories lie on a line.
Optimal strategy is to **match robots to factories in sorted order** (greedy by position), because crossing assignments increases total distance.

This becomes:

* Assign each robot to a factory (with capacity constraint)
* Minimize total distance

Classic **DP + prefix assignment** problem.

---

# Approach

### Step 1: Sort

* Sort `robot`
* Sort `factory` by position

### Step 2: DP definition

Let:

```
dp[i][j] = minimum distance to repair first i robots using first j factories
```

### Step 3: Transition

For factory `j`, we can assign `k` robots (0 ≤ k ≤ limit):

For last `k` robots:

```
cost = sum of |robot[i - t] - factory[j][0]| for t in [1..k]
```

Transition:

```
dp[i][j] = min(
    dp[i][j-1],  // skip this factory
    dp[i-k][j-1] + cost
)
```

### Step 4: Optimization

* Precompute cost incrementally (prefix accumulation)
* Use rolling DP for space optimization

---

# Complexity

* **Time complexity:**$$O(n \cdot m \cdot limit)$$ → worst ~ $(10^6)$

* **Space complexity:** $$O(n)$$

---

# Code

```typescript
function minimumTotalDistance(robot: number[], factory: number[][]): number {
    robot.sort((a, b) => a - b);
    factory.sort((a, b) => a[0] - b[0]);

    const n = robot.length;
    const m = factory.length;

    const dp = new Array(n + 1).fill(Infinity);
    dp[0] = 0;

    for (let j = 0; j < m; j++) {
        const [pos, limit] = factory[j];
        const newDp = [...dp];

        for (let i = 1; i <= n; i++) {
            let cost = 0;

            for (let k = 1; k <= Math.min(limit, i); k++) {
                cost += Math.abs(robot[i - k] - pos);
                newDp[i] = Math.min(newDp[i], dp[i - k] + cost);
            }
        }

        for (let i = 0; i <= n; i++) {
            dp[i] = newDp[i];
        }
    }

    return dp[n];
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
robot = [0,4,6]
factory = [[2,2],[6,2]]
```

Sorted:

* robots → [0,4,6]
* factories → [2,6]

Factory 2 (limit=2):

* assign [0,4] → cost = |2-0| + |2-4| = 2 + 2 = 4

Factory 6:

* assign [6] → cost = 0

Total = 4


## Example 2:

Input:

```
robot = [1,-1]
factory = [[-2,1],[2,1]]
```

Sorted:

* robots → [-1,1]

Assignments:

* -1 → -2 → cost 1
* 1 → 2 → cost 1

Total = 2
