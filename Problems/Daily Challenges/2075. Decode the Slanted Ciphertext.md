[2075. Decode the Slanted Ciphertext](https://leetcode.com/problems/decode-the-slanted-ciphertext/)


# Intuition

The encoded string is formed by:

1. Filling characters diagonally (top-left → bottom-right)
2. Reading row-wise

To reverse:

* Reconstruct the matrix row-wise
* Then read diagonally to recover original text

---

# Approach

### Step 1: Compute dimensions

* `n = encodedText.length`
* Columns = `n / rows`

### Step 2: Build matrix row-wise

Fill matrix row by row using `encodedText`

### Step 3: Traverse diagonally

For each starting column `c`:

* Start at `(row=0, col=c)`
* Move diagonally `(row+1, col+1)`
* Append characters

### Step 4: Trim trailing spaces

Remove trailing `' '` from result

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(n)$$

---

# Code

```typescript
function decodeCiphertext(encodedText: string, rows: number): string {
    if (rows === 1) return encodedText;

    const n = encodedText.length;
    const cols = Math.floor(n / rows);

    // Step 1: build matrix
    const matrix: string[][] = Array.from({ length: rows }, () => new Array(cols));

    let idx = 0;
    for (let i = 0; i < rows; i++) {
        for (let j = 0; j < cols; j++) {
            matrix[i][j] = encodedText[idx++];
        }
    }

    // Step 2: read diagonally
    let result = "";

    for (let startCol = 0; startCol < cols; startCol++) {
        let i = 0, j = startCol;

        while (i < rows && j < cols) {
            result += matrix[i][j];
            i++;
            j++;
        }
    }

    // Step 3: trim trailing spaces
    return result.replace(/\s+$/, "");
}
```
