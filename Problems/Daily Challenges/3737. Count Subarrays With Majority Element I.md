[3737. Count Subarrays With Majority Element I](https://leetcode.com/problems/count-subarrays-with-majority-element-i/)

# Intuition

We need to count subarrays where `target` is the **majority element**—meaning it appears more than half of the length of that subarray.

Key observations:
- The subarray length `L` must satisfy: `target_count > L/2`.
- For a subarray with `target_count = t`, the total length `L` is at least `2t - 1`.
- Since `target` must be the majority, it must appear more than half the times in the subarray.

Given the constraints (`nums.length <= 1000`), we can afford an $O(n^3)$ approach:
- Enumerate all subarrays.
- Count the occurrences of `target` and other elements.
- Check if `target` is the majority.

But we can optimize:
- For each starting index, expand the subarray to the right.
- Keep track of `target` count and total length.
- Count only those subarrays where `target` is the majority.

---

# Approach

1. Loop through all possible starting indices.
2. For each starting index, expand to the right:
   - Count how many times `target` appears.
   - For each subarray, check if `target` is the majority: `target_count > length/2`.
3. Increment the result count accordingly.

---

# Code

```typescript
function countMajoritySubarrays(nums: number[], target: number): number {
    const n = nums.length;
    let count = 0;

    for (let start = 0; start < n; start++) {
        let targetCount = 0;
        for (let end = start; end < n; end++) {
            if (nums[end] === target) targetCount++;
            const length = end - start + 1;
            if (targetCount > length / 2) {
                count++;
            }
        }
    }

    return count;
};

```

---

# Explanation
- For each subarray `[start..end]`, we track how many times `target` appears.
- If `targetCount > (end - start + 1) / 2`, then `target` is the majority.
- We count all such subarrays.

---

# Example
```typescript
console.log(countMajoritySubarrays([1,2,2,3], 2)); // Output: 5
console.log(countMajoritySubarrays([1,1,1,1], 1)); // Output: 10
console.log(countMajoritySubarrays([1,2,3], 4)); // Output: 0
```

This solution is straightforward and efficient enough given the constraints.
