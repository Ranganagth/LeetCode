[2095. Delete the Middle Node of a Linked List](https://leetcode.com/problems/delete-the-middle-node-of-a-linked-list/)

# Intuition

To delete the middle node of a linked list, we need to:
1. Find the middle node efficiently without traversing the list twice.
2. Delete it by updating the previous node's `next` pointer.

The **two-pointer technique** (slow and fast pointers) is perfect for this:
- **Slow pointer** moves 1 step at a time.
- **Fast pointer** moves 2 steps at a time.
- When the fast pointer reaches the end, the slow pointer will be at the middle.

We also need to keep track of the node **before** the middle node so we can update its `next` pointer to skip the middle node.

**Edge case:** If the list has only 1 node, we return `null`.

---

# Approach

1. **Handle edge case:** If the list has only 1 node, return `null`.
2. **Use two pointers:**
   - Create a dummy node pointing to the head (helps handle deletion of the first node).
   - Use `prev` pointer to track the node before the middle.
   - Use `slow` pointer to find the middle node.
   - Use `fast` pointer to detect when we've reached the middle.
3. **Move pointers:**
   - Move `slow` 1 step and `fast` 2 steps at a time.
   - Move `prev` along with `slow`.
4. **Delete the middle node:**
   - Update `prev.next` to skip the middle node: `prev.next = slow.next`.
5. **Return the head** (or `dummy.next` if we used a dummy node).

---

# Complexity

- **Time complexity:** 
  $$O(n)$$ where `n` is the number of nodes. We traverse the list once with the two-pointer technique.

- **Space complexity:** 
  $$O(1)$$ as we only use a constant amount of extra space for pointers.

# Code

```typescript
/**
 * Definition for singly-linked list.
 * class ListNode {
 * val: number
 * next: ListNode | null
 * constructor(val?: number, next?: ListNode | null) {
 * this.val = (val===undefined ? 0 : val)
 * this.next = (next===undefined ? null : next)
 * }
 * }
 */

function deleteMiddle(head: ListNode | null): ListNode | null {
    // Edge case: single node
    if (head === null || head.next === null) {
        return null;
    }

    // Create dummy node to handle edge cases (e.g., deleting head)
    const dummy = new ListNode(0, head);
    let prev = dummy;
    let slow = head;
    let fast = head;

    // Move fast pointer 2 steps and slow pointer 1 step
    // When fast reaches the end, slow will be at the middle
    while (fast !== null && fast.next !== null) {
        prev = slow;
        slow = slow.next!;
        fast = fast.next.next;
    }

    // Delete the middle node by updating prev's next pointer
    prev.next = slow.next;

    // Return the head of the modified list
    return dummy.next;
};

```

# Example Walkthrough

## Example 1:
`head = [1,3,4,7,1,2,6]` (n = 7)

**Initial state:**
```
dummy → 1 → 3 → 4 → 7 → 1 → 2 → 6 → null
prev slow fast
```

**Iteration 1:**
```
dummy → 1 → 3 → 4 → 7 → 1 → 2 → 6 → null
        prev slow fast
```

**Iteration 2:**
```
dummy → 1 → 3 → 4 → 7 → 1 → 2 → 6 → null
             prev slow fast
```

**Iteration 3:**
```
dummy → 1 → 3 → 4 → 7 → 1 → 2 → 6 → null
                  prev slow fast
```

**Iteration 4 (fast reaches null):**
```
dummy → 1 → 3 → 4 → 7 → 1 → 2 → 6 → null
                       prev slow fast
```

**Delete middle node (7):**
- `prev.next = slow.next` → `4.next = 1`
- Result: `[1,3,4,1,2,6]`

# Key Points

- **Dummy node:** Simplifies handling edge cases like deleting the head node.
- **Two pointers:** Efficiently finds the middle in a single pass.
- **Prev pointer:** Tracks the node before the middle for deletion.
- **Time efficient:** O(n) with only one traversal.
