[2573. Find the String with LCP](https://leetcode.com/problems/find-the-string-with-lcp/)

# Intuition

`lcp[i][j]` defines how many consecutive characters match starting from indices `i` and `j`.

Key implications:

* If `lcp[i][j] > 0` → `word[i] == word[j]`
* If `lcp[i][j] = k > 0` → next relation must hold:
  `lcp[i][j] = 1 + lcp[i+1][j+1]`
* Diagonal must satisfy: `lcp[i][i] = n - i`

Goal:

* Assign characters greedily (smallest lexicographically)
* Ensure all constraints are satisfied

---

# Approach

**Step 1: Build string using grouping**

* Iterate `i = 0 → n-1`
* If `word[i]` not assigned:

  * Assign smallest unused character
  * For all `j > i`, if `lcp[i][j] > 0`, assign same character

This groups indices that must be equal.


**Step 2: Validate using LCP rules**
Traverse from bottom-right:

* If `word[i] != word[j]` → `lcp[i][j]` must be 0
* If `word[i] == word[j]`:

  * If at boundary → `lcp[i][j] = 1`
  * Else → must satisfy:
    `lcp[i][j] = 1 + lcp[i+1][j+1]`

**Step 3: Return result or "" if invalid**

---

# Complexity

* **Time complexity:**
  $$O(n^2)$$

* **Space complexity:**
  $$O(n)$$

---

# Code

```typescript [] id="lcp881"
function findTheString(lcp: number[][]): string {
    const n = lcp.length;
    const word: string[] = new Array(n).fill("");
    let currentCharCode = "a".charCodeAt(0);

    // Step 1: Assign characters
    for (let i = 0; i < n; i++) {
        if (word[i] === "") {
            if (currentCharCode > "z".charCodeAt(0)) return "";

            const ch = String.fromCharCode(currentCharCode);
            word[i] = ch;

            for (let j = i + 1; j < n; j++) {
                if (lcp[i][j] > 0) {
                    word[j] = ch;
                }
            }
            currentCharCode++;
        }
    }

    // Step 2: Validate LCP matrix
    for (let i = n - 1; i >= 0; i--) {
        for (let j = n - 1; j >= 0; j--) {

            if (word[i] !== word[j]) {
                if (lcp[i][j] !== 0) return "";
            } else {
                if (i === n - 1 || j === n - 1) {
                    if (lcp[i][j] !== 1) return "";
                } else {
                    if (lcp[i][j] !== lcp[i + 1][j + 1] + 1) {
                        return "";
                    }
                }
            }
        }
    }

    return word.join("");
};

```

## Example 1: Step by step explanation

Input:

```
lcp =
[
 [4,0,2,0],
 [0,3,0,1],
 [2,0,2,0],
 [0,1,0,1]
]
```

Step 1 (grouping):

* i=0 → assign 'a'

  * lcp[0][2] = 2 → word[2] = 'a'
* i=1 → assign 'b'

  * lcp[1][3] = 1 → word[3] = 'b'

Result: `a b a b`

Step 2 (validation):

* All recursive LCP conditions satisfied

**Output: `"abab"`**
