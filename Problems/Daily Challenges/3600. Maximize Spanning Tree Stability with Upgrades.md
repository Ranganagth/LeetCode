[3600. Maximize Spanning Tree Stability with Upgrades](https://leetcode.com/problems/maximize-spanning-tree-stability-with-upgrades/)

# Intuition

Stability of a spanning tree:

```
stability = minimum edge strength inside the tree
```

Goal:

```
maximize this minimum strength
```

Classic transformation:

```
maximize minimum value → binary search on answer
```

Assume a candidate stability `X`.

Check if a spanning tree exists where **every chosen edge has strength ≥ X**.

Edges can become valid in two ways:

```
original strength ≥ X
OR
upgraded strength ≥ X
```

Upgrade rule:

```
strength → 2 * strength
```

Only edges with `must = 0` can be upgraded and only once.

Mandatory edges:

```
must be included
cannot be upgraded
```

If any mandatory edge has strength `< X`, the candidate `X` is impossible.

Graph must also remain **acyclic** while forming a spanning tree.

Use **Union-Find (Disjoint Set Union)** to enforce this.

## Key Observations

Candidate edge validity:

```
mandatory edge: usable if s ≥ X
optional edge: usable if
    s ≥ X          (no upgrade)
 OR 2*s ≥ X        (1 upgrade)
```

If an optional edge requires upgrade:

```
upgradeUsed++
upgradeUsed ≤ k
```

Spanning tree condition:

```
edgesUsed = n - 1
```

If fewer edges can be added → graph cannot connect.

---

# Approach

### Step 1 - Binary Search Stability

Search range:

```
1 → 2 * max(edge strength)
```

Reason: upgrades can double strength.

For each candidate `mid`:

```
check(mid)
```

### Step 2 - Feasibility Check

Procedure:

1. Initialize DSU.
2. Add all mandatory edges first.
3. Reject if:

   * strength < mid
   * cycle forms.
4. Collect optional edges into two groups:

```
noUpgradeEdges   (s ≥ mid)
upgradeEdges     (s < mid AND 2*s ≥ mid)
```

5. Add edges greedily:

```
first noUpgradeEdges
then upgradeEdges while upgrades ≤ k
```

6. If total edges used = `n-1`, feasible.

---

# Complexity

Let

```
N = nodes
M = edges
```

Binary search:

```
log(maxStrength)
```

Feasibility check:

```
Union-Find operations ≈ O(M α(N))
```

## Total time:

```
O(M log(maxStrength))
```

For constraints:

```
N ≤ 1e5
M ≤ 1e5
```

This remains efficient.
## Space Complexity

```
O(N)
```

Union-Find storage.

# Code

```typescript
class DSU {
    parent: number[];
    rank: number[];

    constructor(n: number) {
        this.parent = Array.from({ length: n }, (_, i) => i);
        this.rank = new Array(n).fill(0);
    }

    find(x: number): number {
        if (this.parent[x] !== x) {
            this.parent[x] = this.find(this.parent[x]);
        }
        return this.parent[x];
    }

    union(x: number, y: number): boolean {
        let px = this.find(x);
        let py = this.find(y);

        if (px === py) return false;

        if (this.rank[px] < this.rank[py]) {
            [px, py] = [py, px];
        }

        this.parent[py] = px;

        if (this.rank[px] === this.rank[py]) {
            this.rank[px]++;
        }

        return true;
    }
}

function maxStability(
    n: number,
    edges: number[][],
    k: number
): number {

    let maxStrength = 0;

    for (const [, , s] of edges) {
        maxStrength = Math.max(maxStrength, s);
    }

    let left = 1;
    let right = maxStrength * 2;
    let ans = -1;

    function can(x: number): boolean {

        const dsu = new DSU(n);

        let used = 0;
        let upgrades = 0;

        const normal: number[][] = [];
        const upgrade: number[][] = [];

        for (const [u, v, s, must] of edges) {

            if (must === 1) {

                if (s < x) return false;

                if (!dsu.union(u, v)) return false;

                used++;
            }
            else {

                if (s >= x) normal.push([u, v]);
                else if (2 * s >= x) upgrade.push([u, v]);
            }
        }

        for (const [u, v] of normal) {
            if (dsu.union(u, v)) used++;
        }

        for (const [u, v] of upgrade) {

            if (upgrades === k) break;

            if (dsu.union(u, v)) {
                used++;
                upgrades++;
            }
        }

        return used === n - 1;
    }

    while (left <= right) {

        const mid = Math.floor((left + right) / 2);

        if (can(mid)) {
            ans = mid;
            left = mid + 1;
        }
        else {
            right = mid - 1;
        }
    }

    return ans;
};

```

### Example 1

```
n = 3
edges =
[0,1,2,1]
[1,2,3,0]

k = 1
```

Mandatory edge:

```
0-1 strength = 2
```

Optional edge:

```
1-2 strength = 3
```

Try stability:

```
X = 3
```

Mandatory edge:

```
2 < 3 → invalid
```

So `X = 3` impossible.

Try:

```
X = 2
```

Mandatory:

```
2 ≥ 2 → valid
```

Optional:

```
3 ≥ 2 → usable
```

Tree:

```
0--1 (2)
1--2 (3)
```

Minimum edge:

```
2
```

Result:

```
2
```
