[1732. Find the Highest Altitude](https://leetcode.com/problems/find-the-highest-altitude/)

# Intuition

We need to track the altitude at each point along the trip. Starting at altitude 0, we accumulate the gains to find the altitude at each subsequent point. The answer is the maximum altitude encountered.

---

# Approach

1. Initialize `current_altitude = 0` (starting altitude at point 0).
2. Initialize `max_altitude = 0` (the highest altitude seen so far).
3. Iterate through each gain value:
   - Add the gain to the current altitude.
   - Update the maximum altitude if the current altitude is higher.
4. Return the maximum altitude.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$ where n is the length of the gain array (single pass).
  
- **Space complexity:** 
  $$O(1)$$ (only using constant extra space).

---

# Code

```typescript
function largestAltitude(gain: number[]): number {
    let currentAltitude = 0;
    let maxAltitude = 0;

    for (const g of gain) {
        // 📈 Update current altitude by adding the gain
        currentAltitude += g;
        // 🏔️ Track the maximum altitude seen
        maxAltitude = Math.max(maxAltitude, currentAltitude);
    }

    return maxAltitude;
};

```

- Start: altitude = 0, max = 0
- gain[0] = -5 → altitude = -5, max = 0
- gain[1] = 1 → altitude = -4, max = 0
