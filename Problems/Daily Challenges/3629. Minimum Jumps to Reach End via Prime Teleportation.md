[3629. Minimum Jumps to Reach End via Prime Teleportation](https://leetcode.com/problems/minimum-jumps-to-reach-end-via-prime-teleportation/)

# Intuition
This is a shortest path problem on indices.

From index `i`, possible moves:
1. Adjacent:
   - `i-1`
   - `i+1`
2. Prime teleport:
   - If `nums[i]` is prime `p`
   - Jump to every index `j` where `nums[j] % p == 0`

Since every jump costs 1, use **BFS**.

---

# Approach  

### Step 1: Precompute prime factors  
Use sieve:
$$
factors[x] = prime factors of x
$$

### Step 2: Build teleport graph  
For every index:
- If `nums[i]` itself is prime:
$$
edges[p].push(i)
$$
This means:
- Prime value `p` can teleport to all indices divisible by `p`

### Step 3: BFS  
Start from `n-1` and move backward toward `0`.

At each index:
- adjacent neighbors
- teleport neighbors via prime factors

To avoid repeated teleport processing:
$$
edges[p] = []
$$
after use.

---

# Complexity  
- **Time complexity:**
  $$O(n + M \log \log M)$$  
  where $(M = 10^6)$

- **Space complexity:**
  $$O(n + M)$$  

---

# Code  
```typescript
const MX = 1000001;

// prime factors sieve
const factors: number[][] = Array.from({ length: MX }, () => []);

for (let i = 2; i < MX; i++) {
    if (factors[i].length === 0) {
        for (let j = i; j < MX; j += i) {
            factors[j].push(i);
        }
    }
}

function minJumps(nums: number[]): number {
    const n = nums.length;

    // prime -> indices having that prime value
    const edges = new Map<number, number[]>();

    for (let i = 0; i < n; i++) {
        const val = nums[i];

        // prime check
        if (factors[val].length === 1 && factors[val][0] === val) {
            if (!edges.has(val)) edges.set(val, []);
            edges.get(val)!.push(i);
        }
    }

    const seen = new Array(n).fill(false);
    seen[n - 1] = true;

    let queue = [n - 1];
    let steps = 0;

    while (queue.length > 0) {
        const next: number[] = [];

        for (const i of queue) {
            if (i === 0) return steps;

            // adjacent left
            if (i > 0 && !seen[i - 1]) {
                seen[i - 1] = true;
                next.push(i - 1);
            }

            // adjacent right
            if (i < n - 1 && !seen[i + 1]) {
                seen[i + 1] = true;
                next.push(i + 1);
            }

            // teleport using prime factors
            for (const p of factors[nums[i]]) {
                const list = edges.get(p);

                if (list) {
                    for (const j of list) {
                        if (!seen[j]) {
                            seen[j] = true;
                            next.push(j);
                        }
                    }

                    // avoid repeated processing
                    edges.set(p, []);
                }
            }
        }

        queue = next;
        steps++;
    }

    return -1;
};

```
