[2615. Sum of Distances](https://leetcode.com/problems/sum-of-distances/)


# Intuition

For each value, consider all its indices.
Instead of computing pairwise distances $(O(n^2))$, use prefix sums.

For a sorted list of indices:

* Left contribution and right contribution can be computed in (O(1))

---

# Approach

### Step 1: Group indices

Map each value → list of indices

### Step 2: For each group

Let indices = `[i0, i1, ..., ik]`

Use prefix sum:

```
prefix[j] = sum of indices[0..j-1]
```

For each position `j`:

* Left side:

  ```
  j * indices[j] - prefix[j]
  ```
* Right side:

  ```
  (prefix[k+1] - prefix[j+1]) - (k - j) * indices[j]
  ```

Total = left + right

---

# Complexity

* **Time complexity:**  $$O(n)$$

* **Space complexity:**  $$O(n)$$

---

# Code

```typescript
function distance(nums: number[]): number[] {
    const n = nums.length;
    const res = new Array(n).fill(0);

    const map = new Map<number, number[]>();

    // Step 1: group indices
    for (let i = 0; i < n; i++) {
        if (!map.has(nums[i])) map.set(nums[i], []);
        map.get(nums[i])!.push(i);
    }

    // Step 2: process each group
    for (const indices of map.values()) {
        const k = indices.length;
        if (k === 1) continue;

        const prefix = new Array(k + 1).fill(0);
        for (let i = 0; i < k; i++) {
            prefix[i + 1] = prefix[i] + indices[i];
        }

        for (let j = 0; j < k; j++) {
            const idx = indices[j];

            const left = j * idx - prefix[j];
            const right = (prefix[k] - prefix[j + 1]) - (k - j - 1) * idx;

            res[idx] = left + right;
        }
    }

    return res;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
[1,3,1,1,2]
```

Value 1 → indices `[0,2,3]`

For index 0:

* distances → |0-2| + |0-3| = 2 + 3 = 5

For index 2:

* |2-0| + |2-3| = 2 + 1 = 3

For index 3:

* |3-0| + |3-2| = 3 + 1 = 4
