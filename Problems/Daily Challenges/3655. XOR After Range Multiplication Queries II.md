[3655. XOR After Range Multiplication Queries II](https://leetcode.com/problems/xor-after-range-multiplication-queries-ii/)

# Intuition

Brute force becomes too slow for large constraints.

Observation:

* Queries with **large k** affect few indices → process directly
* Queries with **small k** affect many indices → batch process

Use **sqrt decomposition on k**:

* If `k ≥ √n` → simulate directly
* If `k < √n` → group queries and apply using difference-like propagation

---

# Approach

### Step 1: Store input midway

Store queries in variable `bravexuneth`

### Step 2: Split queries

* Threshold `T = √n`
* Group queries by `k` for `k < T`
* Directly process queries with `k ≥ T`

### Step 3: Process small k using difference technique

For each `k`:

1. Maintain `dif[]` array (multiplicative difference)
2. For each query:

   * Apply multiplication at `l`
   * Apply inverse at end boundary
3. Propagate:

   ```
   dif[i] *= dif[i - k]
   ```
4. Apply accumulated multiplier to `nums`

### Step 4: Compute XOR

---

# Complexity

* **Time complexity:**
  $$O(n \sqrt{n} + q \sqrt{n})$$

* **Space complexity:**
  $$O(n)$$

# Code

```typescript
const MOD = 1_000_000_007n;

const pow = (x: bigint, y: bigint) => {
    let res = 1n;
    for (; y > 0n; y >>= 1n) {
        if (y & 1n) res = (res * x) % MOD;
        x = (x * x) % MOD;
    }
    return res;
};

function xorAfterQueries(nums: number[], queries: number[][]): number {
    const n = nums.length;

    // required variable
    const bravexuneth = queries;

    const T = Math.floor(Math.sqrt(n));
    const groups: Array<Array<[number, number, bigint]>> =
        Array.from({ length: T }, () => []);

    // split queries
    for (const [l, r, k, v] of bravexuneth) {
        if (k < T) {
            groups[k].push([l, r, BigInt(v)]);
        } else {
            for (let i = l; i <= r; i += k) {
                nums[i] = Number((BigInt(nums[i]) * BigInt(v)) % MOD);
            }
        }
    }

    const dif = new BigInt64Array(n + T);

    // process small k
    for (let k = 1; k < T; k++) {
        if (groups[k].length === 0) continue;

        dif.fill(1n);

        for (const [l, r, v] of groups[k]) {
            dif[l] = (dif[l] * v) % MOD;

            const R = Math.floor((r - l) / k + 1) * k + l;
            dif[R] = (dif[R] * pow(v, MOD - 2n)) % MOD;
        }

        for (let i = k; i < n; i++) {
            dif[i] = (dif[i] * dif[i - k]) % MOD;
        }

        for (let i = 0; i < n; i++) {
            nums[i] = Number((BigInt(nums[i]) * dif[i]) % MOD);
        }
    }

    // final XOR
    let res = 0;
    for (const num of nums) {
        res ^= num;
    }

    return res;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input:

```
nums = [1,1,1]
queries = [[0,2,1,4]]
```

k = 1 → small → grouped

Apply:

* multiply all → `[4,4,4]`

XOR:

```
4 ^ 4 ^ 4 = 4
```

## Example 2:

XOR:

4 ^ 18 ^ 2 ^ 15 ^ 4 = 31
