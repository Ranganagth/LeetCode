[3225. Maximum Score From Grid Operations](https://leetcode.com/problems/maximum-score-from-grid-operations/)

# Intuition

Each column independently chooses a height `h` (how many cells from top become black).

A white cell `(i, j)` contributes **only if**:

* It remains white → `i ≥ h[j]`
* At least one adjacent column has black at that row:

  * left column height `> i` OR right column height `> i`

So interaction exists **only between adjacent columns** → DP across columns.

---

# Approach

### State

```
dp[c][currH][prevH]
```

* `c` → current column
* `currH` → chosen height for column `c`
* `prevH` → height of previous column

### Transition logic

Two cases:

#### Case 1: `currH ≤ prevH`

Left column is taller → contributes:

```
rows: [currH, prevH)
```

Contribution:

```
colSum[c][prevH] - colSum[c][currH]
```

#### Case 2: `currH > prevH`

Current column is taller → contribution comes from previous column:

```
rows: [prevH, currH)
```

Contribution:

```
colSum[c-1][currH] - colSum[c-1][prevH]
```

### Optimization

Use:

* `prevMax` → prefix max
* `prevSuffixMax` → suffix max

to avoid $(O(n^3))$ transitions.

---

# Complexity

* **Time complexity:**  $$O(n^3)$$ (optimized constants via prefix/suffix max)

* **Space complexity:**  $$O(n^2)$$

# Code

```typescript
function maximumScore(grid: number[][]): number {
    const n = grid.length;
    if (n === 1) return 0;

    const dp: number[][][] = [];
    const prevMax = Array.from({ length: n + 1 }, () => Array(n + 1).fill(0));
    const prevSuffixMax = Array.from({ length: n + 1 }, () => Array(n + 1).fill(0));

    const colSum = Array.from({ length: n }, () => Array(n + 1).fill(0));

    // prefix sum per column
    for (let c = 0; c < n; c++) {
        dp[c] = Array.from({ length: n + 1 }, () => Array(n + 1).fill(0));
        for (let r = 1; r <= n; r++) {
            colSum[c][r] = colSum[c][r - 1] + grid[r - 1][c];
        }
    }

    for (let c = 1; c < n; c++) {
        for (let currH = 0; currH <= n; currH++) {
            for (let prevH = 0; prevH <= n; prevH++) {

                if (currH <= prevH) {
                    const gain = colSum[c][prevH] - colSum[c][currH];
                    dp[c][currH][prevH] =
                        prevSuffixMax[prevH][0] + gain;
                } else {
                    const gain = colSum[c - 1][currH] - colSum[c - 1][prevH];
                    dp[c][currH][prevH] = Math.max(
                        prevSuffixMax[prevH][currH],
                        prevMax[prevH][currH] + gain
                    );
                }
            }
        }

        // build prefix max
        for (let currH = 0; currH <= n; currH++) {
            prevMax[currH][0] = dp[c][currH][0];
            for (let prevH = 1; prevH <= n; prevH++) {
                const penalty =
                    prevH > currH ? colSum[c][prevH] - colSum[c][currH] : 0;

                prevMax[currH][prevH] = Math.max(
                    prevMax[currH][prevH - 1],
                    dp[c][currH][prevH] - penalty
                );
            }

            // suffix max
            prevSuffixMax[currH][n] = dp[c][currH][n];
            for (let prevH = n - 1; prevH >= 0; prevH--) {
                prevSuffixMax[currH][prevH] = Math.max(
                    prevSuffixMax[currH][prevH + 1],
                    dp[c][currH][prevH]
                );
            }
        }
    }

    let ans = 0;
    for (let k = 0; k <= n; k++) {
        ans = Math.max(ans, dp[n - 1][n][k], dp[n - 1][0][k]);
    }

    return ans;
};

```
