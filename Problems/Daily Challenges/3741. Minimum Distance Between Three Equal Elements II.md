[3741. Minimum Distance Between Three Equal Elements II](https://leetcode.com/problems/minimum-distance-between-three-equal-elements-ii/)

# Intuition

For any triple `(i, j, k)` with `i < j < k`:
$$
|i - j| + |j - k| + |k - i| = (j - i) + (k - j) + (k - i) = 2 \cdot (k - i)
$$

The middle index `j` cancels out.
Minimizing distance reduces to minimizing **(k - i)** for any three equal values.

Thus:
* For each value, pick 3 indices as close as possible
* Best choice is always **3 consecutive occurrences**

---

# Approach

1. Traverse array and group indices by value
2. For each value:

   * If occurrences ≥ 3:
     * Iterate over consecutive triplets:
$$
       indices[i], indices[i+1], indices[i+2]
       $$
     * Compute:
       $$
       distance = 2 * (indices[i+2] - indices[i])
		      $$
		      - Track minimum
3. Return minimum or `-1` if none found

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:**
  $$O(n)$$

---

# Code

```typescript
function minimumDistance(nums: number[]): number {
    const map = new Map<number, number[]>();

    // Step 1: collect indices
    for (let i = 0; i < nums.length; i++) {
        if (!map.has(nums[i])) map.set(nums[i], []);
        map.get(nums[i])!.push(i);
    }

    let ans = Infinity;

    // Step 2: process each group
    for (const indices of map.values()) {
        if (indices.length < 3) continue;

        for (let i = 0; i + 2 < indices.length; i++) {
            const dist = 2 * (indices[i + 2] - indices[i]);
            if (dist < ans) ans = dist;
        }
    }

    return ans === Infinity ? -1 : ans;
};

```
