[2812. Find the Safest Path in a Grid](https://leetcode.com/problems/find-the-safest-path-in-a-grid/)

# Intuition

This problem requires finding a path from (0, 0) to (n-1, n-1) that **maximizes the minimum Manhattan distance** to any thief.

Key insights:
1. **Multi-source BFS**: First, compute the minimum Manhattan distance from each cell to the nearest thief using multi-source BFS starting from all thief cells simultaneously.
2. **Binary search + BFS/DFS**: Use binary search on the safeness factor. For each candidate safeness value, check if there's a valid path from (0, 0) to (n-1, n-1) where every cell has a distance to the nearest thief >= the candidate value.
3. **Alternative: 0-1 BFS or Dijkstra**: Use a priority queue to explore cells in order of their safeness factor (greedy approach).

The most efficient approach is **0-1 BFS with a deque** or **Dijkstra's algorithm** to find the path with maximum minimum safeness.

---

# Approach

1. **Step 1: Multi-source BFS to compute safeness for each cell**
    
    - Initialize a queue with all thief positions.
    - Perform BFS to compute the minimum Manhattan distance from each cell to the nearest thief.
2. **Step 2: Find maximum safeness path using 0-1 BFS or Dijkstra**
    
    - Use a max-heap (priority queue) to explore cells in decreasing order of safeness.
    - Start from (0, 0) and track the minimum safeness along the path.
    - When reaching (n-1, n-1), return the safeness factor.

---

# Complexity

- **Time complexity:** $$O(n^2 \log n)$$ for multi-source BFS + Dijkstra with priority queue.
    
- **Space complexity:** $$O(n^2)$$ for the distance matrix and priority queue.

# Code

```typescript
function maximumSafenessFactor(grid: number[][]): number {
    const n = grid.length;

    // Step 1: Multi-source BFS to compute safeness
    const safeness = Array(n)
        .fill(null)
        .map(() => Array(n).fill(-1));
    const queue: [number, number][] = [];

    for (let i = 0; i < n; i++) {
        for (let j = 0; j < n; j++) {
            if (grid[i][j] === 1) {
                safeness[i][j] = 0;
                queue.push([i, j]);
            }
        }
    }

    const directions = [[0, 1], [0, -1], [1, 0], [-1, 0]];
    let idx = 0;
    while (idx < queue.length) {
        const [r, c] = queue[idx++];
        for (const [dr, dc] of directions) {
            const nr = r + dr;
            const nc = c + dc;
            if (nr >= 0 && nr < n && nc >= 0 && nc < n && safeness[nr][nc] === -1) {
                safeness[nr][nc] = safeness[r][c] + 1;
                queue.push([nr, nc]);
            }
        }
    }

    // Step 2: Binary search on safeness factor
    const canReach = (minSafeness: number): boolean => {
        if (safeness[0][0] < minSafeness) return false;

        const visited = Array(n)
            .fill(null)
            .map(() => Array(n).fill(false));
        const bfsQueue: [number, number][] = [[0, 0]];
        visited[0][0] = true;

        let bfsIdx = 0;
        while (bfsIdx < bfsQueue.length) {
            const [r, c] = bfsQueue[bfsIdx++];

            if (r === n - 1 && c === n - 1) return true;

            for (const [dr, dc] of directions) {
                const nr = r + dr;
                const nc = c + dc;
                if (
                    nr >= 0 &&
                    nr < n &&
                    nc >= 0 &&
                    nc < n &&
                    !visited[nr][nc] &&
                    safeness[nr][nc] >= minSafeness
                ) {
                    visited[nr][nc] = true;
                    bfsQueue.push([nr, nc]);
                }
            }
        }

        return false;
    };

    // Binary search
    let left = 0,
        right = 2 * n;
    let result = 0;

    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        if (canReach(mid)) {
            result = mid;
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    return result;
};

```

# Explanation

- **Multi-source BFS (Step 1):** Computes the minimum distance from each cell to the nearest thief in O(n²).
- **Binary search (Step 2):** Searches for the maximum safeness factor in O(log(2n)) = O(log n) iterations.
- **Validation BFS (canReach):** For each candidate safeness, checks if a path exists using BFS in O(n²).
- **Overall complexity:** O(n² log n), which is much faster than the previous approach.
  
  
> This solution ensures that we efficiently compute the maximum safeness factor for the path from the top-left corner to the bottom-right corner of the grid using the two-step approach outlined above.
