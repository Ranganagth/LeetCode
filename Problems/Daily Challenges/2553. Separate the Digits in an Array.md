[2553. Separate the Digits in an Array](https://leetcode.com/problems/separate-the-digits-in-an-array/)

# Intuition

For each number:

* Extract digits in order
* Append them into the result array

Simplest way:

* Convert number to string
* Iterate characters

---

# Approach

1. Initialize empty array `answer`
2. Traverse every number in `nums`
3. Convert number to string
4. Push each digit into result array
5. Return result

---

# Complexity

* **Time complexity:**
  $$O(d)$$
  where `d` = total digits across all numbers

* **Space complexity:**
  $$O(d)$$

---

# Code

```typescript
function separateDigits(nums: number[]): number[] {
    const answer: number[] = [];

    for (const num of nums) {
        for (const ch of num.toString()) {
            answer.push(Number(ch));
        }
    }

    return answer;
};

```
