[3558. Number of Ways to Assign Edge Weights I](https://leetcode.com/problems/number-of-ways-to-assign-edge-weights-i/)

# Intuition
This solution is elegant and efficient. Let me break down the key insights:

**Core Idea:**
- The maximum depth of the tree tells us how many edges are in the path from node 1 to any deepest node.
- For a path with `m` edges, we need to count assignments where the sum is odd.
- A sum is odd ⟺ we have an **odd number of odd values (1's)**.
- Out of $2^m$ total assignments, exactly $2⁽ᵐ⁻¹⁾$ result in an odd sum.

**Why $2⁽ᵐ⁻¹⁾$?**
- For `m` edges, we can choose any subset to assign weight 1 (odd).
- Total subsets = $2^m$
- Subsets with odd cardinality = $2⁽ᵐ⁻¹⁾$ (by combinatorial symmetry)

**Algorithm:**
1. Build the adjacency list from edges.
2. Find the **maximum depth** of the tree using DFS from node 1.
3. Return $2⁽ᵐᵃˣᴰᵉᵖᵗʰ ⁻ ¹⁾  mod (10^9 + 7)$.

# Detailed Explanation

## Why Maximum Depth?
- All nodes at maximum depth are equidistant from node 1.
- The path length (number of edges) is the same for any node at maximum depth.
- So we just need to find the maximum depth.

## DFS Function

```typescript
const dfs = (g: number[][], x: number, f: number): number => {
    let maxDep: number = 0;
    for (const y of g[x]) {
        if (y === f) continue; // Skip parent
        maxDep = Math.max(maxDep, dfs(g, y, x) + 1);
    }
    return maxDep;
};

```
- Returns the maximum depth of the subtree rooted at `x`.
- `f` is the parent node (to avoid revisiting).
- For each child `y`, recursively find its max depth and add 1.

## Modular Exponentiation

```typescript
const qpow = (x: number, y: number): number => {
    let res: bigint = 1n;
    let base: bigint = BigInt(x);
    let exp: number = y;

    while (exp > 0) {
        if (exp & 1) res = (res * base) % MOD;
        base = (base * base) % MOD;
        exp >>= 1;
    }
    return Number(res);
};

```
- Computes `x^y mod MOD` efficiently in O(log y) time.
- Uses binary exponentiation (fast exponentiation).

# Complete Annotated Solution

```typescript
var assignEdgeWeights = function (edges: number[][]): number {
    const MOD: bigint = 1000000007n;

    // Fast modular exponentiation: computes (x^y) % MOD
    const qpow = (x: number, y: number): number => {
        let res: bigint = 1n;
        let base: bigint = BigInt(x);
        let exp: number = y;

        while (exp > 0) {
            if (exp & 1) {
                // If current bit is 1, multiply result by base
                res = (res * base) % MOD;
            }
            // Square the base and shift exponent right
            base = (base * base) % MOD;
            exp >>= 1;
        }
        return Number(res);
    };

    // DFS to find maximum depth of subtree rooted at x
    // f is the parent node (to avoid revisiting in undirected graph)
    const dfs = (g: number[][], x: number, f: number): number => {
        let maxDep: number = 0;
        for (const y of g[x]) {
            if (y === f) continue; // Skip parent
            // Recursively find max depth of child subtree
            maxDep = Math.max(maxDep, dfs(g, y, x) + 1);
        }
        return maxDep;
    };

    // Build adjacency list
    const n: number = edges.length + 1;
    const g: number[][] = Array.from({ length: n + 1 }, () => []);

    for (let i = 0; i < edges.length; i++) {
        const u: number = edges[i][0];
        const v: number = edges[i][1];
        g[u].push(v);
        g[v].push(u);
    }

    // Find maximum depth from node 1
    const maxDep: number = dfs(g, 1, 0);

    // Return 2^(maxDep - 1) mod MOD
    return qpow(2, maxDep - 1);
};

```

# Example Walkthrough

## Example 1: `edges = [[1,2]]`

```
Tree: 1 — 2
```

- Build graph: `g[1] = [2]`, `g[2] = [1]`
- DFS from node 1:
  - Visit node 1, check child 2
  - DFS(2, parent=1): no children → return 0
  - maxDep at node 1 = 0 + 1 = 1
- Answer: `2^(1-1) = 2^0 = 1`

---

## Example 2: `edges = [[1,2],[1,3],[3,4],[3,5]]`

```
Tree:
    1
   / \
  2 3
     / \
    4 5
```

- Build graph:
  - `g[1] = [2, 3]`
  - `g[2] = [1]`
  - `g[3] = [1, 4, 5]`
  - `g[4] = [3]`
  - `g[5] = [3]`

- DFS from node 1:
  - Visit node 1, check children 2 and 3
  - DFS(2, parent=1): no children → return 0
  - DFS(3, parent=1):
    - Check children 4 and 5
    - DFS(4, parent=3): no children → return 0
    - DFS(5, parent=3): no children → return 0
    - maxDep at node 3 = max(0+1, 0+1) = 1
    - return 1
  - maxDep at node 1 = max(0+1, 1+1) = 2
- Answer: `2^(2-1) = 2^1 = 2`

# Why This Works

**Correctness:**
- Maximum depth = number of edges in path to deepest node
- For `m` edges with weights 1 or 2, exactly `2^(m-1)` assignments give odd sum
- This is because we need an odd number of 1's, and C(m,1) + C(m,3) + ... = 2^(m-1)

**Efficiency:**
- Single DFS pass to find max depth
- Fast exponentiation for large exponents
- Handles modulo arithmetic with BigInt to prevent overflow
