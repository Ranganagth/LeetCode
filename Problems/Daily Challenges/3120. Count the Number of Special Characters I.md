[3120. Count the Number of Special Characters I](https://leetcode.com/problems/count-the-number-of-special-characters-i/)

# Intuition

To find the number of **special letters**-letters that appear in both lowercase and uppercase-we need to identify characters that occur in both forms within the string. Using sets to track lowercase and uppercase characters separately allows efficient lookups and comparisons.

---

# Approach

1. Initialize two sets:
   - `lowercaseSet` to store lowercase characters.
   - `uppercaseSet` to store uppercase characters.
2. Iterate through each character in `word`:
   - If the character is lowercase, add it to `lowercaseSet`.
   - If uppercase, add its lowercase equivalent to `uppercaseSet`.
3. Count how many characters are present in both sets, which indicates they are **special**.

---

# Complexity

- **Time complexity:** $$O(n)$$, where \( n \) is the length of `word`, since each character is processed once.
- **Space complexity:**  $$O(26)$$, since only English alphabet characters are involved, making the sets at most size 26.

---

# Code

```typescript
function numberOfSpecialChars(word: string): number {
    const lowercaseSet = new Set<string>();
    const uppercaseSet = new Set<string>();

    for (const ch of word) {
        if (ch >= 'a' && ch <= 'z') {
            lowercaseSet.add(ch);
        } else if (ch >= 'A' && ch <= 'Z') {
            uppercaseSet.add(ch.toLowerCase());
        }
    }

    let count = 0;
    for (const ch of lowercaseSet) {
        if (uppercaseSet.has(ch)) {
            count++;
        }
    }

    return count;
};

```

# Example walkthrough

## Example 1:
`word = "aaAbcBC"`

| Character | Set updates | lowercaseSet | uppercaseSet | Explanation |
|------------|--------------|--------------|--------------|--------------|
| 'a' | add to lowercaseSet | {'a'} | {} | 'a' lowercase |
| 'a' | already in set | {'a'} | {} | duplicate, ignore |
| 'A' | add 'a' to uppercaseSet | {'a'} | {'a'} | 'A' uppercase, add 'a' to uppercaseSet |
| 'b' | add to lowercaseSet | {'a', 'b'} | {'a'} | 'b' lowercase |
| 'c' | add to lowercaseSet | {'a', 'b', 'c'} | {'a'} | 'c' lowercase |
| 'B' | add 'b' to uppercaseSet | {'a', 'b', 'c'} | {'a', 'b'} | 'B' uppercase |
| 'C' | add 'c' to uppercaseSet | {'a', 'b', 'c'} | {'a', 'b', 'c'} | 'C' uppercase |

Count characters in `lowercaseSet` also in `uppercaseSet`: `'a'`, `'b'`, `'c'` → total **3**.

**Output:** `3`

---

## Example 2:
`word = "abc"`

No uppercase characters, so the result is `0`.

## Example 3:
`word = "abBCab"`

| 'a' | add to lowercaseSet | {'a'} | {} |
| 'b' | add to lowercaseSet | {'a', 'b'} | {} |
| 'B' | add 'b' to uppercaseSet | {'a', 'b'} | {'b'} |
| 'C' | add to lowercaseSet | {'a', 'b', 'c'} | {'b'} |
| 'a' | already in lowercaseSet | {'a', 'b', 'c'} | {'b'} |
| 'b' | already in lowercaseSet | {'a', 'b', 'c'} | {'b'} |

Characters in both sets: `'b'` → total **1**.

**Output:** `1`
