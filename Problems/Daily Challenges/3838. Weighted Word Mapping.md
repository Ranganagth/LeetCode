[3838. Weighted Word Mapping](https://leetcode.com/problems/weighted-word-mapping/)

# Intuition
The problem requires us to:
1. Calculate the total weight of each word by summing the weights of its characters.
2. Map each weight to a character using reverse alphabetical order (0 → 'z', 1 → 'y', ..., 25 → 'a').
3. Concatenate all mapped characters.

The key insight is that the mapping is simply: `modulo result → character at position (25 - modulo)` in the alphabet, or equivalently, `'z' - modulo`.

---

# Approach
1. **Iterate through each word** in the `words` array.
2. **Calculate total weight** for each word:
   - For each character in the word, get its weight from the `weights` array.
   - Sum all character weights.
3. **Apply modulo 26** to the total weight.
4. **Map to character** using reverse alphabetical order:
   - Convert the modulo result to a character: `String.fromCharCode('z'.charCodeAt(0) - modulo)`
   - This maps 0 → 'z', 1 → 'y', ..., 25 → 'a'.
5. **Concatenate** all mapped characters to form the result string.

---

# Complexity
- **Time complexity:** 
  $$O(n \times m)$$ where `n` is the number of words and `m` is the average length of each word. We iterate through each character of each word once.

- **Space complexity:** 
  $$O(1)$$ excluding the output string. We only use a constant amount of extra space for variables.

---

# Code

```typescript
function mapWordWeights(words: string[], weights: number[]): string {
    let result = '';

    for (const word of words) {
        // Calculate total weight of the word
        let totalWeight = 0;
        for (const ch of word) {
            // Get the character index (0-25 for 'a'-'z')
            const charIndex = ch.charCodeAt(0) - 'a'.charCodeAt(0);
            totalWeight += weights[charIndex];
        }

        // Apply modulo 26
        const modVal = totalWeight % 26;

        // Map to character using reverse alphabetical order
        // 0 -> 'z', 1 -> 'y', ..., 25 -> 'a'
        const mappedChar = String.fromCharCode('z'.charCodeAt(0) - modVal);
        result += mappedChar;
    }

    return result;
};

```

# Example Walkthrough

## Example 1:

`words = ["abcd","def","xyz"]`, `weights = [5,3,12,14,1,2,3,2,10,6,6,9,7,8,7,10,8,9,6,9,9,8,3,7,7,2]`

**Word "abcd":**
- 'a' (index 0): weight = 5
- 'b' (index 1): weight = 3
- 'c' (index 2): weight = 12
- 'd' (index 3): weight = 14
- Total = 5 + 3 + 12 + 14 = 34
- 34 % 26 = 8
- Mapped char = 'z' - 8 = 'r' 

**Word "def":**
- 'd' (index 3): weight = 14
- 'e' (index 4): weight = 1
- 'f' (index 5): weight = 2
- Total = 14 + 1 + 2 = 17
- 17 % 26 = 17
- Mapped char = 'z' - 17 = 'i' 

**Word "xyz":**
- 'x' (index 23): weight = 7
- 'y' (index 24): weight = 7
- 'z' (index 25): weight = 2
- Total = 7 + 7 + 2 = 16
- 16 % 26 = 16
- Mapped char = 'z' - 16 = 'j' 

**Output:** `"rij"`
