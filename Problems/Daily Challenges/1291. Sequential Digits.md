[1291. Sequential Digits](https://leetcode.com/problems/sequential-digits/description/?envType=daily-question&envId=2026-07-13)

# Intuition

The key insight is that there are only a limited number of sequential digit numbers possible. Since we only have digits 0-9, the longest sequential number is `123456789` (9 digits).

We can generate all possible sequential digit numbers by:
- Starting with each digit 1-9
- Continuously appending the next sequential digit
- Checking if the number falls within our range

---

# Approach

## Strategy 1: Generate All Sequential Numbers

1. **Iterate through starting digits** (1-9)
2. **For each starting digit**, build sequential numbers by appending consecutive digits
3. **Stop when** the next digit would be > 9 or the number exceeds `high`
4. **Collect valid numbers** that fall within [low, high]
5. **Sort the result** (though generation order naturally produces sorted output)

## Strategy 2: BFS/Queue Approach (Alternative)

Use a queue to generate numbers level by level, starting from single digits and expanding.

---

# Complexity

- **Time complexity:** $$O(1)$$ - There are at most 36 sequential digit numbers possible (1+2+3+4+5+6+7+8+9 = 45 combinations, but many exceed 10^9)

- **Space complexity:** $$O(1)$$ - The result array has at most ~36 elements

---

# Code

```typescript
function sequentialDigits(low: number, high: number): number[] {
    const result: number[] = [];
    
    // Try each starting digit from 1 to 9
    for (let start = 1; start <= 9; start++) {
        let num = start;
        let nextDigit = start + 1;
        
        // Keep appending consecutive digits
        while (nextDigit <= 9) {
            num = num * 10 + nextDigit;
            
            // If number is within range, add to result
            if (num >= low && num <= high) {
                result.push(num);
            }
            
            // If number exceeds high, no point continuing
            if (num > high) {
                break;
            }
            
            nextDigit++;
        }
    }
    
    // Sort the result (numbers are generated in a specific order)
    result.sort((a, b) => a - b);
    
    return result;
};

```

## Alternative Approach: String-Based Generation

```typescript
function sequentialDigits(low: number, high: number): number[] {
    const result: number[] = [];
    const sequence = "123456789";
    
    // Try all possible lengths
    for (let length = 2; length <= 9; length++) {
        // Try all starting positions
        for (let start = 0; start <= 9 - length; start++) {
            const numStr = sequence.substring(start, start + length);
            const num = parseInt(numStr);
            
            if (num >= low && num <= high) {
                result.push(num);
            }
        }
    }
    
    return result;
};

```

# Example Walkthrough

## Example 1:

**Input:** `low = 100, high = 300`

**Explanation:**

**Generation Process:**

Starting digit = 1:
- 1 → 12 (< 100, skip)
- 12 → 123  (100 ≤ 123 ≤ 300)
- 123 → 1234 (> 300, stop)

Starting digit = 2:
- 2 → 23 (< 100, skip)
- 23 → 234  (100 ≤ 234 ≤ 300)
- 234 → 2345 (> 300, stop)

Starting digit = 3:
- 3 → 34 (< 100, skip)
- 34 → 345 (> 300, stop)

Starting digits 4-9: All generate numbers > 300

**Output:** `[123, 234]`

## Key Insights

- **No complex algorithms needed** - Simple generation suffices
- **Natural ordering** - Generation by length produces sorted output
- **Efficient** - O(1) time complexity due to fixed upper bound
