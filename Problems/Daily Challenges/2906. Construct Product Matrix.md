[2906. Construct Product Matrix](https://leetcode.com/problems/construct-product-matrix/)

# Intuition

Direct multiplication for each cell leads to repeated work: for every element, recomputing product of all others results in $(O((nm)^2))$.

Flatten the grid into a 1D array and reuse the idea of **product of array except self** using prefix and suffix products. This avoids division (important due to modulo and zeros).

---

# Approach

1. Flatten the 2D grid into a 1D array `arr` of size `k = n * m`.
2. Create two arrays:

   * `prefix[i]` = product of all elements before index `i` (mod 12345)
   * `suffix[i]` = product of all elements after index `i` (mod 12345)
3. For each index `i`:

   * result[i] = (prefix[i] * suffix[i]) % 12345
 
1. Convert the result array back into 2D form.

Key points:

* No division used → works even with zeros.
* Modular multiplication applied at each step to avoid overflow.

---

# Complexity

* **Time complexity:**
  $$O(n \cdot m)$$

* **Space complexity:**
  $$O(n \cdot m)$$

---

# Code

```typescript []
function constructProductMatrix(grid: number[][]): number[][] {
    const MOD = 12345;
    const n = grid.length;
    const m = grid[0].length;

    const size = n * m;
    const arr: number[] = new Array(size);

    // Flatten grid
    let idx = 0;
    for (let i = 0; i < n; i++) {
        for (let j = 0; j < m; j++) {
            arr[idx++] = grid[i][j] % MOD;
        }
    }

    const prefix: number[] = new Array(size).fill(1);
    const suffix: number[] = new Array(size).fill(1);

    // Prefix product
    for (let i = 1; i < size; i++) {
        prefix[i] = (prefix[i - 1] * arr[i - 1]) % MOD;
    }

    // Suffix product
    for (let i = size - 2; i >= 0; i--) {
        suffix[i] = (suffix[i + 1] * arr[i + 1]) % MOD;
    }

    // Result array
    const result: number[] = new Array(size);
    for (let i = 0; i < size; i++) {
        result[i] = (prefix[i] * suffix[i]) % MOD;
    }

    // Convert back to 2D
    const resGrid: number[][] = [];
    idx = 0;
    for (let i = 0; i < n; i++) {
        const row: number[] = [];
        for (let j = 0; j < m; j++) {
            row.push(result[idx++]);
        }
        resGrid.push(row);
    }

    return resGrid;
};

```

# Example Walkthrough

## Example 1: Step by step explanation

Input: `[[1,2],[3,4]]`

Flatten → `[1,2,3,4]`

Prefix:

* prefix[0] = 1
* prefix[1] = 1
* prefix[2] = 1×2 = 2
* prefix[3] = 2×3 = 6

→ `[1,1,2,6]`

Suffix:

* suffix[3] = 1
* suffix[2] = 4
* suffix[1] = 4×3 = 12
* suffix[0] = 12×2 = 24

→ `[24,12,4,1]`

Result:

* index 0 → 1×24 = 24
* index 1 → 1×12 = 12
* index 2 → 2×4 = 8
* index 3 → 6×1 = 6

Convert back:
`[[24,12],[8,6]]`

---

## Example 2:

Input: `[[12345],[2],[1]]`

Flatten → `[12345, 2, 1]`

After modulo:
`[0,2,1]`

Prefix → `[1,0,0]`
Suffix → `[2,1,1]`

Result:

* index 0 → 1×2 = 2
* index 1 → 0×1 = 0
* index 2 → 0×1 = 0

Output:
`[[2],[0],[0]]`
