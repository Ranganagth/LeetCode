[1848. Minimum Distance to the Target Element](https://leetcode.com/problems/minimum-distance-to-the-target-element/)

# Intuition

Find all indices where `nums[i] == target`.
Among them, choose the one closest to `start`.

This is a direct scan problem: compute `abs(i - start)` and track minimum.

---

# Approach

1. Initialize `minDist = Infinity`
2. Traverse array:

   * If `nums[i] == target`:

     * Compute `abs(i - start)`
     * Update minimum
3. Return `minDist`

---

# Complexity

* **Time complexity:**
  $$O(n)$$

* **Space complexity:** $$O(1)$$

---

# Code

```typescript
function getMinDistance(nums: number[], target: number, start: number): number {
    let minDist = Infinity;

    for (let i = 0; i < nums.length; i++) {
        if (nums[i] === target) {
            const dist = Math.abs(i - start);
            if (dist < minDist) {
                minDist = dist;
            }
        }
    }

    return minDist;
};

```
