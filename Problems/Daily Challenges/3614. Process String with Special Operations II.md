[3614. Process String with Special Operations II](https://leetcode.com/problems/process-string-with-special-operations-ii/)

# Intuition

We cannot build the final string explicitly because its length can be up to 1e15. Instead, we:
- Track only the length of the result after each character in `s` during a forward pass.
- If `k` is out of bounds, return `'.'`.
- Otherwise, walk backward through `s` and map the index `k` back through each operation to an earlier state until it lands on a specific appended letter.

This reverse-index mapping avoids materializing the string while still finding the k-th character.

---

# Approach

1. Forward pass to compute `len[i]`: the length of the result after processing `s[0..i]`:
   - If letter: `len++`
   - If `*`: `len = max(0, len - 1)`
   - If `#`: `len = min(CAP, len * 2)` (cap to avoid overflow; CAP ≈ 1e15 + small margin)
   - If `%`: length unchanged
2. If `k >= final_length`, return `'.'`.
3. Backward pass (from the end of `s` to the start):
   - Let `Lcurr = len[i]`, `Lprev = (i > 0 ? len[i-1] : 0)`.
   - Map `k` back through the operation at `s[i]`:
     - Letter `c`: if `k == Lcurr - 1`, return `c`; else `k` unchanged.
     - `*`: a deletion from the end — any valid `k` stays the same.
     - `#`: result is `prev + prev` — if `k >= Lprev`, set `k -= Lprev`.
     - `%`: reverse — if `Lprev > 0`, set `k = Lprev - 1 - k`.
4. If no letter is returned by the end, return `'.'` (shouldn’t happen if `k` was in bounds).

---

# Complexity

- **Time complexity:**
  `O(n)` for two passes over `s`.

- **Space complexity:**
  `O(n)` for storing the length array.

---

# Code

```typescript
function processStr(s: string, k: number): string {
    const n = s.length;
    const CAP = 1_000_000_000_000 + 5; // 1e15 + small margin

    // Forward pass: compute length after each operation
    const len: number[] = new Array(n);
    let currLen = 0;
    for (let i = 0; i < n; i++) {
        const ch = s[i];
        if (ch >= 'a' && ch <= 'z') {
            currLen = Math.min(CAP, currLen + 1);
        } else if (ch === '*') {
            if (currLen > 0) currLen--;
        } else if (ch === '#') {
            currLen = Math.min(CAP, currLen * 2);
        } else if (ch === '%') {
            // length unchanged
        }
        len[i] = currLen;
    }

    const finalLen = currLen;
    if (k < 0 || k >= finalLen) return '.';

    // Backward pass: map k back through operations to the originating letter
    for (let i = n - 1; i >= 0; i--) {
        const ch = s[i];
        const Lcurr = len[i];
        const Lprev = i > 0 ? len[i - 1] : 0;

        if (ch >= 'a' && ch <= 'z') {
            // The letter appended at this step occupies position Lcurr - 1
            if (k === Lcurr - 1) {
                return ch;
            }
            // Otherwise, k refers to earlier content (unchanged)
        } else if (ch === '*') {
            // Deletion from the end; any valid k remains same index in previous
            // (no change to k)
        } else if (ch === '#') {
            // String became prev + prev
            if (Lprev > 0 && k >= Lprev) {
                k -= Lprev;
            }
        } else if (ch === '%') {
            // Reverse the previous string
            if (Lprev > 0) {
                k = Lprev - 1 - k;
            }
        }
    }

    return '.';
};

```

# Example walkthrough
## Example 1:
**Input:**
s = "a#b%*", k = 1

**Process/Approach explanation:**
- Forward lengths: after 'a' → 1, after '#' → 2, after 'b' → 3, after '%' → 3, after '*' → 2.
- finalLen = 2, k = 1 < 2 → valid.
- Backward mapping:
  - '*' → k stays 1.
  - '%' → k = 3 - 1 - 1 = 1.
  - 'b' → k != 2 → continue.
  - '#' → Lprev = 1, k >= 1 → k = 1 - 1 = 0.
  - 'a' → k == 0 == Lcurr - 1 → return 'a'.

**Output:**
"a"

## Example 2:
**Input:**
s = "cd%#*#", k = 3

**Process/Approach explanation:**
- Forward result evolution: "c"(1) → "cd"(2) → reverse "dc"(2) → duplicate "dcdc"(4) → delete last "dcd"(3) → duplicate "dcddcd"(6).
- finalLen = 6, k = 3 < 6 → valid.
- Backward mapping lands on the letter 'd'.

**Output:**
"d"
