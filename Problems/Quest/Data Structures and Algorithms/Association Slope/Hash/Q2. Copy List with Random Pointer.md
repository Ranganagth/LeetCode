# Intuition

The challenge is that we need to copy both `next` and `random` pointers, but the `random` pointer can point to any node in the list. We need a way to map original nodes to their copies.

There are two main approaches:
1. **Hash Map (Two Pass)**: First pass creates all nodes and stores the mapping, second pass sets up pointers.
2. **Interweaving (Three Pass)**: Interweave copied nodes with original nodes, then separate them.

The **hash map approach** is more intuitive and easier to understand.

---

# Approach 1: Hash Map (Recommended)

1. **First pass**: Create a mapping of original nodes to their copies using a hash map.
2. **Second pass**: Set up the `next` and `random` pointers for all copied nodes using the mapping.
3. Return the head of the copied list.

---

# Approach 2: Interweaving (O(1) Space)

1. **First pass**: Create copies and interweave them with original nodes.
   - Original: A → B → C
   - After: A → A' → B → B' → C → C'
2. **Second pass**: Set up random pointers for copied nodes.
3. **Third pass**: Separate the two lists.

---

# Complexity Analysis

## Approach 1 (Hash Map):
- **Time complexity:** $$O(n)$$ - two passes through the list.
- **Space complexity:** $$O(n)$$ - for the hash map.

## Approach 2 (Interweaving):
- **Time complexity:** $$O(n)$$ - three passes through the list.
- **Space complexity:** $$O(1)$$ - no extra data structures (excluding output).

---

# Implementation 1: Hash Map (Recommended)

```typescript
/**
 * Definition for _Node.
 * class _Node {
 * val: number
 * next: _Node | null
 * random: _Node | null
 * 
 * constructor(val?: number, next?: _Node, random?: _Node) {
 * this.val = (val===undefined ? 0 : val)
 * this.next = (next===undefined ? null : next)
 * this.random = (random===undefined ? null : random)
 * }
 * }
 */

function copyRandomList(head: _Node | null): _Node | null {
    if (head === null) {
        return null;
    }

    // Map to store original node -> copied node
    const map = new Map<_Node, _Node>();

    // First pass: create all nodes and store in map
    let current = head;
    while (current !== null) {
        map.set(current, new _Node(current.val));
        current = current.next;
    }

    // Second pass: set up next and random pointers
    current = head;
    while (current !== null) {
        const copy = map.get(current)!;
        copy.next = current.next !== null ? map.get(current.next)! : null;
        copy.random = current.random !== null ? map.get(current.random)! : null;
        current = current.next;
    }

    return map.get(head)!;
};

```

# Implementation 2: Interweaving (O(1) Space)

```typescript
/**
 * Definition for _Node.
 * class _Node {
 * val: number
 * next: _Node | null
 * random: _Node | null
 * 
 * constructor(val?: number, next?: _Node, random?: _Node) {
 * this.val = (val===undefined ? 0 : val)
 * this.next = (next===undefined ? null : next)
 * this.random = (random===undefined ? null : random)
 * }
 * }
 */

function copyRandomList(head: _Node | null): _Node | null {
    if (head === null) {
        return null;
    }

    // First pass: create copies and interweave
    let current = head;
    while (current !== null) {
        const copy = new _Node(current.val);
        copy.next = current.next;
        current.next = copy;
        current = copy.next;
    }

    // Second pass: set up random pointers
    current = head;
    while (current !== null) {
        if (current.random !== null) {
            current.next!.random = current.random.next;
        }
        current = current.next!.next;
    }

    // Third pass: separate the lists
    current = head;
    const copiedHead = head.next;
    let copiedCurrent = copiedHead;

    while (current !== null) {
        current.next = current.next!.next;
        copiedCurrent!.next = copiedCurrent!.next !== null ? copiedCurrent!.next.next : null;
        current = current.next;
        copiedCurrent = copiedCurrent!.next;
    }

    return copiedHead;
};

```

# Example walkthrough
## Example 1:
**Input:** `head = [[7,null],[13,0],[11,4],[10,2],[1,0]]`

### Using Hash Map Approach:

```
Original list:
7 (random: null) → 13 (random: 7) → 11 (random: 1) → 10 (random: 11) → 1 (random: 7)

First pass: Create copies
map = {
  7: 7',
  13: 13',
  11: 11',
  10: 10',
  1: 1'
}

Second pass: Set up pointers
7'.next = 13', 7'.random = null
13'.next = 11', 13'.random = 7'
11'.next = 10', 11'.random = 1'
10'.next = 1', 10'.random = 11'
1'.next = null, 1'.random = 7'

Copied list:
7' (random: null) → 13' (random: 7') → 11' (random: 1') → 10' (random: 11') → 1' (random: 7')
```

**Output:** `[[7,null],[13,0],[11,4],[10,2],[1,0]]`

## Example 2:
**Input:** `head = [[1,1],[2,1]]`

```
Original list:
1 (random: 2) → 2 (random: 2)

First pass: Create copies
map = {1: 1', 2: 2'}

Second pass: Set up pointers
1'.next = 2', 1'.random = 2'
2'.next = null, 2'.random = 2'

Copied list:
1' (random: 2') → 2' (random: 2')
```

**Output:** `[[1,1],[2,1]]`

## Example 3:
**Input:** `head = [[3,null],[3,0],[3,null]]`

```
Original list:
3 (random: null) → 3 (random: 3) → 3 (random: null)

First pass: Create copies
map = {3a: 3a', 3b: 3b', 3c: 3c'}

Second pass: Set up pointers
3a'.next = 3b', 3a'.random = null
3b'.next = 3c', 3b'.random = 3a'
3c'.next = null, 3c'.random = null

Copied list:
3' (random: null) → 3' (random: 3') → 3' (random: null)
```

**Output:** `[[3,null],[3,0],[3,null]]`

# Comparison

| Aspect | Hash Map | Interweaving |
|--------|----------|--------------|
| **Time** | O(n) | O(n) |
| **Space** | O(n) | O(1) |
| **Readability** | Clear | Complex |
| **Ease of Implementation** | Easy | Tricky |

---

# Recommendation

Use the **hash map approach** for interviews and production code. It's:
- Easy to understand and implement
- Less error-prone
- Commonly accepted in interviews

> The interweaving approach is good to know but harder to implement correctly without bugs.
