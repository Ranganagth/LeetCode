# Intuition

We need to find a subarray of length at least 2 whose sum is a multiple of k.

**Key insight:**
- If two prefix sums have the same remainder when divided by k, the subarray between them has a sum that is a multiple of k.
- Use a **hash map** to store the first occurrence of each remainder.
- For each position, check if the current remainder was seen before at a distance of at least 2.

**Why this works:**
- Let `prefixSum[i]` = sum of elements from index 0 to i.
- If `prefixSum[i] % k === prefixSum[j] % k` where `i > j`, then:
  - `(prefixSum[i] - prefixSum[j]) % k === 0`
  - This means the subarray from index `j+1` to `i` has a sum divisible by k.
  - The length is `i - j ≥ 2` (since we need at least 2 elements).

---

# Approach

1. Use a hash map to store `{remainder: firstIndex}`.
2. Initialize with `{0: -1}` to handle subarrays starting from index 0.
3. Iterate through the array:
   - Calculate the prefix sum modulo k.
   - If this remainder exists in the map and the distance is at least 2, return true.
   - If this remainder doesn't exist, add it to the map.
4. Return false if no valid subarray is found.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the array. Single pass with O(1) hash map operations.

- **Space complexity:** 
  $$O(\min(n, k))$$ for the hash map (at most k different remainders).

---

# Implementation

```typescript
function checkSubarraySum(nums: number[], k: number): boolean {
    // Map to store {remainder: firstIndex}
    const remainderMap = new Map<number, number>();
    remainderMap.set(0, -1); // Base case: remainder 0 at index -1

    let prefixSum = 0;

    for (let i = 0; i < nums.length; i++) {
        prefixSum += nums[i];
        const remainder = prefixSum % k;

        // Handle negative remainders (for negative prefix sums)
        const normalizedRemainder = remainder < 0 ? remainder + k : remainder;

        // Check if we've seen this remainder before
        if (remainderMap.has(normalizedRemainder)) {
            const prevIndex = remainderMap.get(normalizedRemainder)!;
            // Subarray length must be at least 2
            if (i - prevIndex >= 2) {
                return true;
            }
        } else {
            // Store the first occurrence of this remainder
            remainderMap.set(normalizedRemainder, i);
        }
    }

    return false;
};

```

## Example 1:
**Input:** `nums = [23,2,4,6,7], k = 6`

```
remainderMap = {0: -1}
prefixSum = 0

i = 0: nums[0] = 23
  prefixSum = 23
  remainder = 23 % 6 = 5
  remainderMap.has(5)? No
  remainderMap = {0: -1, 5: 0}

i = 1: nums[1] = 2
  prefixSum = 25
  remainder = 25 % 6 = 1
  remainderMap.has(1)? No
  remainderMap = {0: -1, 5: 0, 1: 1}

i = 2: nums[2] = 4
  prefixSum = 29
  remainder = 29 % 6 = 5
  remainderMap.has(5)? Yes, at index 0
  i - prevIndex = 2 - 0 = 2 ≥ 2
  return true
```

**Explanation:** Subarray [2, 4] from index 1 to 2 has sum 6, which is divisible by 6.
**Output:** `true`

# Edge Cases

## Edge Case 1: Single element
**Input:** `nums = [5], k = 3`
- No subarray of length ≥ 2, return false.

## Edge Case 2: Two elements with sum divisible by k
**Input:** `nums = [3, 3], k = 6`
```
i = 0: prefixSum = 3, remainder = 3, remainderMap = {0: -1, 3: 0}
i = 1: prefixSum = 6, remainder = 0
  remainderMap.has(0)? Yes, at index -1
  i - prevIndex = 1 - (-1) = 2 ≥ 2
  return true
```

## Edge Case 3: Zero in array
**Input:** `nums = [0, 0], k = 1`
```
i = 0: prefixSum = 0, remainder = 0
  remainderMap.has(0)? Yes, at index -1
  i - prevIndex = 0 - (-1) = 1 < 2 ✗

i = 1: prefixSum = 0, remainder = 0
  remainderMap.has(0)? Yes, at index -1
  i - prevIndex = 1 - (-1) = 2 ≥ 2
  return true
```

# Key Points

- **Prefix sum modulo**: Two equal remainders mean the subarray between them is divisible by k.
- **Base case**: Initialize with `{0: -1}` to handle subarrays starting from index 0.
- **Distance check**: Ensure the subarray length is at least 2 by checking `i - prevIndex >= 2`.
- **First occurrence**: Only store the first occurrence of each remainder to maximize subarray length.
- **Negative remainders**: Handle negative remainders by normalizing them.
