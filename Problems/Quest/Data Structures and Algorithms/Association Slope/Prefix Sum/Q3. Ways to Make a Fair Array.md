# Intuition

When we remove an element at index `i`, the indices of all elements after it shift. We need to check if the resulting array is fair (sum of even-indexed elements equals sum of odd-indexed elements).

**Key insight:**
- Precompute prefix sums for even and odd indices.
- For each removal, calculate the new even and odd sums using prefix sums.
- When we remove an element at index `i`:
  - Elements before index `i` keep their parity.
  - Elements after index `i` shift by one position, so their parity flips.

**Approach:**
1. Calculate prefix sums for even-indexed and odd-indexed elements.
2. For each index `i` to remove:
   - Calculate the sum of even-indexed elements in the remaining array.
   - Calculate the sum of odd-indexed elements in the remaining array.
   - Check if they're equal.

---

# Approach

1. **Precompute prefix sums:**
   - `evenPrefix[i]` = sum of elements at even indices up to index i.
   - `oddPrefix[i]` = sum of elements at odd indices up to index i.

2. **For each index i to remove:**
   - Elements before i: keep their parity.
   - Elements after i: their parity flips (even becomes odd, odd becomes even).
   - Calculate new even sum and odd sum.
   - Check if they're equal.

3. **Count valid removals.**

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the array. One pass to compute prefix sums, one pass to check each removal.

- **Space complexity:** 
  $$O(n)$$ for the prefix sum arrays.

# Implementation

```typescript
function waysToMakeFair(nums: number[]): number {
    const n = nums.length;

    // Compute prefix sums for even and odd indices
    const evenPrefix = new Array(n + 1).fill(0);
    const oddPrefix = new Array(n + 1).fill(0);

    for (let i = 0; i < n; i++) {
        evenPrefix[i + 1] = evenPrefix[i];
        oddPrefix[i + 1] = oddPrefix[i];

        if (i % 2 === 0) {
            evenPrefix[i + 1] += nums[i];
        } else {
            oddPrefix[i + 1] += nums[i];
        }
    }

    let count = 0;

    // Try removing each index
    for (let i = 0; i < n; i++) {
        // Sum of even-indexed elements in remaining array
        let newEvenSum = evenPrefix[i]; // Elements before i at even indices

        // Elements after i at odd indices become even indices
        newEvenSum += oddPrefix[n] - oddPrefix[i + 1];

        // Sum of odd-indexed elements in remaining array
        let newOddSum = oddPrefix[i]; // Elements before i at odd indices

        // Elements after i at even indices become odd indices
        newOddSum += evenPrefix[n] - evenPrefix[i + 1];

        if (newEvenSum === newOddSum) {
            count++;
        }
    }

    return count;
};

```

# Example walkthrough
## Example 1:
**Input:** `nums = [2,1,6,4]`

```
Indices: 0 1 2 3
Values: 2 1 6 4

Prefix sums:
evenPrefix = [0, 2, 2, 8, 8] (indices 0, 2)
oddPrefix = [0, 0, 1, 1, 5] (indices 1, 3)

Remove index 0 (value 2):
  newEvenSum = evenPrefix[0] + (oddPrefix[4] - oddPrefix[1])
             = 0 + (5 - 0) = 5
  newOddSum = oddPrefix[0] + (evenPrefix[4] - evenPrefix[1])
            = 0 + (8 - 2) = 6
  5 ≠ 6, not fair

Remove index 1 (value 1):
  newEvenSum = evenPrefix[1] + (oddPrefix[4] - oddPrefix[2])
             = 2 + (5 - 1) = 6
  newOddSum = oddPrefix[1] + (evenPrefix[4] - evenPrefix[2])
            = 0 + (8 - 2) = 6
  6 === 6, fair

Remove index 2 (value 6):
  newEvenSum = evenPrefix[2] + (oddPrefix[4] - oddPrefix[3])
             = 2 + (5 - 1) = 6
  newOddSum = oddPrefix[2] + (evenPrefix[4] - evenPrefix[3])
            = 1 + (8 - 8) = 1
  6 ≠ 1, not fair

Remove index 3 (value 4):
  newEvenSum = evenPrefix[3] + (oddPrefix[4] - oddPrefix[4])
             = 8 + (5 - 5) = 8
  newOddSum = oddPrefix[3] + (evenPrefix[4] - evenPrefix[4])
            = 1 + (8 - 8) = 1
  8 ≠ 1, not fair

Count = 1
```

**Output:** `1`

# Key Points

- **Prefix sums**: Precompute to avoid recalculating sums for each removal.
- **Parity flip**: Elements after the removed index flip their parity.
- **Formula**: 
  - New even sum = even elements before + odd elements after
  - New odd sum = odd elements before + even elements after
- **Efficiency**: O(n) time with two passes (one for prefix sums, one for checking).
