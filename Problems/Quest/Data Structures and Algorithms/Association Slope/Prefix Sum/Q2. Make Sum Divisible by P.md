# Intuition

We need to find the smallest subarray to remove such that the remaining sum is divisible by `p`.

**Key insight:**
- Let `totalSum` be the sum of all elements.
- If we remove a subarray with sum `S`, the remaining sum is `totalSum - S`.
- We want: `(totalSum - S) % p === 0`
- This means: `S % p === totalSum % p`

So we need to find the **shortest subarray** whose sum modulo `p` equals `totalSum % p`.

**Approach:**
Use a **prefix sum with hash map** to track remainders:
1. Calculate `target = totalSum % p`.
2. If `target === 0`, the sum is already divisible by `p`, return 0.
3. Use a hash map to store the most recent index of each remainder.
4. For each position, check if we can find a subarray ending at this position with the desired remainder.

# Approach

1. Calculate `totalSum` and `target = totalSum % p`.
2. If `target === 0`, return 0 (already divisible).
3. Use a hash map to store `{remainder: mostRecentIndex}`.
4. Iterate through the array:
   - Calculate prefix sum and its remainder.
   - Check if `(currentRemainder - target + p) % p` exists in the map.
   - If yes, we found a subarray with the desired remainder.
   - Update the minimum length.
   - Store the current remainder in the map.
5. Return the minimum length, or -1 if not found.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the array. Single pass with O(1) hash map operations.

- **Space complexity:** 
  $$O(p)$$ for the hash map (at most `p` different remainders).

# Implementation

```typescript
function minSubarray(nums: number[], p: number): number {
    // Calculate total sum and target remainder
    let totalSum = 0;
    for (const num of nums) {
        totalSum += num;
    }

    const target = totalSum % p;

    // If already divisible, no need to remove anything
    if (target === 0) {
        return 0;
    }

    // Map to store {remainder: mostRecentIndex}
    const remainderMap = new Map<number, number>();
    remainderMap.set(0, -1); // Base case: remainder 0 at index -1

    let minLength = nums.length;
    let prefixSum = 0;

    for (let i = 0; i < nums.length; i++) {
        prefixSum += nums[i];
        const currentRemainder = prefixSum % p;

        // We want to find a subarray with remainder === target
        // If prefixSum has remainder R, and we remove a subarray with remainder T,
        // the remaining sum has remainder (R - T) % p
        // We want (R - T) % p === 0, so T === R
        // But we want the subarray sum to have remainder === target
        // So we need: (currentRemainder - target + p) % p
        const needed = (currentRemainder - target + p) % p;

        if (remainderMap.has(needed)) {
            const prevIndex = remainderMap.get(needed)!;
            minLength = Math.min(minLength, i - prevIndex);
        }

        remainderMap.set(currentRemainder, i);
    }

    // If minLength is still nums.length, it means we need to remove the whole array
    return minLength === nums.length ? -1 : minLength;
};

```

# Example walkthrough
## Example 1:
**Input:** `nums = [3,1,4,2], p = 6`

```
totalSum = 3 + 1 + 4 + 2 = 10
target = 10 % 6 = 4

remainderMap = {0: -1}
minLength = 4

i = 0: nums[0] = 3
  prefixSum = 3
  currentRemainder = 3 % 6 = 3
  needed = (3 - 4 + 6) % 6 = 5
  remainderMap.has(5)? No
  remainderMap = {0: -1, 3: 0}

i = 1: nums[1] = 1
  prefixSum = 4
  currentRemainder = 4 % 6 = 4
  needed = (4 - 4 + 6) % 6 = 0
  remainderMap.has(0)? Yes, at index -1
  minLength = min(4, 1 - (-1)) = min(4, 2) = 2
  remainderMap = {0: -1, 3: 0, 4: 1}

i = 2: nums[2] = 4
  prefixSum = 8
  currentRemainder = 8 % 6 = 2
  needed = (2 - 4 + 6) % 6 = 4
  remainderMap.has(4)? Yes, at index 1
  minLength = min(2, 2 - 1) = min(2, 1) = 1
  remainderMap = {0: -1, 3: 0, 4: 1, 2: 2}

i = 3: nums[3] = 2
  prefixSum = 10
  currentRemainder = 10 % 6 = 4
  needed = (4 - 4 + 6) % 6 = 0
  remainderMap.has(0)? Yes, at index -1
  minLength = min(1, 3 - (-1)) = min(1, 4) = 1
  remainderMap = {0: -1, 3: 0, 4: 3, 2: 2}

Return 1
```

**Explanation:** Remove subarray `[4]` (index 2), remaining sum = 6, which is divisible by 6.
**Output:** `1`

# Key Points

- **Modulo arithmetic**: `(a - b + m) % m` handles negative remainders correctly.
- **Prefix sum**: Track cumulative sums to find subarrays efficiently.
- **Hash map**: Store the most recent index of each remainder for O(1) lookup.
- **Base case**: Initialize with `{0: -1}` to handle subarrays starting from index 0.
- **Edge case**: Return -1 if we need to remove the entire array.
