# Intuition

The problem asks us to find the highest altitude reached during the trip. Since we know:
- Starting altitude is 0
- Each `gain[i]` represents the altitude change between consecutive points

We can:
1. Track the current altitude as we accumulate gains.
2. Keep track of the maximum altitude seen so far.
3. Return the maximum.

---

# Approach

1. Initialize `currentAltitude = 0` (starting point).
2. Initialize `maxAltitude = 0` (starting altitude is the initial maximum).
3. For each gain value:
   - Add it to `currentAltitude`.
   - Update `maxAltitude` if the current altitude is higher.
4. Return `maxAltitude`.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the gain array. Single pass through the array.

- **Space complexity:** 
  $$O(1)$$ - only using two variables.

---

# Implementation
```typescript
function largestAltitude(gain: number[]): number {
    let currentAltitude = 0;
    let maxAltitude = 0;

    for (const g of gain) {
        currentAltitude += g;
        maxAltitude = Math.max(maxAltitude, currentAltitude);
    }

    return maxAltitude;
};

```

# Example walkthrough
## Example 1:
**Input:** `gain = [-5,1,5,0,-7]`

```
Start: currentAltitude = 0, maxAltitude = 0

g = -5:
  currentAltitude = 0 + (-5) = -5
  maxAltitude = max(0, -5) = 0

g = 1:
  currentAltitude = -5 + 1 = -4
  maxAltitude = max(0, -4) = 0

g = 5:
  currentAltitude = -4 + 5 = 1
  maxAltitude = max(0, 1) = 1

g = 0:
  currentAltitude = 1 + 0 = 1
  maxAltitude = max(1, 1) = 1

g = -7:
  currentAltitude = 1 + (-7) = -6
  maxAltitude = max(1, -6) = 1

Return 1
```

**Altitudes:** `[0, -5, -4, 1, 1, -6]`
**Output:** `1`

## Example 2:
**Input:** `gain = [-4,-3,-2,-1,4,3,2]`

```
Start: currentAltitude = 0, maxAltitude = 0

g = -4: currentAltitude = -4, maxAltitude = 0
g = -3: currentAltitude = -7, maxAltitude = 0
g = -2: currentAltitude = -9, maxAltitude = 0
g = -1: currentAltitude = -10, maxAltitude = 0
g = 4: currentAltitude = -6, maxAltitude = 0
g = 3: currentAltitude = -3, maxAltitude = 0
g = 2: currentAltitude = -1, maxAltitude = 0

Return 0
```

**Altitudes:** `[0, -4, -7, -9, -10, -6, -3, -1]`
**Output:** `0`

# Why the Solution is Excellent

- **Correct**: Handles all cases including negative gains and when max is the starting point.
- **Optimal**: O(n) time and O(1) space - can't do better.
- **Clean**: Simple, readable, and easy to understand.
- **Efficient**: Single pass, no unnecessary operations.

function largestAltitude(gain: number[]): number {
    let maxAltitude = 0;
    let altitude = 0;

        maxAltitude = Math.max(maxAltitude, altitude);
