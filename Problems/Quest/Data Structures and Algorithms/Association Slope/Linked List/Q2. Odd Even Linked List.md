# Intuition

We need to separate the linked list into two groups:
1. **Odd-indexed nodes** (1st, 3rd, 5th, ...) - indices 0, 2, 4, ...
2. **Even-indexed nodes** (2nd, 4th, 6th, ...) - indices 1, 3, 5, ...

Then concatenate the odd group followed by the even group.

**Key insight:**
- Maintain two pointers: one for the odd group and one for the even group.
- Also maintain the head of the even group to connect it later.
- Traverse the list and alternately add nodes to odd and even groups.
- Finally, connect the last odd node to the first even node.

---

# Approach

1. Handle edge cases: if the list is empty or has only one node, return as is.
2. Create two pointers: `odd` (starting at head) and `even` (starting at head.next).
3. Store the head of the even group (`evenHead`) to connect later.
4. Traverse the list:
   - Connect odd nodes together.
   - Connect even nodes together.
   - Move to the next odd and even nodes.
5. Connect the last odd node to the first even node.
6. Return the head.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the number of nodes. We traverse the list once.

- **Space complexity:** 
  $$O(1)$$ - constant extra space (only using pointers).

---

# Implementation

```typescript
/**
 * Definition for singly-linked list.
 * class ListNode {
 * val: number
 * next: ListNode | null
 * constructor(val?: number, next?: ListNode | null) {
 * this.val = (val===undefined ? 0 : val)
 * this.next = (next===undefined ? null : next)
 * }
 * }
 */

function oddEvenList(head: ListNode | null): ListNode | null {
    // Handle edge cases
    if (head === null || head.next === null) {
        return head;
    }

    // Initialize pointers
    let odd = head;
    let even = head.next;
    const evenHead = even; // Store the head of even group

    // Traverse and separate odd and even nodes
    while (even !== null && even.next !== null) {
        // Connect odd nodes
        odd.next = odd.next.next;
        odd = odd.next;

        // Connect even nodes
        even.next = even.next.next;
        even = even.next;
    }

    // Connect odd group to even group
    odd.next = evenHead;

    return head;
};

```

# Example walkthrough
## Example 1:
**Input:** `head = [1,2,3,4,5]`

```
Initial:
odd = 1, even = 2, evenHead = 2

Iteration 1:
  odd.next = odd.next.next → 1.next = 3
  odd = 3
  even.next = even.next.next → 2.next = 4
  even = 4
  List: 1 → 3 → 2 → 4 → 5

Iteration 2:
  odd.next = odd.next.next → 3.next = 5
  odd = 5
  even.next = even.next.next → 4.next = null
  even = null
  List: 1 → 3 → 5 → 2 → 4

Connect odd to even:
  odd.next = evenHead → 5.next = 2
  List: 1 → 3 → 5 → 2 → 4
```

**Output:** `[1,3,5,2,4]`

## Example 2:
**Input:** `head = [2,1,3,5,6,4,7]`

```
Initial:
odd = 2, even = 1, evenHead = 1

Iteration 1:
  odd.next = odd.next.next → 2.next = 3
  odd = 3
  even.next = even.next.next → 1.next = 5
  even = 5
  List: 2 → 3 → 1 → 5 → 6 → 4 → 7

Iteration 2:
  odd.next = odd.next.next → 3.next = 6
  odd = 6
  even.next = even.next.next → 5.next = 4
  even = 4
  List: 2 → 3 → 6 → 1 → 5 → 4 → 7

Iteration 3:
  odd.next = odd.next.next → 6.next = 7
  odd = 7
  even.next = even.next.next → 4.next = null
  even = null
  List: 2 → 3 → 6 → 7 → 1 → 5 → 4

Connect odd to even:
  odd.next = evenHead → 7.next = 1
  List: 2 → 3 → 6 → 7 → 1 → 5 → 4
```

**Output:** `[2,3,6,7,1,5,4]`

## Example 3 (Edge case):
**Input:** `head = [1]`

- head.next === null → return head

```

Loop condition: even !== null && even.next !== null

Connect odd to even:
  odd.next = evenHead → 1.next = 2

```

**Output:** `[1,2]`

# Key Points

- **Two pointers**: Maintain separate pointers for odd and even nodes.
- **Store evenHead**: Remember the head of the even group to connect later.
- **Skip nodes**: Use `node.next.next` to skip to the next odd/even node.
- **O(1) space**: No extra data structures, only pointers.
- **O(n) time**: Single pass through the list.

> This solution efficiently reorders the list while maintaining the relative order within each group.
