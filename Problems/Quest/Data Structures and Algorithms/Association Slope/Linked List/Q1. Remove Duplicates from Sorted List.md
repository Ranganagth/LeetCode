# Intuition

Since the linked list is **sorted**, duplicate elements are adjacent to each other. We can traverse the list and skip nodes that have the same value as the current node.

**Approach:**
- Start from the head.
- For each node, if the next node has the same value, skip it by updating the `next` pointer.
- Continue until we reach the end of the list.

---

# Approach

1. Handle edge case: if the list is empty, return `null`.
2. Start with the head node.
3. While the current node and its next node exist:
   - If the current node's value equals the next node's value, skip the next node by setting `current.next = current.next.next`.
   - Otherwise, move to the next node.
4. Return the head.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the number of nodes. We traverse the list once.

- **Space complexity:** 
  $$O(1)$$ - constant extra space (only using pointers).

---

# Implementation

```typescript
/**
 * Definition for singly-linked list.
 * class ListNode {
 * val: number
 * next: ListNode | null
 * constructor(val?: number, next?: ListNode | null) {
 * this.val = (val===undefined ? 0 : val)
 * this.next = (next===undefined ? null : next)
 * }
 * }
 */

function deleteDuplicates(head: ListNode | null): ListNode | null {
    // Handle empty list
    if (head === null) {
        return null;
    }

    let current = head;

    // Traverse the list
    while (current !== null && current.next !== null) {
        // If current node and next node have the same value
        if (current.val === current.next.val) {
            // Skip the next node
            current.next = current.next.next;
        } else {
            // Move to the next node
            current = current.next;
        }
    }

    return head;
};

```

# Example walkthrough
## Example 1:
**Input:** `head = [1,1,2]`

```
Initial: 1 → 1 → 2 → null

Step 1: current = 1 (first)
  current.val (1) === current.next.val (1)
  Skip: current.next = current.next.next
  List: 1 → 2 → null

Step 2: current = 1
  current.val (1) !== current.next.val (2)
  Move: current = current.next

Step 3: current = 2
  current.next === null
  Exit loop

Result: 1 → 2 → null
```

**Output:** `[1,2]`

# Key Points
- **No extra space**: We modify the list in-place by adjusting pointers.
- **Single pass**: We traverse the list only once.
- **Sorted property**: We rely on the fact that the list is sorted, so duplicates are adjacent.
- **Pointer manipulation**: When we find a duplicate, we skip it by setting `current.next = current.next.next`.

> This solution is optimal for this problem with O(n) time and O(1) space complexity.
