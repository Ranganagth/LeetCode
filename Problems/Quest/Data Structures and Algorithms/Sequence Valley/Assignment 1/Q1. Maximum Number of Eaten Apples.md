# Intuition

This is a **greedy problem with a priority queue (max-heap)**. The key insight is:

1. On each day, we can eat at most one apple.
2. We should prioritize eating apples that will **rot soonest** (earliest expiration date).
3. We use a **min-heap** (by expiration date) to always eat the apple closest to rotting.

**Strategy:**
- For each day, add newly grown apples to the heap with their expiration dates.
- Remove apples that have already rotted.
- Eat the apple with the earliest expiration date (greedy choice).
- Continue eating even after day n until we run out of apples or all remaining apples are rotten.

---

# Approach

1. Use a **min-heap** to store apples by their expiration date: `[expirationDate, count]`.
2. For each day from 0 to n-1:
   - Add the apples grown on that day to the heap with expiration date = `day + days[i]`.
   - Remove all apples that have expired (expiration date <= current day).
   - If there are apples available, eat one and decrement its count.
   - If count becomes 0, remove it from the heap.
3. After day n-1, continue eating apples until:
   - The heap is empty, OR
   - All remaining apples are rotten.

---

# Complexity

- **Time complexity:** 
  $$O(n \log n)$$, where `n` is the length of the arrays. Each apple is added and removed from the heap once.

- **Space complexity:** 
  $$O(n)$$ for the heap in the worst case.

# Implementation

```typescript
function eatenApples(apples: number[], days: number[]): number {
    const n = apples.length;
    let eaten = 0;
    let day = 0;

    // Min-heap: [expirationDate, count]
    // We use an array and manually manage the heap
    const heap: [number, number][] = [];

    const heapPush = (expiration: number, count: number) => {
        heap.push([expiration, count]);
        let idx = heap.length - 1;
        while (idx > 0) {
            const parentIdx = Math.floor((idx - 1) / 2);
            if (heap[parentIdx][0] <= heap[idx][0]) break;
            [heap[parentIdx], heap[idx]] = [heap[idx], heap[parentIdx]];
            idx = parentIdx;
        }
    };

    const heapPop = (): [number, number] | null => {
        if (heap.length === 0) return null;
        const min = heap[0];
        const last = heap.pop()!;
        if (heap.length > 0) {
            heap[0] = last;
            let idx = 0;
            while (true) {
                const leftIdx = 2 * idx + 1;
                const rightIdx = 2 * idx + 2;
                let smallestIdx = idx;

                if (leftIdx < heap.length && heap[leftIdx][0] < heap[smallestIdx][0]) {
                    smallestIdx = leftIdx;
                }
                if (rightIdx < heap.length && heap[rightIdx][0] < heap[smallestIdx][0]) {
                    smallestIdx = rightIdx;
                }
                if (smallestIdx === idx) break;
                [heap[smallestIdx], heap[idx]] = [heap[idx], heap[smallestIdx]];
                idx = smallestIdx;
            }
        }
        return min;
    };

    // Process days 0 to n-1
    for (day = 0; day < n; day++) {
        // Add apples grown on this day
        if (apples[day] > 0) {
            heapPush(day + days[day], apples[day]);
        }

        // Remove expired apples
        while (heap.length > 0 && heap[0][0] <= day) {
            heapPop();
        }

        // Eat one apple if available
        if (heap.length > 0) {
            const [expiration, count] = heapPop()!;
            eaten++;
            if (count > 1) {
                heapPush(expiration, count - 1);
            }
        }
    }

    // Continue eating after day n-1
    day = n;
    while (heap.length > 0) {
        // Remove expired apples
        while (heap.length > 0 && heap[0][0] <= day) {
            heapPop();
        }

        if (heap.length === 0) break;

        // Eat one apple
        const [expiration, count] = heapPop()!;
        eaten++;
        if (count > 1) {
            heapPush(expiration, count - 1);
        }
        day++;
    }

    return eaten;
};

```

# Simplified Implementation (Using Array as Heap)

```typescript
function eatenApples(apples: number[], days: number[]): number {
    const n = apples.length;
    let eaten = 0;

    // Min-heap: [expirationDate, count]
    const heap: [number, number][] = [];

    const push = (exp: number, cnt: number) => {
        heap.push([exp, cnt]);
        let i = heap.length - 1;
        while (i > 0) {
            const p = Math.floor((i - 1) / 2);
            if (heap[p][0] <= heap[i][0]) break;
            [heap[p], heap[i]] = [heap[i], heap[p]];
            i = p;
        }
    };

    const pop = () => {
        const min = heap[0];
        const last = heap.pop();
        if (heap.length > 0) {
            heap[0] = last!;
            let i = 0;
            while (true) {
                const l = 2 * i + 1;
                const r = 2 * i + 2;
                let smallest = i;
                if (l < heap.length && heap[l][0] < heap[smallest][0]) smallest = l;
                if (r < heap.length && heap[r][0] < heap[smallest][0]) smallest = r;
                if (smallest === i) break;
                [heap[smallest], heap[i]] = [heap[i], heap[smallest]];
                i = smallest;
            }
        }
        return min;
    };

    let day = 0;

    // Days 0 to n-1
    for (day = 0; day < n; day++) {
        if (apples[day] > 0) {
            push(day + days[day], apples[day]);
        }

        while (heap.length > 0 && heap[0][0] <= day) {
            pop();
        }

        if (heap.length > 0) {
            const [exp, cnt] = pop();
            eaten++;
            if (cnt > 1) push(exp, cnt - 1);
        }
    }

    // Days n onwards
    while (heap.length > 0) {
        while (heap.length > 0 && heap[0][0] <= day) {
            pop();
        }

        if (heap.length === 0) break;

        const [exp, cnt] = pop();
        eaten++;
        if (cnt > 1) push(exp, cnt - 1);
        day++;
    }

    return eaten;
};

```

# Example walkthrough
## Example 1:
**Input:** `apples = [1,2,3,5,2], days = [3,2,1,4,2]`

```
Day 0: Add (3, 1), eat 1 apple → eaten = 1
Day 1: Add (3, 2), eat from (3, 2) → eaten = 2
Day 2: Add (3, 3), eat from (3, 3) → eaten = 3
Day 3: Add (7, 5), eat from (3, 2) → eaten = 4
Day 4: Add (6, 2), eat from (6, 2) → eaten = 5
Day 5: eat from (6, 1) → eaten = 6
Day 6: eat from (7, 4) → eaten = 7
```

**Output:** `7`

## Example 2:
**Input:** `apples = [3,0,0,0,0,2], days = [3,0,0,0,0,2]`

```
Day 0: Add (3, 3), eat 1 → eaten = 1
Day 1: eat from (3, 2) → eaten = 2
Day 2: eat from (3, 1) → eaten = 3
Day 3: (3, 0) expires, heap empty
Day 4: heap empty
Day 5: Add (7, 2), eat 1 → eaten = 4
Day 6: eat from (7, 1) → eaten = 5
```

**Output:** `5`

> This greedy approach with a min-heap ensures we always eat the apple closest to rotting, maximizing the total apples eaten.
