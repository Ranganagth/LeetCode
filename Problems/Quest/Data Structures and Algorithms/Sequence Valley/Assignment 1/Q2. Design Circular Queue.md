# Intuition

A circular queue uses a fixed-size array where the end wraps around to the beginning. We need to track:
- **Front pointer**: Points to the first element.
- **Rear pointer**: Points to the last element.
- **Size/Count**: Number of elements currently in the queue.

Using a size counter is simpler than trying to distinguish between empty and full states with just pointers.

**Key operations:**
- **enQueue**: Add at rear, move rear pointer forward (with wrapping).
- **deQueue**: Remove from front, move front pointer forward (with wrapping).
- **Front/Rear**: Return values at front/rear pointers.
- **isEmpty**: Check if size == 0.
- **isFull**: Check if size == capacity.

---

# Approach

1. Create a fixed-size array of length `k`.
2. Maintain `front`, `rear`, and `size` pointers.
3. Use modulo arithmetic to wrap around: `(index + 1) % k`.
4. For enQueue: Add at rear, increment rear and size.
5. For deQueue: Remove from front, increment front and decrement size.
6. For isEmpty/isFull: Check size against 0 and capacity.

---

# Complexity

- **Time complexity:** 
  $$O(1)$$ for all operations.

- **Space complexity:** 
  $$O(k)$$ for the array of size k.

# Implementation

```typescript
class MyCircularQueue {
    private queue: number[];
    private front: number;
    private rear: number;
    private size: number;
    private capacity: number;

    constructor(k: number) {
        this.queue = new Array(k);
        this.front = 0;
        this.rear = -1;
        this.size = 0;
        this.capacity = k;
    }

    enQueue(value: number): boolean {
        // Cannot enqueue if queue is full
        if (this.isFull()) {
            return false;
        }

        // Move rear pointer to next position (with wrapping)
        this.rear = (this.rear + 1) % this.capacity;
        this.queue[this.rear] = value;
        this.size++;
        return true;
    }

    deQueue(): boolean {
        // Cannot dequeue if queue is empty
        if (this.isEmpty()) {
            return false;
        }

        // Move front pointer to next position (with wrapping)
        this.front = (this.front + 1) % this.capacity;
        this.size--;
        return true;
    }

    Front(): number {
        // Return -1 if queue is empty
        if (this.isEmpty()) {
            return -1;
        }
        return this.queue[this.front];
    }

    Rear(): number {
        // Return -1 if queue is empty
        if (this.isEmpty()) {
            return -1;
        }
        return this.queue[this.rear];
    }

    isEmpty(): boolean {
        return this.size === 0;
    }

    isFull(): boolean {
        return this.size === this.capacity;
    }
};

```

# Example walkthrough

**Input:**
```
["MyCircularQueue", "enQueue", "enQueue", "enQueue", "enQueue", "Rear", "isFull", "deQueue", "enQueue", "Rear"]
[[3], [1], [2], [3], [4], [], [], [], [4], []]
```

**Step-by-step execution:**

```
MyCircularQueue(3):
  queue = [_, _, _], front = 0, rear = -1, size = 0

enQueue(1):
  rear = (−1 + 1) % 3 = 0
  queue[0] = 1
  size = 1
  return true

enQueue(2):
  rear = (0 + 1) % 3 = 1
  queue[1] = 2
  size = 2
  return true

enQueue(3):
  rear = (1 + 1) % 3 = 2
  queue[2] = 3
  size = 3
  return true

enQueue(4):
  size = 3 = capacity → isFull() = true
  return false

Rear():
  queue[rear] = queue[2] = 3
  return 3

isFull():
  size = 3 = capacity
  return true

deQueue():
  front = (0 + 1) % 3 = 1
  size = 2
  return true
  queue = [1, 2, 3], front = 1, rear = 2, size = 2

enQueue(4):
  rear = (2 + 1) % 3 = 0
  queue[0] = 4
  size = 3
  return true
  queue = [4, 2, 3], front = 1, rear = 0, size = 3

Rear():
  queue[rear] = queue[0] = 4
  return 4
```

**Output:** `[null, true, true, true, false, 3, true, true, true, 4]`
