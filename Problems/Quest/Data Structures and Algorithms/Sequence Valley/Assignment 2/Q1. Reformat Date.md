# Intuition

We need to parse a date string in the format "Day Month Year" and convert it to "YYYY-MM-DD".

**Steps:**
1. Split the date string into parts: day, month, year.
2. Extract the numeric day (remove the suffix like "st", "nd", "rd", "th").
3. Convert the month name to a numeric month (01-12).
4. Format the result as "YYYY-MM-DD" with zero-padding.

---

# Approach

1. Split the date string by spaces to get `[day, month, year]`.
2. Extract the numeric day by taking the first part of the day string (remove suffix).
3. Create a mapping of month names to month numbers.
4. Look up the month number from the mapping.
5. Format the result with zero-padding using `padStart()`.

---

# Complexity

- **Time complexity:** 
  $$O(1)$$ - constant time since the input size is fixed.

- **Space complexity:** 
  $$O(1)$$ - constant space for the month mapping.

---

# Implementation

```typescript
function reformatDate(date: string): string {
    // Split the date string
    const parts = date.split(' ');
    const dayStr = parts[0];
    const monthStr = parts[1];
    const yearStr = parts[2];

    // Extract numeric day (remove suffix like "st", "nd", "rd", "th")
    const day = dayStr.slice(0, -2);

    // Month mapping
    const monthMap: { [key: string]: string } = {
        'Jan': '01',
        'Feb': '02',
        'Mar': '03',
        'Apr': '04',
        'May': '05',
        'Jun': '06',
        'Jul': '07',
        'Aug': '08',
        'Sep': '09',
        'Oct': '10',
        'Nov': '11',
        'Dec': '12'
    };

    const month = monthMap[monthStr];

    // Format as YYYY-MM-DD with zero-padding
    return `${yearStr}-${month}-${day.padStart(2, '0')}`;
};

```

# Alternative Implementation (More Concise)

```typescript
function reformatDate(date: string): string {
    const [dayStr, monthStr, yearStr] = date.split(' ');
    
    const months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 
                    'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'];
    
    const day = dayStr.slice(0, -2).padStart(2, '0');
    const month = (months.indexOf(monthStr) + 1).toString().padStart(2, '0');
    
    return `${yearStr}-${month}-${day}`;
};

```

# Example walkthrough
## Example 1:
**Input:** `date = "20th Oct 2052"`

- Split: `["20th", "Oct", "2052"]`
- Day: `"20th".slice(0, -2)` = `"20"` → padStart = `"20"`
- Month: `monthMap["Oct"]` = `"10"`
- Year: `"2052"`
- Result: `"2052-10-20"`

**Output:** `"2052-10-20"`

# Key Points
- **slice(0, -2)**: Removes the last 2 characters (the suffix).
- **padStart(2, '0')**: Pads the string with leading zeros to ensure 2 digits.
- **Month mapping**: Maps month names to their numeric values (01-12).

This solution is simple, efficient, and handles all the formatting requirements correctly.
