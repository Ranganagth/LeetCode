# Intuition

We need to find the maximum number of times `word` can be concatenated such that the result is still a substring of `sequence`.

**Approach:**
1. Start with `k = 1` and keep increasing it.
2. For each `k`, check if `word` repeated `k` times is a substring of `sequence`.
3. Stop when the repeated word is no longer a substring.
4. Return the last valid `k`.

**Optimization:**
- The maximum possible `k` is bounded by `len(sequence) / len(word)`, so we don't need to check beyond that.

---

# Approach

1. Calculate the maximum possible `k` as `floor(len(sequence) / len(word))`.
2. Iterate from `k = 1` to `max_k`.
3. For each `k`, check if `word * k` is a substring of `sequence`.
4. Return the largest `k` for which this is true.
5. If no valid `k` is found, return 0.

---

# Complexity

- **Time complexity:** 
  $$O(n \times m)$$, where `n = len(sequence)` and `m = len(word)`. In the worst case, we check up to `n/m` repetitions, and each substring check takes O(n).

- **Space complexity:** 
  $$O(n)$$ for storing the repeated word string.

---

# Implementation

```typescript
function maxRepeating(sequence: string, word: string): number {
    const maxK = Math.floor(sequence.length / word.length);

    for (let k = maxK; k >= 1; k--) {
        const repeated = word.repeat(k);
        if (sequence.includes(repeated)) {
            return k;
        }
    }

    return 0;
};

```

---

# Alternative Implementation (Iterative from 1)

```typescript
function maxRepeating(sequence: string, word: string): number {
    let k = 0;
    let repeated = '';

    while (true) {
        repeated += word;
        if (sequence.includes(repeated)) {
            k++;
        } else {
            break;
        }
    }

    return k;
};

```

# Example walkthrough
## Example 1:
**Input:** `sequence = "ababc", word = "ab"`

- maxK = floor(5 / 2) = 2
- k = 2: repeated = "abab", "ababc".includes("abab") = true → return 2

**Output:** `2`

## Example 2:
**Input:** `sequence = "ababc", word = "ba"`

- maxK = floor(5 / 2) = 2
- k = 2: repeated = "baba", "ababc".includes("baba") = false
- k = 1: repeated = "ba", "ababc".includes("ba") = true → return 1

**Output:** `1`

# Recommendation

Use the **first implementation** (iterating from maxK downwards) as it's more efficient:
- It starts from the maximum possible value and works downward.
- As soon as we find a valid `k`, we return immediately.
- This avoids unnecessary checks for smaller values.

The **alternative implementation** is also good if you prefer building the repeated string incrementally, which can be more intuitive.

> Both solutions are efficient given the small constraints (sequence and word length <= 100).
