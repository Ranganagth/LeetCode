[Q2. Find K Pairs with Smallest Sums](https://leetcode.com/problems/find-k-pairs-with-smallest-sums/)

# Intuition

To find the k pairs with the smallest sums, we can use a **min-heap** (priority queue) to efficiently track and extract pairs in order of their sums. Since both arrays are sorted, we can start with the smallest possible pair `(nums1[0], nums2[0])` and progressively explore pairs with the next smallest sums.

The key insight is that for each pair `(nums1[i], nums2[j])`, the next candidates with potentially small sums are:
- `(nums1[i+1], nums2[j])` - move to the next element in nums1
- `(nums1[i], nums2[j+1])` - move to the next element in nums2

We use a min-heap to always extract the pair with the smallest sum and add new candidate pairs.

---

# Approach

1. Create a min-heap to store pairs as `[sum, i, j]` where `sum = nums1[i] + nums2[j]`.
2. Initialize the heap with pairs `(nums1[i], nums2[0])` for all valid `i` (up to `min(k, nums1.length)`).
3. Extract the smallest pair from the heap k times:
   - Add the pair `[nums1[i], nums2[j]]` to the result.
   - If `j + 1 < nums2.length`, push the next pair `(nums1[i], nums2[j+1])` to the heap.
4. Return the result array.

---

# Complexity

- **Time complexity:** 
  $$O(k \log k)$$, where we perform k extractions from the heap, each taking O(log k) time.

- **Space complexity:** 
  $$O(k)$$ for the heap and result array.

# Code

```typescript
function kSmallestPairs(nums1: number[], nums2: number[], k: number): number[][] {
    const result: number[][] = [];
    if (nums1.length === 0 || nums2.length === 0 || k === 0) return result;

    // Min-heap: [sum, i, j]
    const minHeap: [number, number, number][] = [];

    // Helper function to maintain min-heap property
    const heapPush = (item: [number, number, number]) => {
        minHeap.push(item);
        let idx = minHeap.length - 1;
        while (idx > 0) {
            const parentIdx = Math.floor((idx - 1) / 2);
            if (minHeap[parentIdx][0] <= minHeap[idx][0]) break;
            [minHeap[parentIdx], minHeap[idx]] = [minHeap[idx], minHeap[parentIdx]];
            idx = parentIdx;
        }
    };

    const heapPop = (): [number, number, number] => {
        const min = minHeap[0];
        const last = minHeap.pop()!;
        if (minHeap.length > 0) {
            minHeap[0] = last;
            let idx = 0;
            while (true) {
                const leftIdx = 2 * idx + 1;
                const rightIdx = 2 * idx + 2;
                let smallestIdx = idx;

                if (leftIdx < minHeap.length && minHeap[leftIdx][0] < minHeap[smallestIdx][0]) {
                    smallestIdx = leftIdx;
                }
                if (rightIdx < minHeap.length && minHeap[rightIdx][0] < minHeap[smallestIdx][0]) {
                    smallestIdx = rightIdx;
                }
                if (smallestIdx === idx) break;
                [minHeap[smallestIdx], minHeap[idx]] = [minHeap[idx], minHeap[smallestIdx]];
                idx = smallestIdx;
            }
        }
        return min;
    };

    // Initialize heap with pairs (nums1[i], nums2[0])
    for (let i = 0; i < Math.min(k, nums1.length); i++) {
        heapPush([nums1[i] + nums2[0], i, 0]);
    }

    // Extract k smallest pairs
    while (result.length < k && minHeap.length > 0) {
        const [sum, i, j] = heapPop();
        result.push([nums1[i], nums2[j]]);

        // Add next candidate pair
        if (j + 1 < nums2.length) {
            heapPush([nums1[i] + nums2[j + 1], i, j + 1]);
        }
    }

    return result;
};

```

# Example walkthrough
## Example 1:
**Input:** `nums1 = [1,7,11], nums2 = [2,4,6], k = 3`

**Process/Approach explanation:**
1. Initialize heap with: `[[1+2, 0, 0], [7+2, 1, 0], [11+2, 2, 0]]` → `[[3, 0, 0], [9, 1, 0], [13, 2, 0]]`
2. Extract `[3, 0, 0]` → result: `[[1,2]]`, push `[1+4, 0, 1]` → heap: `[[5, 0, 1], [9, 1, 0], [13, 2, 0]]`
3. Extract `[5, 0, 1]` → result: `[[1,2], [1,4]]`, push `[1+6, 0, 2]` → heap: `[[7, 0, 2], [9, 1, 0], [13, 2, 0]]`
4. Extract `[7, 0, 2]` → result: `[[1,2], [1,4], [1,6]]`

**Output:** `[[1,2],[1,4],[1,6]]`
