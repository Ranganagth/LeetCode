[Q3. Construct Target Array With Multiple Sums](https://leetcode.com/problems/construct-target-array-with-multiple-sums/)

# Intuition

We need to work **backwards** from the target array to the starting array `[1, 1, ..., 1]`. The key insight is:
- At each step, the largest element in the array was created by replacing some previous value with the sum of all elements.
- If the current largest element is `max` and the sum of all elements is `sum`, then the previous value at that position was `max - (sum - max) = 2*max - sum`.

We can use a **max-heap** to always work with the largest element and reverse the operations until we reach the starting array or determine it's impossible.

---

# Approach

1. Calculate the total sum of the target array.
2. Use a max-heap to track the largest element.
3. While the largest element is greater than 1:
   - Extract the largest element `max`.
   - Calculate the sum of the rest: `rest = sum - max`.
   - If `rest == 0` or `rest >= max`, return false (impossible case).
   - Calculate the previous value: `prev = max % rest` (using modulo to handle multiple steps at once).
   - If `prev == 0` and `rest != 1`, return false (we can't reach 1).
   - If `prev == 0`, set `prev = rest` (special case when max is a multiple of rest).
   - Update the sum and insert the previous value back into the heap.
4. Return true if we successfully reduce all elements to 1.

---

# Complexity

- **Time complexity:** 
  $$O(n \log n \log M)$$, where `n` is the array length and `M` is the maximum value in the target array. Each heap operation is O(log n), and we may need O(log M) iterations.

- **Space complexity:** 
  $$O(n)$$ for the heap.

# Code

```typescript
function isPossible(target: number[]): boolean {
    if (target.length === 1) return target[0] === 1;

    // Create max-heap
    const maxHeap: number[] = [...target];
    maxHeap.sort((a, b) => b - a);

    let sum = target.reduce((acc, val) => acc + val, 0);

    const heapPush = (val: number) => {
        maxHeap.push(val);
        let idx = maxHeap.length - 1;
        while (idx > 0) {
            const parentIdx = Math.floor((idx - 1) / 2);
            if (maxHeap[parentIdx] >= maxHeap[idx]) break;
            [maxHeap[parentIdx], maxHeap[idx]] = [maxHeap[idx], maxHeap[parentIdx]];
            idx = parentIdx;
        }
    };

    const heapPop = (): number => {
        const max = maxHeap[0];
        const last = maxHeap.pop()!;
        if (maxHeap.length > 0) {
            maxHeap[0] = last;
            let idx = 0;
            while (true) {
                const leftIdx = 2 * idx + 1;
                const rightIdx = 2 * idx + 2;
                let largestIdx = idx;

                if (leftIdx < maxHeap.length && maxHeap[leftIdx] > maxHeap[largestIdx]) {
                    largestIdx = leftIdx;
                }
                if (rightIdx < maxHeap.length && maxHeap[rightIdx] > maxHeap[largestIdx]) {
                    largestIdx = rightIdx;
                }
                if (largestIdx === idx) break;
                [maxHeap[largestIdx], maxHeap[idx]] = [maxHeap[idx], maxHeap[largestIdx]];
                idx = largestIdx;
            }
        }
        return max;
    };

    while (maxHeap[0] > 1) {
        const max = heapPop();
        const rest = sum - max;

        // Edge cases
        if (rest === 0 || rest >= max) return false;

        // Calculate previous value using modulo
        let prev = max % rest;
        
        // If prev is 0, it means max is a multiple of rest
        if (prev === 0) {
            if (rest !== 1) return false;
            prev = rest;
        }

        sum = sum - max + prev;
        heapPush(prev);
    }

    return true;
};

```

2. prev = 9 % 8 = 1, sum = 9, heap = [5, 3, 1]
3. max = 5, rest = 4, prev = 5 % 4 = 1, sum = 5, heap = [3, 1, 1]
