[Q1. Last Stone Weight](https://leetcode.com/problems/last-stone-weight/)

# Intuition

The problem requires repeatedly finding and smashing the two heaviest stones until at most one stone remains. A **max-heap** (priority queue) is the ideal data structure for this task, as it allows us to efficiently extract the maximum element in O(log n) time. We need to implement a custom max-heap since JavaScript/TypeScript doesn't have a built-in priority queue.

---

# Approach

1. Implement a custom max-heap class with `insert`, `extractMax`, and helper methods (`bubbleUp` and `bubbleDown`).
2. Insert all stones into the max-heap.
3. While there are more than one stone in the heap:
   - Extract the two heaviest stones (`y` and `x` where `y >= x`).
   - If they are not equal, insert the difference (`y - x`) back into the heap.
4. Return the last remaining stone if one exists, otherwise return 0.

---

# Complexity

- **Time complexity:**  $$O(n \log n)$$, where `n` is the number of stones. Each insertion and extraction takes O(log n) time, and we perform up to `n` operations.

- **Space complexity:**  $$O(n)$$ for storing the heap.

---

# Code

```typescript
class StoneHeap {
    private heap: number[];

    constructor() {
        this.heap = [];
    }

    insert(val: number): void {
        this.heap.push(val);
        this.bubbleUp(this.heap.length - 1);
    }

    extractMax(): number | null {
        if (this.heap.length === 0) return null;
        const max = this.heap[0];
        const end = this.heap.pop()!;
        if (this.heap.length > 0) {
            this.heap[0] = end;
            this.bubbleDown(0);
        }
        return max;
    }

    size(): number {
        return this.heap.length;
    }

    private bubbleUp(index: number): void {
        while (index > 0) {
            const parentIdx = Math.floor((index - 1) / 2);
            if (this.heap[parentIdx] >= this.heap[index]) break;
            [this.heap[parentIdx], this.heap[index]] = [this.heap[index], this.heap[parentIdx]];
            index = parentIdx;
        }
    }

    private bubbleDown(index: number): void {
        const length = this.heap.length;
        while (true) {
            let leftIdx = 2 * index + 1;
            let rightIdx = 2 * index + 2;
            let largestIdx = index;

            if (leftIdx < length && this.heap[leftIdx] > this.heap[largestIdx]) {
                largestIdx = leftIdx;
            }
            if (rightIdx < length && this.heap[rightIdx] > this.heap[largestIdx]) {
                largestIdx = rightIdx;
            }
            if (largestIdx === index) break;
            [this.heap[largestIdx], this.heap[index]] = [this.heap[index], this.heap[largestIdx]];
            index = largestIdx;
        }
    }
}

function lastStoneWeight(stones: number[]): number {
    const heap = new StoneHeap();
    for (const stone of stones) {
        heap.insert(stone);
    }

    while (heap.size() > 1) {
        const y = heap.extractMax()!;
        const x = heap.extractMax()!;
        if (x !== y) {
            heap.insert(y - x);
        }
    }

    return heap.size() === 0 ? 0 : heap.extractMax()!;
};

```

# Example walkthrough
## Example 1:
**Input:** `stones = [2,7,4,1,8,1]`

**Process/Approach explanation:**
1. Build max-heap: Insert all stones → heap structure maintains largest at top
2. Extract 8 and 7 → difference = 1 → insert 1 back
3. Extract 4 and 2 → difference = 2 → insert 2 back
4. Extract 2 and 1 → difference = 1 → insert 1 back
5. Extract 1 and 1 → equal, both destroyed
6. One stone remains with weight 1

**Output:** `1`
