# Intuition

A string `s` can be rotated to become `goal` if and only if:
1. Both strings have the **same length**.
2. `goal` is a **substring of `s + s`**.

**Why this works:**
- If we concatenate `s` with itself, all possible rotations of `s` appear as substrings.
- For example, if `s = "abcde"`, then `s + s = "abcdeabcde"` contains all rotations: "abcde", "bcdea", "cdeab", "deabc", "eabcd".

---

# Approach

1. Check if `s` and `goal` have the **same length**. If not, return false.
2. Check if `goal` is a **substring of `s + s`**. If yes, return true; otherwise, return false.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the string. The `indexOf` method uses optimized string search algorithms.

- **Space complexity:** 
  $$O(n)$$ for the concatenated string `s + s`.

---

# Implementation

```typescript
function rotateString(s: string, goal: string): boolean {
    // Check if lengths are equal
    if (s.length !== goal.length) {
        return false;
    }

    // Check if goal is a substring of s + s
    return (s + s).includes(goal);
};

```

# Example walkthrough
## Example 1:
**Input:** `s = "abcde", goal = "cdeab"`

- Length check: 5 === 5
- `s + s = "abcdeabcde"`
- `"abcdeabcde".includes("cdeab")` → true (found at index 2)

**Output:** `true`

## Example 2:
**Input:** `s = "abcde", goal = "abced"`

- Length check: 5 === 5
- `s + s = "abcdeabcde"`
- `"abcdeabcde".includes("abced")` → false (not found)

**Output:** `false`

## Example 3 (Edge case):
**Input:** `s = "a", goal = "a"`

- Length check: 1 === 1
- `s + s = "aa"`
- `"aa".includes("a")` → true

**Output:** `true`

## Example 4 (Edge case):
**Input:** `s = "ab", goal = "ba"`

- Length check: 2 === 2
- `s + s = "abab"`
- `"abab".includes("ba")` → true (found at index 1)

**Output:** `true`

# Alternative Implementation (Manual Rotation)

If you want to explicitly check all rotations:

```typescript
function rotateString(s: string, goal: string): boolean {
    if (s.length !== goal.length) {
        return false;
    }

    let current = s;
    for (let i = 0; i < s.length; i++) {
        if (current === goal) {
            return true;
        }
        // Rotate: move first character to the end
        current = current.substring(1) + current[0];
    }

    return false;
};

```

## Complexity of alternative:

- **Time complexity:** $$O(n^2)$$ - n rotations, each taking O(n) to create and compare.
- **Space complexity:** $$O(n)$$ for the rotated string.

---

# Recommendation

Use the **substring approach** (first implementation) as it's more elegant, concise, and efficient. It's a classic trick for rotation problems.
