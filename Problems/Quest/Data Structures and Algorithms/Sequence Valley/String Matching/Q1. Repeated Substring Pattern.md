# Intuition

A string can be constructed by repeating a substring if and only if the string length is divisible by the substring length, and the substring repeated forms the original string.

There are several approaches:

1. **Brute Force**: Try all possible substring lengths that divide the string length.
2. **String Concatenation Trick**: If `s` is made of repeating substring, then `s` will appear in `(s + s).substring(1, 2*n-1)`.
3. **KMP Algorithm**: Use the KMP failure function to detect the pattern.

The **string concatenation trick** is elegant and efficient.

---

# Approach 1: String Concatenation Trick (Most Elegant)

If string `s` is formed by repeating a substring, then:
- `(s + s)` contains `s` as a substring starting at index 1 (or any index from 1 to n-1).
- By removing the first and last character from `(s + s)`, we still find `s` if it's a repeated pattern.

**Why it works:**
- If `s = "abc" + "abc"`, then `(s + s) = "abcabcabcabc"`.
- Removing first and last char: `"bcabcabcab"` still contains `"abcabc"`.

---

# Approach 2: Brute Force with Divisors

1. Try all divisors of the string length.
2. For each divisor `d`, check if repeating `s[0:d]` gives the original string.

---

# Approach 3: KMP Failure Function

Use the KMP algorithm's failure function to detect repeating patterns.

---

# Implementation (Approach 1 - Most Elegant)

```typescript
function repeatedSubstringPattern(s: string): boolean {
    return (s + s).indexOf(s, 1) !== s.length;
};

```

**Explanation:**
- `(s + s)` concatenates the string with itself.
- `indexOf(s, 1)` searches for `s` starting from index 1.
- If `s` is a repeated pattern, it will be found before index `s.length`.
- If the index equals `s.length`, it means `s` is not a repeated pattern.

---

# Implementation (Approach 2 - Brute Force)

```typescript
function repeatedSubstringPattern(s: string): boolean {
    const n = s.length;

    // Try all divisors of n
    for (let i = 1; i <= n / 2; i++) {
        if (n % i === 0) {
            const pattern = s.substring(0, i);
            let isValid = true;

            // Check if repeating pattern forms the original string
            for (let j = i; j < n; j++) {
                if (s[j] !== pattern[j % i]) {
                    isValid = false;
                    break;
                }
            }

            if (isValid) return true;
        }
    }

    return false;
};

```

# Implementation (Approach 3 - KMP Failure Function)

```typescript
function repeatedSubstringPattern(s: string): boolean {
    const n = s.length;

    // Build KMP failure function
    const lps = new Array(n).fill(0);
    for (let i = 1; i < n; i++) {
        let j = lps[i - 1];
        while (j > 0 && s[i] !== s[j]) {
            j = lps[j - 1];
        }
        if (s[i] === s[j]) {
            j++;
        }
        lps[i] = j;
    }

    // If lps[n-1] > 0 and n is divisible by (n - lps[n-1]),
    // then the string is a repeated pattern
    const patternLength = n - lps[n - 1];
    return lps[n - 1] > 0 && n % patternLength === 0;
};

```

# Complexity Analysis

## Approach 1 (String Concatenation):
- **Time complexity:** $$O(n)$$ - `indexOf` uses optimized string search.
- **Space complexity:** $$O(n)$$ - for the concatenated string.

## Approach 2 (Brute Force):
- **Time complexity:** $$O(n \sqrt{n})$$ - trying all divisors and checking each.
- **Space complexity:** $$O(n)$$ - for the substring.

## Approach 3 (KMP):
- **Time complexity:** $$O(n)$$ - KMP is linear.
- **Space complexity:** $$O(n)$$ - for the LPS array.

# Example walkthrough
## Example 1:
**Input:** `s = "abab"`

**Approach 1:**
- `(s + s) = "abababab"`
- `indexOf("abab", 1)` finds "abab" at index 2
- 2 !== 4 → return true

**Output:** `true`

## Example 2:
**Input:** `s = "aba"`

**Approach 1:**
- `(s + s) = "abaaba"`
- `indexOf("aba", 1)` finds "aba" at index 3
- 3 === 3 → return false

**Output:** `false`

## Example 3:
**Input:** `s = "abcabcabcabc"`

**Approach 1:**
- `(s + s) = "abcabcabcabcabcabcabcabc"`
- `indexOf("abcabcabcabc", 1)` finds it at index 3
- 3 !== 12 → return true

**Output:** `true`

# Recommendation

Use **Approach 1** (string concatenation trick) for its elegance and simplicity. It's a one-liner and very efficient in practice.
