# Intuition

We need to find the minimum number of times to repeat string `a` so that `b` becomes a substring of the repeated string.

Key observations:
1. If `b` contains characters not in `a`, it's impossible → return -1.
2. The maximum repetitions needed is bounded. If we repeat `a` enough times to have length >= `len(a) + len(b)`, we're guaranteed to find `b` if it exists.
3. We can start with `ceil(len(b) / len(a))` repetitions and check if `b` is a substring.
4. If not found, try one more repetition (to handle edge cases where `b` spans across boundaries).

**Why the bound works:**
- If `b` is a substring of repeated `a`, then `b` must fit within a window of size `len(a) + len(b) - 1`.
- So we need at most `ceil(len(b) / len(a)) + 1` repetitions.

---

# Approach

1. **Check if it's possible**: Verify that all characters in `b` exist in `a`. If not, return -1.
2. **Calculate minimum repetitions**: Start with `k = ceil(len(b) / len(a))`.
3. **Check if `b` is a substring**: 
   - Repeat `a` `k` times and check if `b` is in it.
   - If yes, return `k`.
   - If no, try `k + 1` repetitions.
   - If still not found, return -1.

---

# Complexity

- **Time complexity:** 
  $$O(n + m)$$, where `n = len(a)` and `m = len(b)`. We create a string of length at most `n + m` and search for `b` in it.

- **Space complexity:** 
  $$O(n + m)$$ for the repeated string.

---

# Implementation

```typescript
function repeatedStringMatch(a: string, b: string): number {
    // Check if all characters in b exist in a
    const aChars = new Set(a);
    for (const char of b) {
        if (!aChars.has(char)) {
            return -1;
        }
    }

    // Calculate minimum repetitions needed
    let k = Math.ceil(b.length / a.length);

    // Try k and k+1 repetitions
    for (let i = 0; i < 2; i++) {
        const repeated = a.repeat(k + i);
        if (repeated.includes(b)) {
            return k + i;
        }
    }

    return -1;
};

```

# Example walkthrough
## Example 1:
**Input:** `a = "abcd", b = "cdabcdab"`

- Check characters: All chars in "cdabcdab" exist in "abcd" ✓
- `k = ceil(8 / 4) = 2`
- Try k = 2: `"abcd".repeat(2) = "abcdabcd"` → "cdabcdab" NOT in "abcdabcd" ✗
- Try k = 3: `"abcd".repeat(3) = "abcdabcdabcd"` → "cdabcdab" IS in "abcdabcdabcd" ✓
- Return 3

**Output:** `3` ✓

## Example 2:
**Input:** `a = "a", b = "aa"`

- Check characters: All chars in "aa" exist in "a" ✓
- `k = ceil(2 / 1) = 2`
- Try k = 2: `"a".repeat(2) = "aa"` → "aa" IS in "aa" ✓
- Return 2

**Output:** `2` ✓

## Example 3 (Impossible case):
**Input:** `a = "ab", b = "ba"`

- Check characters: All chars in "ba" exist in "ab" ✓
- `k = ceil(2 / 2) = 1`
- Try k = 1: `"ab".repeat(1) = "ab"` → "ba" NOT in "ab" ✗
- Try k = 2: `"ab".repeat(2) = "abab"` → "ba" IS in "abab" ✓
- Return 2

**Output:** `2` ✓

## Example 4 (Impossible case):
**Input:** `a = "abc", b = "wxyz"`

- Check characters: 'w' NOT in "abc" ✗
- Return -1

**Output:** `-1` ✓

# Alternative Implementation (More Explicit)

```typescript
function repeatedStringMatch(a: string, b: string): number {
    // Check if it's possible
    const aSet = new Set(a);
    for (const char of b) {
        if (!aSet.has(char)) {
            return -1;
        }
    }

    // Start with minimum repetitions
    let repeated = a;
    let count = 1;

    // Keep repeating until b is found or we exceed reasonable limit
    while (repeated.length < b.length + a.length) {
        if (repeated.includes(b)) {
            return count;
        }
        repeated += a;
        count++;
    }

    // Check one more time
    if (repeated.includes(b)) {
        return count;
    }

    return -1;
};

```

# Recommendation

Use the **first implementation** as it's more concise and efficient. It directly calculates the minimum repetitions needed without unnecessary iterations.
