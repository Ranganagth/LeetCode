[Q1. Detect Capital](https://leetcode.com/problems/detect-capital/)

# Intuition

The problem asks us to verify if the usage of capitals in a given word follows one of the three valid patterns:
- All letters are uppercase.
- All letters are lowercase.
- Only the first letter is uppercase.

We can check these conditions directly:
- Check if all characters are uppercase.
- Check if all characters are lowercase.
- Check if only the first character is uppercase and the rest are lowercase.

If any of these conditions hold, return `true`; otherwise, `false`.

---

# Approach

1. Check if the entire word is uppercase.
2. Check if the entire word is lowercase.
3. Check if the first character is uppercase and the rest are lowercase.
4. Return `true` if any condition is satisfied, else `false`.

---

# Code

```typescript
function detectCapitalUse(word: string): boolean {
    const isAllUpper = word === word.toUpperCase();
    const isAllLower = word === word.toLowerCase();
    const isFirstUpperRestLower = 
        word[0] === word[0].toUpperCase() && 
        word.slice(1) === word.slice(1).toLowerCase();

    return isAllUpper || isAllLower || isFirstUpperRestLower;
};

```

# Example walkthrough
## Example 1:
Input: `"USA"`

- `isAllUpper` = true
- `isAllLower` = false
- `isFirstUpperRestLower` = false
- Result: `true`

## Example 2:
Input: `"FlaG"`

- `isAllUpper` = false
- `isAllLower` = false
- `isFirstUpperRestLower` = false (since 'aG' is not all lowercase)
- Result: `false`

This straightforward approach efficiently checks all conditions with minimal code.
