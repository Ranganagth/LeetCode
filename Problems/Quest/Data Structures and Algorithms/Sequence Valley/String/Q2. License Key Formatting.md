[Q2. License Key Formatting](https://leetcode.com/problems/license-key-formatting/)

# Intuition

The goal is to reformat the license key string so that:
- All characters are uppercase.
- Dashes are inserted to separate groups of exactly `k` characters, except possibly the first group which can be shorter.
- Remove all existing dashes and then re-divide the string from the end, ensuring the last groups are of size `k`.

This approach simplifies the problem by:
1. Removing all existing dashes.
2. Converting all characters to uppercase.
3. Building the new string from the end, inserting dashes after every `k` characters.

---

# Approach

1. Remove all dashes from the string.
2. Convert the string to uppercase.
3. Iterate from the end of the cleaned string:
   - Append characters to a result array.
   - After every `k` characters, insert a dash.
4. Reverse the result array and join to form the final string.

---

# Code

```typescript
function licenseKeyFormatting(s: string, k: number): string {
    // Remove all dashes and convert to uppercase
    const cleaned = s.replace(/-/g, '').toUpperCase();
    const result: string[] = [];

    // Build from the end
    for (let i = cleaned.length - 1; i >= 0; i--) {
        result.push(cleaned[i]);
        // Insert dash after every k characters, but not at the start
        if ((cleaned.length - i) % k === 0 && i !== 0) {
            result.push('-');
        }
    }

    // Reverse to get the correct order
    return result.reverse().join('');
};

```

# Example walkthrough

## Example 1:
Input: `s = "5F3Z-2e-9-w"`, `k = 4`

- Remove dashes: `"5F3Z2E9W"`
- Convert to uppercase: `"5F3Z2E9W"` (already uppercase)
- Build from end:
  - Take `"W"`, `"9"`, `"E"`, `"2"` → first group: `"W9E2"`
  - Next characters: `"Z"`, `"3"`, `"F"` → second group: `"Z3F"`
- Insert dash between groups: `"5F3Z-2E9W"`

## Example 2:
Input: `s = "2-5g-3-J"`, `k = 2`

- Remove dashes: `"25g3J"`
- Uppercase: `"25G3J"`
- Build from end:
  - `"J"`, `"3"` → `"J3"`
  - `"G"`, `"5"` → `"G5"`
- Final: `"25G-3J"`

This method ensures the string is correctly formatted with minimal complexity.
