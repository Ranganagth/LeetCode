[Q3. Masking Personal Information](https://leetcode.com/problems/masking-personal-information/)

# Intuition
The problem involves two types of personal info: email and phone number. We need to identify which type the input `s` is, then mask it accordingly.

- For **emails**:
  - Convert all characters to lowercase.
  - Mask the middle part of the name with 5 asterisks, keeping only the first and last character.
  - Keep the domain intact, also in lowercase.

- For **phone numbers**:
  - Remove all non-digit characters.
  - Extract the last 10 digits as the local number.
  - The remaining digits (0-3) form the country code.
  - Mask the number with the specified format, adding country code prefix if present.

---

# Approach
1. **Identify the type**:
   - If `s` contains `'@'`, it's an email.
   - Else, it's a phone number.

2. **Email masking**:
   - Convert to lowercase.
   - Split into name and domain parts.
   - Mask the name: first letter + 5 asterisks + last letter.
   - Reconstruct as `masked_name@domain`.

3. **Phone number masking**:
   - Remove all non-digit characters.
   - Extract last 10 digits as local number.
   - Count remaining digits as country code.
   - Format according to rules.

---

# Implementation
```typescript
function maskPII(s: string): string {
    if (s.includes('@')) {
        // Email case
        s = s.toLowerCase();
        const [name, domain] = s.split('@');
        const maskedName = name[0] + '*****' + name[name.length - 1];
        return `${maskedName}@${domain}`;
    } else {
        // Phone number case
        const digits = s.replace(/\D/g, '');
        const localNumber = digits.slice(-10);
        const countryCodeLength = digits.length - 10;
        const countryCode = countryCodeLength > 0 ? '+' + '*'.repeat(countryCodeLength) + '-' : '';

        return `${countryCode}***-***-${localNumber.slice(-4)}`;
    }
}
```

---

# Explanation
- For emails, the code converts the entire string to lowercase, masks the middle of the name, and reconstructs.
- For phone numbers, it strips all non-digit characters, then slices the last 10 digits for the local number, and counts remaining digits for the country code, formatting accordingly.

---

# Examples
```typescript
console.log(maskPII("LeetCode@LeetCode.com")); // "l*****e@leetcode.com"
console.log(maskPII("AB@qq.com")); // "a*****b@qq.com"
console.log(maskPII("1(234)567-890")); // "***-***-7890"
```
