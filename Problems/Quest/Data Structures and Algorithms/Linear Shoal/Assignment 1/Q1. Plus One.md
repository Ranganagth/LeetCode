# Intuition

To increment a large integer represented as an array by one, we need to:

1. Start from the **rightmost digit** (least significant).
2. Add 1 to it.
3. Handle the **carry** if the digit becomes 10.
4. Propagate the carry to the left until there's no carry or we've processed all digits.
5. If there's still a carry after processing all digits, prepend a 1 to the array.

This is similar to manual addition we do on paper.

---

# Approach

1. Start from the last digit (index `length - 1`).
2. Add 1 to the last digit.
3. If the digit is 10, set it to 0 and continue with carry to the next digit.
4. If the digit is less than 10, we're done.
5. If we finish the loop and still have a carry, prepend 1 to the array.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the array. In the worst case, we iterate through all digits (e.g., [9,9,9]).

- **Space complexity:** 
  $$O(1)$$ if we don't count the output array, or $$O(n)$$ if we do (for the case where we need to prepend 1).

---

# Code

```typescript
function plusOne(digits: number[]): number[] {
    // Start from the rightmost digit
    for (let i = digits.length - 1; i >= 0; i--) {
        // Increment the current digit
        digits[i]++;

        // If no carry, we're done
        if (digits[i] < 10) {
            return digits;
        }

        // Set to 0 and continue with carry
        digits[i] = 0;
    }

    // If we reach here, all digits were 9, so we need to prepend 1
    return [1, ...digits];
};

```

# Example walkthrough
## Example 1:
**Input:** `digits = [1,2,3]`

- i = 2: digits[2] = 3 + 1 = 4 (< 10) → return [1,2,4]

**Output:** `[1,2,4]`

## Example 2:
**Input:** `digits = [4,3,2,1]`

- i = 3: digits[3] = 1 + 1 = 2 (< 10) → return [4,3,2,2]

**Output:** `[4,3,2,2]`

## Example 3:
**Input:** `digits = [9]`

- i = 0: digits[0] = 9 + 1 = 10 (>= 10) → set to 0, continue
- Loop ends, prepend 1 → return [1,0]

**Output:** `[1,0]`

## Example 4 (Edge case):
**Input:** `digits = [9,9,9]`

- i = 2: digits[2] = 9 + 1 = 10 → set to 0, continue
- i = 1: digits[1] = 9 + 1 = 10 → set to 0, continue
- i = 0: digits[0] = 9 + 1 = 10 → set to 0, continue
- Loop ends, prepend 1 → return [1,0,0,0]

**Output:** `[1,0,0,0]`

# Alternative Implementation Code(More explicit)

```typescript
function plusOne(digits: number[]): number[] {
    let carry = 1;

    for (let i = digits.length - 1; i >= 0 && carry > 0; i--) {
        const sum = digits[i] + carry;
        digits[i] = sum % 10;
        carry = Math.floor(sum / 10);
    }

    if (carry > 0) {
        return [carry, ...digits];
    }

    return digits;
}
```

> Both implementations are efficient and handle all edge cases correctly. The first approach is slightly more intuitive, while the second is more general (could be adapted for adding any number).
