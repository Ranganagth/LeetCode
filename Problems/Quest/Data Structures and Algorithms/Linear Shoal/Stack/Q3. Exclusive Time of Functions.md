[Q3. Exclusive Time of Functions](https://leetcode.com/problems/exclusive-time-of-functions/)

# Intuition

We need to track the exclusive execution time of each function based on logs that indicate when functions start and end. Using a stack helps manage nested calls, and by updating the times when functions start and end, we can accumulate each function's total exclusive time.

---

# Approach

1. Parse each log into function ID, event type ("start" or "end"), and timestamp.
2. Use a stack to keep track of active functions.
3. Maintain an array `res` of size `n` to store the total exclusive time for each function.
4. Iterate through logs:
   - When a "start" log occurs:
     - If there's a function already running, update its time with the difference between the current timestamp and the previous timestamp.
     - Push the new function ID onto the stack.
     - Update the previous timestamp.
   - When an "end" log occurs:
     - Pop the function ID from the stack.
     - Calculate its total time as the difference between current timestamp and previous timestamp + 1 (since end timestamp is inclusive).
     - Add this time to the function's total in `res`.
     - Update the previous timestamp to current timestamp + 1.
5. Return the `res` array.

---

# Complexity

- **Time complexity:**  $$O(m)$$, where `m` is the number of logs, since each log is processed once.

- **Space complexity:** $$O(n)$$ for the result array and stack.

---

# Code

```typescript
function exclusiveTime(n: number, logs: string[]): number[] {
    const res = new Array(n).fill(0);
    const stack: number[] = [];
    let prevTime = 0;

    for (const log of logs) {
        const [idStr, type, timeStr] = log.split(':');
        const id = parseInt(idStr);
        const timestamp = parseInt(timeStr);

        if (type === 'start') {
            if (stack.length > 0) {
                res[stack[stack.length - 1]] += timestamp - prevTime;
            }
            stack.push(id);
            prevTime = timestamp;
        } else {
            res[stack.pop()!] += timestamp - prevTime + 1;
            prevTime = timestamp + 1;
        }
    }

    return res;
};

```

# Example walkthrough
## Example 1:
**Input:** `["0:start:0","1:start:2","1:end:5","0:end:6"]`

**Process/Approach explanation:**
- Start function 0 at 0: stack=[0], prevTime=0
- Start function 1 at 2:
  - Add (2-0)=2 to function 0's time
  - stack=[0,1], prevTime=2
- End function 1 at 5:
  - Add (5-2+1)=4 to function 1's time
  - stack=[0], prevTime=6
- End function 0 at 6:
  - Add (6-6+1)=1 to function 0's time
  - stack=[], prevTime=7

**Output:** `[3,4]`

This approach correctly computes the exclusive times for nested function calls.
