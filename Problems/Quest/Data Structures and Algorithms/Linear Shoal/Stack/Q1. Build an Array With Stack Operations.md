[Q1. Build an Array With Stack Operations](https://leetcode.com/problems/build-an-array-with-stack-operations/)

# Intuition

Since `target` is strictly increasing and within `[1, n]`, we can simulate the process:
- For each number in `target`, push numbers from 1 up to that number.
- If the current number in the stream is greater than the target number, push it and then pop it immediately to discard it.
- When the current number matches the target, just push it and move to the next target number.
- Continue until all target numbers are processed.

This approach ensures minimal operations and directly constructs the sequence of "Push" and "Pop" commands.

---

# Approach

1. Initialize an empty result array `operations`.
2. Maintain a variable `current` starting from 1, representing the next number to push from the stream.
3. For each number `t` in `target`:
   - While `current < t`, push `current` and then pop it (to discard).
   - When `current == t`, push it.
   - Increment `current` after each push.
4. Return the list of operations.

---

# Complexity

- **Time complexity:** $$O(n)$$, as each number from 1 to `n` is pushed and popped at most once.
  
- **Space complexity:** $$O(n)$$ for the output list.

---

# Code

```typescript
function buildArray(target: number[], n: number): string[] {
    const operations: string[] = [];
    let current = 1;
    for (const t of target) {
        while (current < t) {
            operations.push("Push");
            operations.push("Pop");
            current++;
        }
        operations.push("Push");
        current++;
    }
    return operations;
};

```

# Example walkthrough
## Example 1:

Input: `target = [1,3], n=3`

- current=1:
  - target=1: push 1 → ["Push"], current=2
- target=3:
  - current=2 < 3: push 2, pop 2 → ["Push", "Pop"]
  - current=3:
    - push 3 → ["Push", "Pop", "Push"]
    - current=4 (but loop ends since target is reached)

Output: `["Push", "Push", "Pop", "Push"]`

This matches the expected sequence and efficiently constructs the required operations.
