[Q1. Final Prices With a Special Discount in a Shop](https://leetcode.com/problems/final-prices-with-a-special-discount-in-a-shop/)

# Intuition

For each item, we need to find the next item with a price less than or equal to the current item's price. This can be efficiently achieved using a stack:
- Push indices onto the stack.
- When encountering a new price, pop from the stack while the current price is less than or equal to the price at the top of the stack.
- For each popped index, apply the discount (subtract the current price).

This approach ensures we process each element only once, leading to an efficient solution.

---

# Approach

1. Initialize an answer array `res` as a copy of `prices`.
2. Use a stack to store indices of items.
3. Iterate through each price:
   - While the stack is not empty and the current price is less than or equal to the price at the index on top of the stack:
     - Pop the index from the stack.
     - Update `res` at that index by subtracting the current price.
   - Push the current index onto the stack.
4. Return the `res` array.

---

# Complexity

- **Time complexity:** $$O(n)$$, as each element is pushed and popped at most once.
  
- **Space complexity:**  $$O(n)$$ for the stack and the output array.

---

# Code

```typescript
function finalPrices(prices: number[]): number[] {
    const res = [...prices];
    const stack: number[] = [];

    for (let i = 0; i < prices.length; i++) {
        while (stack.length > 0 && prices[i] <= prices[stack[stack.length - 1]]) {
            const index = stack.pop()!;
            res[index] -= prices[i];
        }
        stack.push(i);
    }

    return res;
};

```

# Example walkthrough
## Example 1:

Input: `[8,4,6,2,3]`

- i=0: stack=[0]
- i=1: prices[1]=4 ≤ prices[0]=8 → pop 0, res[0]=8-4=4, stack=[]
  - push 1 → stack=[1]
- i=2: prices[2]=6 > prices[1]=4 → push 2 → stack=[1,2]
- i=3: prices[3]=2 ≤ prices[2]=6 → pop 2, res[2]=6-2=4
  - prices[3]=2 ≤ prices[1]=4 → pop 1, res[1]=4-2=2
  - push 3 → stack=[3]
- i=4: prices[4]=3 > prices[3]=2 → push 4 → stack=[3,4]

Final res: `[4,2,4,2,3]`

This method efficiently computes the discounted prices for each item.
