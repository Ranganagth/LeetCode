[Q2. Daily Temperatures](https://leetcode.com/problems/daily-temperatures/)

# Intuition

This problem is a classic "Next Greater Element" problem, which can be efficiently solved using a stack:
- We traverse the temperature list.
- For each temperature, we check if it is higher than the temperature at the top of the stack.
- If it is, then for the index at the top of the stack, the current day is the next warmer day.
- We pop from the stack and update the answer for that index.
- We push the current index onto the stack to find its next warmer day later.

This approach ensures each element is pushed and popped at most once, leading to an efficient solution.

---

# Approach

1. Initialize an answer array `res` with zeros.
2. Use a stack to store indices of days.
3. Loop through each temperature:
   - While the stack is not empty and the current temperature is greater than the temperature at the index on top of the stack:
     - Pop the index from the stack.
     - Calculate the difference between current index and popped index, assign it to `res` at the popped index.
   - Push the current index onto the stack.
4. Return the `res` array.

---

# Complexity

- **Time complexity:** $$O(n)$$, since each element is pushed and popped at most once.
  
- **Space complexity:**  $$O(n)$$ for the stack and output array.

---

# Code

```typescript
function dailyTemperatures(temperatures: number[]): number[] {
    const res = new Array(temperatures.length).fill(0);
    const stack: number[] = [];

    for (let i = 0; i < temperatures.length; i++) {
        while (stack.length > 0 && temperatures[i] > temperatures[stack[stack.length - 1]]) {
            const prevIndex = stack.pop()!;
            res[prevIndex] = i - prevIndex;
        }
        stack.push(i);
    }

    return res;
};

```

# Example walkthrough
## Example 1:

Input: `[73,74,75,71,69,72,76,73]`

- i=0: stack=[0]
- i=1: 74 > 73 → pop 0, res[0]=1-0=1; stack=[]
  - push 1 → stack=[1]
- i=2: 75 > 74 → pop 1, res[1]=2-1=1; stack=[]
  - push 2 → stack=[2]
- i=3: 71 ≤ 75 → push 3 → stack=[2,3]
- i=4: 69 ≤ 71 → push 4 → stack=[2,3,4]
- i=5: 72 > 69 → pop 4, res[4]=5-4=1
  - 72 > 71 → pop 3, res[3]=5-3=2
  - 72 ≤ 75 → push 5 → stack=[2,5]
- i=6: 76 > 72 → pop 5, res[5]=6-5=1
  - 76 > 75 → pop 2, res[2]=6-2=4
  - push 6 → stack=[6]
- i=7: 73 ≤ 76 → push 7 → stack=[6,7]

Final `res`: `[1,1,4,2,1,1,0,0]`

This approach efficiently finds the number of days until a warmer temperature for each day.
