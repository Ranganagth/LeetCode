[Q1. Set Mismatch](https://leetcode.com/problems/set-mismatch/)

# Intuition

Since the array `nums` is supposed to contain all numbers from 1 to n exactly once, but due to an error, one number is duplicated and another is missing, we need to identify:
- The number that appears twice.
- The number that is missing.

A straightforward approach is to use the properties of the array:
- The sum of numbers from 1 to n.
- The sum of squares of numbers from 1 to n.

By comparing these with the actual sums of the array, we can derive the missing and duplicated numbers.

---

# Approach

1. Calculate:
   - `sum_nums`: sum of all elements in `nums`.
   - `sum_squares`: sum of squares of all elements in `nums`.
2. Calculate:
   - `expected_sum`: sum of numbers from 1 to n.
   - `expected_sum_squares`: sum of squares of numbers from 1 to n.
3. Set up equations:
   - `duplicate - missing = sum_nums - expected_sum` (Equation 1)
   - `duplicate^2 - missing^2 = sum_squares - expected_sum_squares` (Equation 2)
4. Simplify Equation 2:
   - `(duplicate - missing) * (duplicate + missing) = sum_squares - expected_sum_squares`
   - Use Equation 1 to substitute `duplicate - missing`.
5. Solve for `duplicate` and `missing`.

---

# Code

```typescript
function findErrorNums(nums: number[]): number[] {
    const n = nums.length;
    let sum_nums = 0;
    let sum_squares = 0;
    for (const num of nums) {
        sum_nums += num;
        sum_squares += num * num;
    }

    const expected_sum = (n * (n + 1)) / 2;
    const expected_sum_squares = (n * (n + 1) * (2 * n + 1)) / 6;

    const diff = sum_nums - expected_sum; // duplicate - missing
    const diffSquares = sum_squares - expected_sum_squares; // duplicate^2 - missing^2

    // (duplicate + missing) = diffSquares / diff
    const sum_dup_missing = diffSquares / diff;

    const duplicate = (diff + sum_dup_missing) / 2;
    const missing = sum_dup_missing - duplicate;

    return [duplicate, missing];
};

```

# Example walkthrough
## Example 1:

Input: `[1,2,2,4]`

- sum_nums = 1+2+2+4=9
- sum_squares=1+4+4+16=25
- expected_sum= (4*5)/2=10
- expected_sum_squares= (4*5*9)/6=30

- diff=9-10=-1 (duplicate - missing)
- diffSquares=25-30=-5

- duplicate + missing = -5 / -1=5
- duplicate = (diff + sum_dup_missing)/2 = (-1 + 5)/2=2
- missing=5 - 2=3

Output: `[2,3]`

# Summary

This approach efficiently finds the duplicated and missing numbers using sum and sum of squares, with $O(n)$ time complexity and $O(1)$ extra space.
