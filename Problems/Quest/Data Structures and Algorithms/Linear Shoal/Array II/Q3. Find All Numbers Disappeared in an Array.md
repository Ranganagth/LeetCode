[Q3. Find All Numbers Disappeared in an Array](https://leetcode.com/problems/find-all-numbers-disappeared-in-an-array/)

# Intuition

Since all numbers are in the range `[1, n]`, we can use the input array itself to track which numbers have appeared by marking visited positions. The key idea is to:
- Iterate through `nums`.
- For each number `num`, mark the position `abs(num) - 1` as visited by negating the value at that position.
- After processing, the indices that remain positive correspond to numbers that did not appear in `nums`.

This approach allows us to find missing numbers **without extra space** and in **O(n)** time.

---

# Approach

1. Loop through each element in `nums`.
2. For each element, take its absolute value to handle previously negated numbers.
3. Use this value to find the index (`abs(nums[i]) - 1`) and negate the number at that index if it's positive.
4. After the first pass, iterate through `nums` again:
   - The indices with positive values indicate the missing numbers (`index + 1`).
5. Collect these missing numbers into a result array.

---

# Complexity

- **Time complexity:**  $$O(n)$$, as we do two passes over the array.
  
- **Space complexity:** $$O(1)$$ (excluding the output list), since we modify the input array in-place.

---

# Code

```typescript
function findDisappearedNumbers(nums: number[]): number[] {
    const result: number[] = [];

    // Mark visited positions
    for (let i = 0; i < nums.length; i++) {
        const index = Math.abs(nums[i]) - 1;
        if (nums[index] > 0) {
            nums[index] = -nums[index];
        }
    }

    // Collect missing numbers
    for (let i = 0; i < nums.length; i++) {
        if (nums[i] > 0) {
            result.push(i + 1);
        }
    }

    return result;
};

```
