[Q1. Concatenation of Array](https://leetcode.com/problems/concatenation-of-array/)

# Intuition

The problem is straightforward: we need to create a new array `ans` which is the concatenation of `nums` with itself. Essentially, `ans` contains two copies of `nums` back-to-back.

---

# Approach

1. Initialize an array `ans` of length `2 * n`.
2. Fill the first `n` elements of `ans` with `nums`.
3. Fill the next `n` elements with `nums` again.
4. Return `ans`.

---

# Complexity

- **Time complexity:**  $$O(n)$$, as we just copy the array twice.
  
- **Space complexity:**  $$O(n)$$ for the output array.

---

# Code

```typescript
function getConcatenation(nums: number[]): number[] {
    const n = nums.length;
    const ans = new Array(2 * n);
    for (let i = 0; i < n; i++) {
        ans[i] = nums[i];
        ans[i + n] = nums[i];
    }
    return ans;
};

```

# Example walkthrough
## Example 1:
Input: `nums = [1, 2, 1]`

- ans[0] = 1, ans[3] = 1
- ans[1] = 2, ans[4] = 2
- ans[2] = 1, ans[5] = 1

Output: `[1, 2, 1, 1, 2, 1]`

## Example 2:
Input: `nums = [1, 3, 2, 1]`

- ans = `[1, 3, 2, 1, 1, 3, 2, 1]`

This simple approach efficiently produces the desired concatenated array.
