[Q3. Max Consecutive Ones](https://leetcode.com/problems/max-consecutive-ones/)

# Intuition

We need to find the longest streak of consecutive 1's in the array. We can do this by iterating through the array and keeping track of the current streak of 1's, updating the maximum streak whenever the current streak exceeds it.

---

# Approach

1. Initialize two variables: `maxCount` to keep track of the maximum consecutive 1's found so far, and `currentCount` to count the current streak.
2. Loop through each element in `nums`:
   - If the element is 1, increment `currentCount`.
   - If the element is 0, reset `currentCount` to 0.
   - Update `maxCount` with the maximum of itself and `currentCount`.
3. Return `maxCount`.

---

# Complexity

- **Time complexity:**  $$O(n)$$, where `n` is the length of `nums`, since we scan the array once.

- **Space complexity:**  $$O(1)$$, as we only use a few variables.

---

# Code

```typescript
function findMaxConsecutiveOnes(nums: number[]): number {
    let maxCount = 0;
    let currentCount = 0;

    for (const num of nums) {
        if (num === 1) {
            currentCount++;
            maxCount = Math.max(maxCount, currentCount);
        } else {
            currentCount = 0;
        }
    }

    return maxCount;
};

```

# Example walkthrough
## Example 1:

Input: `[1,1,0,1,1,1]`

- Start: maxCount=0, currentCount=0
- Read 1: currentCount=1, maxCount=1
- Read 1: currentCount=2, maxCount=2
- Read 0: reset currentCount=0
- Read 1: currentCount=1, maxCount=2
- Read 1: currentCount=2, maxCount=2
- Read 1: currentCount=3, maxCount=3

Output: `3`

## Example 2:

Input: `[1,0,1,1,0,1]`

- Max streaks are 1, 2, 1, so output is 2.
