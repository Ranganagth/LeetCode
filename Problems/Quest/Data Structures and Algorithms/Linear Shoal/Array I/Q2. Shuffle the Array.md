[Q2. Shuffle the Array](https://leetcode.com/problems/shuffle-the-array/)
# Intuition

The array `nums` is split into two parts:
- First `n` elements: `[x₁, x₂, ..., xₙ]`
- Last `n` elements: `[y₁, y₂, ..., yₙ]`

We need to interleave these two parts: `[x₁, y₁, x₂, y₂, ..., xₙ, yₙ]`.

---

# Approach

1. Initialize an empty array `result`.
2. Loop through indices from `0` to `n-1`.
3. For each index `i`, append `nums[i]` (from the first part) and `nums[i + n]` (from the second part) to `result`.
4. Return `result`.

---

# Complexity

- **Time complexity:**  $$O(n)$$, as we iterate through the array once.
  
- **Space complexity:**  $$O(n)$$ for the output array.

---

# Code

```typescript
function shuffle(nums: number[], n: number): number[] {
    const result: number[] = [];
    for (let i = 0; i < n; i++) {
        result.push(nums[i], nums[i + n]);
    }
    return result;
};

```

# Example walkthrough
## Example 1:

Input: `nums = [2,5,1,3,4,7]`, `n=3`

- Loop i=0: push `nums[0]=2`, `nums[3]=3` → `[2,3]`
- Loop i=1: push `nums[1]=5`, `nums[4]=4` → `[2,3,5,4]`
- Loop i=2: push `nums[2]=1`, `nums[5]=7` → `[2,3,5,4,1,7]`

Output: `[2,3,5,4,1,7]`

## Example 2:

Input: `nums = [1,2,3,4,4,3,2,1]`, `n=4`

- Result: `[1,4,2,3,3,2,4,1]`

This approach efficiently interleaves the two parts of the array.
