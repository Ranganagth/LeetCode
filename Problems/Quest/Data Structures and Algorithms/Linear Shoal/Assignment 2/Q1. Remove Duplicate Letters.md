# Intuition

We need to remove duplicate letters while maintaining the smallest lexicographical order. This is a **greedy problem with a stack**.

Key insights:
1. Each letter should appear exactly once in the result.
2. We want the lexicographically smallest result.
3. We can use a **stack** to build the result greedily.
4. For each character:
   - If it's already in the result, skip it.
   - If it's smaller than the top of the stack AND the top character appears later in the string, we can safely remove the top.
   - Add the current character to the stack.

We need to track:
- **Character count**: How many times each character appears in the remaining string.
- **In stack**: Which characters are currently in the result.
- **Stack**: The result being built.

---

# Approach

1. Count the frequency of each character.
2. Use a stack to build the result and a set to track which characters are in the stack.
3. For each character in the string:
   - Decrease its count.
   - If it's already in the stack, skip it.
   - While the stack is not empty AND the top character is greater than the current character AND the top character appears later (count > 0):
     - Pop the top character and remove it from the "in stack" set.
   - Push the current character to the stack and mark it as "in stack".
4. Return the stack as a string.

---

# Complexity

- **Time complexity:** 
  $$O(n)$$, where `n` is the length of the string. Each character is pushed and popped from the stack at most once.

- **Space complexity:** 
  $$O(1)$$ for the stack and set (at most 26 lowercase letters).

# Code

```typescript
function removeDuplicateLetters(s: string): string {
    // Count frequency of each character
    const charCount = new Map<string, number>();
    for (const char of s) {
        charCount.set(char, (charCount.get(char) || 0) + 1);
    }

    // Track which characters are in the stack
    const inStack = new Set<string>();

    // Stack to build the result
    const stack: string[] = [];

    for (const char of s) {
        // Decrease the count of current character
        charCount.set(char, charCount.get(char)! - 1);

        // If character is already in stack, skip it
        if (inStack.has(char)) {
            continue;
        }

        // Remove characters from stack if:
        // 1. Top of stack is greater than current character
        // 2. Top character appears later in the string (count > 0)
        while (
            stack.length > 0 &&
            stack[stack.length - 1] > char &&
            charCount.get(stack[stack.length - 1])! > 0
        ) {
            const removed = stack.pop()!;
            inStack.delete(removed);
        }

        // Add current character to stack
        stack.push(char);
        inStack.add(char);
    }

    return stack.join('');
};

```
