# Intuition

We need to sort without built-in functions in O(n log n) time with minimal space. The best algorithms for this are:

1. **Merge Sort** - O(n log n) time, O(n) space (stable)
2. **Quick Sort** - O(n log n) average, O(log n) space (in-place but not stable)
3. **Heap Sort** - O(n log n) time, O(1) space (in-place but not stable)

For smallest space complexity, **Heap Sort** is optimal with O(1) extra space. However, **Quick Sort** with O(log n) stack space is also acceptable and often faster in practice.

---

# Approach

## Solution 1: Heap Sort (O(1) Space - Optimal)

1. **Build max heap:** Heapify the array
2. **Extract elements:** Repeatedly swap root with last element and heapify
3. **In-place sorting:** No extra array needed

## Solution 2: Quick Sort (O(log n) Space)

1. **Choose pivot:** Select a pivot element
2. **Partition:** Rearrange so elements < pivot are left, > pivot are right
3. **Recursively sort:** Sort left and right partitions
4. **Optimization:** Use random pivot to avoid worst case

## Solution 3: Merge Sort (O(n) Space)

1. **Divide:** Split array into halves
2. **Conquer:** Recursively sort each half
3. **Merge:** Combine sorted halves

---

# Complexity

**Heap Sort:**
- **Time complexity:** O(n log n) - guaranteed
- **Space complexity:** O(1) - in-place

**Quick Sort:**
- **Time complexity:** O(n log n) average, O(n^2) worst
- **Space complexity:** O(log n) - recursion stack

**Merge Sort:**
- **Time complexity:** O(n log n) - guaranteed
- **Space complexity:** O(n) - temporary arrays

# Code

## Solution 1: Heap Sort (Smallest Space)

```typescript
function sortArray(nums: number[]): number[] {
    const n = nums.length;
    
    // Build max heap
    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
        heapify(nums, n, i);
    }
    
    // Extract elements from heap one by one
    for (let i = n - 1; i > 0; i--) {
        // Move current root to end
        [nums[0], nums[i]] = [nums[i], nums[0]];
        
        // Heapify the reduced heap
        heapify(nums, i, 0);
    }
    
    return nums;
    
    function heapify(arr: number[], size: number, root: number): void {
        let largest = root;
        const left = 2 * root + 1;
        const right = 2 * root + 2;
        
        // If left child is larger than root
        if (left < size && arr[left] > arr[largest]) {
            largest = left;
        }
        
        // If right child is larger than largest so far
        if (right < size && arr[right] > arr[largest]) {
            largest = right;
        }
        
        // If largest is not root
        if (largest !== root) {
            [arr[root], arr[largest]] = [arr[largest], arr[root]];
            
            // Recursively heapify the affected sub-tree
            heapify(arr, size, largest);
        }
    }
};

```

## Solution 2: Quick Sort (Good Average Performance)

```typescript
function sortArray(nums: number[]): number[] {
    quickSort(nums, 0, nums.length - 1);
    return nums;
    
    function quickSort(arr: number[], low: number, high: number): void {
        if (low < high) {
            // Partition and get pivot index
            const pivotIndex = partition(arr, low, high);
            
            // Recursively sort elements before and after partition
            quickSort(arr, low, pivotIndex - 1);
            quickSort(arr, pivotIndex + 1, high);
        }
    }
    
    function partition(arr: number[], low: number, high: number): number {
        // Choose random pivot to avoid worst case
        const randomIndex = low + Math.floor(Math.random() * (high - low + 1));
        [arr[randomIndex], arr[high]] = [arr[high], arr[randomIndex]];
        
        const pivot = arr[high];
        let i = low - 1;
        
        for (let j = low; j < high; j++) {
            if (arr[j] <= pivot) {
                i++;
                [arr[i], arr[j]] = [arr[j], arr[i]];
            }
        }
        
        [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
        return i + 1;
    }
};

```

## Solution 3: Merge Sort (Stable Sort)

```typescript
function sortArray(nums: number[]): number[] {
    if (nums.length <= 1) return nums;
    
    mergeSort(nums, 0, nums.length - 1);
    return nums;
    
    function mergeSort(arr: number[], left: number, right: number): void {
        if (left < right) {
            const mid = Math.floor((left + right) / 2);
            
            // Sort first and second halves
            mergeSort(arr, left, mid);
            mergeSort(arr, mid + 1, right);
            
            // Merge the sorted halves
            merge(arr, left, mid, right);
        }
    }
    
    function merge(arr: number[], left: number, mid: number, right: number): void {
        const n1 = mid - left + 1;
        const n2 = right - mid;
        
        // Create temp arrays
        const L = new Array(n1);
        const R = new Array(n2);
        
        // Copy data to temp arrays
        for (let i = 0; i < n1; i++) {
            L[i] = arr[left + i];
        }
        for (let j = 0; j < n2; j++) {
            R[j] = arr[mid + 1 + j];
        }
        
        // Merge the temp arrays back
        let i = 0, j = 0, k = left;
        
        while (i < n1 && j < n2) {
            if (L[i] <= R[j]) {
                arr[k] = L[i];
                i++;
            } else {
                arr[k] = R[j];
                j++;
            }
            k++;
        }
        
        // Copy remaining elements
        while (i < n1) {
            arr[k] = L[i];
            i++;
            k++;
        }
        
        while (j < n2) {
            arr[k] = R[j];
            j++;
            k++;
        }
    }
};

```

## Solution 4: Optimized Quick Sort (3-Way Partition)

```typescript
function sortArray(nums: number[]): number[] {
    quickSort3Way(nums, 0, nums.length - 1);
    return nums;
    
    function quickSort3Way(arr: number[], low: number, high: number): void {
        if (low >= high) return;
        
        // 3-way partition
        const [lt, gt] = partition3Way(arr, low, high);
        
        // Recursively sort
        quickSort3Way(arr, low, lt - 1);
        quickSort3Way(arr, gt + 1, high);
    }
    
    function partition3Way(arr: number[], low: number, high: number): [number, number] {
        // Random pivot
        const randomIndex = low + Math.floor(Math.random() * (high - low + 1));
        [arr[low], arr[randomIndex]] = [arr[randomIndex], arr[low]];
        
        const pivot = arr[low];
        let lt = low; // arr[low..lt-1] < pivot
        let i = low + 1; // arr[lt..i-1] == pivot
        let gt = high; // arr[gt+1..high] > pivot
        
        while (i <= gt) {
            if (arr[i] < pivot) {
                [arr[lt], arr[i]] = [arr[i], arr[lt]];
                lt++;
                i++;
            } else if (arr[i] > pivot) {
                [arr[i], arr[gt]] = [arr[gt], arr[i]];
                gt--;
            } else {
                i++;
            }
        }
        
        return [lt, gt];
    }
};

```

# Example Walkthrough

## Example 1: Heap Sort

**Input:** `nums = [5,2,3,1]`

**Step 1: Build Max Heap**
```
Initial: [5,2,3,1]
Heapify from index 1: [5,2,3,1]
Heapify from index 0: [5,2,3,1]
Max heap: [5,2,3,1]
```

**Step 2: Extract Elements**
```
Swap 5 and 1: [1,2,3,5]
Heapify: [3,2,1,5]

Swap 3 and 1: [1,2,3,5]
Heapify: [2,1,3,5]

Swap 2 and 1: [1,2,3,5]
Done!
```

**Output:** `[1,2,3,5]`

---

## Example 2: Quick Sort

**Input:** `nums = [5,1,1,2,0,0]`

**Partition with pivot=0:**
```
[0,0,1,2,5,1]
Pivot at correct position
```

**Recursively sort left and right:**
```
Left: [0,0] - already sorted
Right: [1,2,5,1] - continue partitioning
```

**Final:** `[0,0,1,1,2,5]`

## Performance Comparison

| Algorithm | Time (Best) | Time (Avg) | Time (Worst) | Space | Stable |
|-----------|-------------|------------|--------------|-------|--------|
| Heap Sort | O(n log n) | O(n log n) | O(n log n) | O(1) | No |
| Quick Sort | O(n log n) | O(n log n) | O(n^2) | O(log n) | No |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) | Yes |
| 3-Way Quick | O(n) | O(n log n) | O(n^2) | O(log n) | No |

---

## Key Insights

- **Heap Sort:** Best for space-constrained environments
- **Quick Sort:** Fastest average case, good cache performance
- **Merge Sort:** Guaranteed O(n log n), stable sorting
- **3-Way Partition:** Excellent for arrays with many duplicates
- **Random Pivot:** Prevents worst-case O(n^2) in Quick Sort

---

## Recommended Solution

For **smallest space complexity**: Use **Heap Sort** (Solution 1)
For **best average performance**: Use **Quick Sort with random pivot** (Solution 2 or 4)
For **guaranteed O(n log n)**: Use **Merge Sort** (Solution 3)
