# Intuition

We need to find if there exist two integers `a` and `b` such that `a^2 + b^2 = c`.

The key insight is that if such integers exist, then `a` and `b` must be in the range `[0, sqrt(c)]` because if either is larger than `sqrt(c)`, their square alone would exceed `c`.

We can use a two-pointer approach: start with `a = 0` and `b = sqrt(c)`, then adjust based on whether `a^2 + b^2` is less than, equal to, or greater than `c`.

---

# Approach

## Solution 1: Two Pointers (Optimal)

1. **Initialize pointers:** `left = 0`, `right = floor(sqrt(c))`

2. **Two-pointer search:** While `left <= right`:
   - Calculate `sum = left^2 + right^2`
   - If `sum == c`, return `true`
   - If `sum < c`, increment `left` (need larger sum)
   - If `sum > c`, decrement `right` (need smaller sum)

3. **Not found:** Return `false`

## Solution 2: Hash Set

1. For each `a` from `0` to `sqrt(c)`:
   - Calculate `b^2 = c - a^2`
   - Check if `b^2` is a perfect square

---

# Complexity

**Two Pointers:**
- **Time complexity:** O(sqrt(c)) - iterate from 0 to sqrt(c)
- **Space complexity:** O(1) - only using pointers

**Hash Set:**
- **Time complexity:** O(sqrt(c))
- **Space complexity:** O(sqrt(c)) - storing squares

# Code

## Solution 1: Two Pointers (Optimal)

```typescript
function judgeSquareSum(c: number): boolean {
    let left = 0;
    let right = Math.floor(Math.sqrt(c));
    
    while (left <= right) {
        const sum = left * left + right * right;
        
        if (sum === c) {
            return true;
        } else if (sum < c) {
            left++;
        } else {
            right--;
        }
    }
    
    return false;
};

```

## Solution 2: Using Set for Perfect Squares

```typescript
function judgeSquareSum(c: number): boolean {
    const maxVal = Math.floor(Math.sqrt(c));
    const squares = new Set<number>();
    
    // Store all perfect squares up to c
    for (let i = 0; i <= maxVal; i++) {
        squares.add(i * i);
    }
    
    // Check if c - a^2 is also a perfect square
    for (let a = 0; a <= maxVal; a++) {
        const aSquared = a * a;
        const bSquared = c - aSquared;
        
        if (squares.has(bSquared)) {
            return true;
        }
    }
    
    return false;
};

```

## Solution 3: Direct Square Root Check

```typescript
function judgeSquareSum(c: number): boolean {
    for (let a = 0; a * a <= c; a++) {
        const bSquared = c - a * a;
        const b = Math.sqrt(bSquared);
        
        // Check if b is an integer
        if (Number.isInteger(b)) {
            return true;
        }
    }
    
    return false;
};

```

## Solution 4: Optimized with Early Termination

```typescript
function judgeSquareSum(c: number): boolean {
    // Special case
    if (c === 0) return true;
    
    let left = 0;
    let right = Math.floor(Math.sqrt(c));
    
    while (left <= right) {
        // Use BigInt to avoid overflow for large numbers
        const sum = left * left + right * right;
        
        if (sum === c) {
            return true;
        } else if (sum < c) {
            left++;
        } else {
            right--;
        }
    }
    
    return false;
};

```

## Key Insights

- **Two pointers work:** Because the sum is monotonic in both directions
- **Square root bound:** Both `a` and `b` must be at most `sqrt(c)`
- **Integer check:** When using square root, verify the result is an integer
- **No overflow:** For c up to 2^31-1, intermediate calculations fit in JavaScript numbers

## Mathematical Background

**Fermat's Theorem on Sums of Two Squares:**

A positive integer `c` can be expressed as the sum of two squares if and only if in its prime factorization, every prime of the form `4k + 3` occurs to an even power.

However, the two-pointer approach is simpler and more efficient for this problem.

## Why Two Pointers Works

**Monotonicity property:**
- If `left^2 + right^2 < c`, increasing `left` increases the sum
- If `left^2 + right^2 > c`, decreasing `right` decreases the sum
- We explore all possible combinations without missing any

**Correctness:**
- We start with the full range `[0, sqrt(c)]`
- Each step eliminates one value that cannot be part of the solution
- If a solution exists, we will find it
